## 事業部構成

```
アサヒグループ食品
├── 【BtoC事業】
│   ├── 菓子・飲料事業（ミンティア等）
│   ├── フリーズドライ食品事業（アマノフーズ）
│   ├── ベビー・シニア事業（和光堂）

```

## 各事業部詳細

### BU-1: 菓子・飲料事業

| 項目 | 内容 |
|------|------|
| 主要ブランド | ミンティア、一本満足バー |
| 市場ポジション | タブレット市場シェアNo.1 |
| 売上規模 | 約207億円（ミンティア単体、2017年） |
| 顧客セグメント | 一般消費者（20-50代ビジネスパーソン中心） |
| チャネル | コンビニ、スーパー、ドラッグストア |
| 競合 | フリスク（クラシエ）、クロレッツ（モンデリーズ） |
| 差別化要素 | 多様なフレーバー展開、リーズナブル価格 |

**事業特性**:
- 高頻度リピート購買（衝動買い型）
- 季節限定商品による話題性創出
- コンビニチャネルへの依存度高

## 事業間関係性マトリクス

| 関係性 | BU-1 菓子 | BU-2 FD | BU-3 ベビー | BU-4 サプリ | BU-5 原料 |
|--------|----------|---------|-------------|-------------|-----------|
| **BU-1 菓子** | - | 低 | 低 | 中（健康軸） | 低 |
| **BU-2 FD** | 低 | - | 中（和食軸） | 低 | 高（技術） |
| **BU-3 ベビー** | 低 | 中 | - | 中（栄養軸） | 中 |
| **BU-4 サプリ** | 中 | 低 | 中 | - | 高（酵母） |
| **BU-5 原料** | 低 | 高 | 中 | 高 | - |

**シナジーの種類**:
- **技術シナジー**: 発酵・酵母技術（BU-4↔BU-5）、フリーズドライ技術（BU-2↔BU-5）
- **顧客シナジー**: 限定的（各事業で顧客セグメントが異なる）
- **チャネルシナジー**: ドラッグストアチャネル（BU-3, BU-4）
- **ブランドシナジー**: 限定的（各ブランドが独立して認知）

## コングロマリット判定材料

### 事業多角化の程度

| 観点 | 評価 | 根拠 |
|------|------|------|
| 事業ドメインの分散 | 中 | 食品領域内に収まるが、菓子・FD・ベビー・サプリと幅広い |
| 顧客セグメントの分散 | 高 | ビジネスパーソン、主婦、乳幼児保護者、高齢者、法人と多様 |
| チャネルの分散 | 中 | コンビニ、スーパー、ドラッグストア、通販、BtoB |
| 競合の分散 | 高 | 各事業で異なる競合と対峙 |
| 技術基盤の共通性 | 中 | 発酵・フリーズドライは共通、菓子は別系統 |

### 売上構成（推定）

```
BU-4 サプリメント: 30%（成長ドライバー）
BU-2 フリーズドライ: 25%
BU-3 ベビー・シニア: 25%
BU-1 菓子・飲料: 15%
BU-5 食品原料: 5%
```
