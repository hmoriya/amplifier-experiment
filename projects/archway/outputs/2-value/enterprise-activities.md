# エンタープライズ活動 - 株式会社アークウェイ

## エンタープライズ活動マップ

### 活動カテゴリー
1. **戦略的活動**: ビジネス方向性を定める活動
2. **コア活動**: 直接的な顧客価値を生む活動
3. **支援活動**: コア活動を支える活動
4. **管理活動**: 組織運営に必要な活動

## VS0: ビジョン策定と戦略立案

### 戦略的活動

#### SA1: デジタル戦略コンサルティング
- **内容**: 経営層との戦略対話とビジョン策定
- **プロセス**:
  1. 経営課題ヒアリング
  2. 業界分析・競合分析
  3. デジタル成熟度評価
  4. 戦略オプション提示
  5. 実行計画策定
- **成果物**: デジタル戦略書、ロードマップ
- **スキル要件**: 戦略コンサルティング、業界知識

#### SA2: IT投資計画策定
- **内容**: ROIに基づく投資優先順位付け
- **プロセス**:
  1. 現状IT投資分析
  2. 必要投資の洗い出し
  3. 効果予測・ROI算出
  4. 段階的投資計画作成
- **成果物**: IT投資計画書、ROI分析
- **スキル要件**: 財務分析、IT投資評価

## 活動間の連携

### 情報フロー
```
戦略的活動（方向性）
    ↓
コア活動（価値創造）
    ↕
支援活動（実現支援）
    ↑
管理活動（統制）
```

### 重要な連携ポイント
1. SA1→CA3: 戦略からアーキテクチャへの落とし込み
2. CA5→SA3: DevOps導入と組織変革の同期
3. CA7→SA7: 改善知見の教育への反映
4. SA5→CA5: ツールとプロセスの統合
