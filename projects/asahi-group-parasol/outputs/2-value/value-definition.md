## 1. 企業理念と価値観

```yaml
ステートメント: "Make the world shine - おいしさと楽しさで、世界を輝かせる"

意味:
  - 世界中の人々のより良い明日への強い想い
  - おいしさと楽しさの提供を通じた社会貢献

  - 2024年3月に新グループコーポレートロゴマークとともに策定
```

```yaml

  "期待を超えるおいしさ、楽しい生活文化の創造"
```

## 2. 価値提供の本質

### 顧客への価値提供
```yaml
Core Promise: "おいしさの約束と、すべての人により多くの楽しさをもたらす"

提供価値の3層構造:

レベル1 - 機能的価値:
  - 最高品質の飲料・食品
  - 安全・安心な製品
  - 多様な選択肢（ビール、ノンアルコール、機能性飲料等）
  - 利便性（購入・消費の容易さ）

レベル2 - 経験的価値:
  - 期待を超える美味しさ体験
  - ブランド体験（スーパードライ、カルピス等）
  - 楽しいライフスタイル文化
  - 特別な瞬間の演出

レベル3 - 社会的価値:
  - サステナブルな未来への貢献
  - 地域コミュニティとの共生
  - 健康的なライフスタイル支援
  - 文化的価値の創造
```
