# アサヒグループ 価値ストリームマッピング

## プロジェクト基本情報

**作成日**: 2024年11月20日
**対象組織**: アサヒグループホールディングス株式会社
**フレームワーク**: Parasol V4 Lite - Value Streams (VS0-VS7)

## Value Stream概要

価値ストリーム（Value Stream）は、顧客に価値を届けるまでのエンドツーエンドのプロセスを表します。アサヒグループでは、ビジョン策定から価値の振り返りまで、8段階の価値ストリームを通じて顧客価値を創造します。

```yaml
  - サステナビリティビジョンの統合
  - グローバルブランド戦略の方向性決定

ステークホルダー対話:
  - 株主・投資家との対話
  - 従業員エンゲージメント
  - 社会・地域との対話
  - 規制当局との協議

組織整合:
  - グループ全体の方向性統一
  - 3地域（日本・欧州・アジア太平洋）の戦略整合
  - 機能横断の連携体制構築
```

### インプット

```yaml
  - 規制環境変化
```

## バリューストリーム統合ビュー

### エンドツーエンドフロー

```
VS0: ビジョン策定
    ↓ (戦略方針)
VS1: 市場機会発見
    ↓ (機会特定)
VS2: 価値提案設計
    ↓ (製品コンセプト)
VS3: 製品・サービス開発
    ↓ (完成製品)
VS4: 製造・品質保証
    ↓ (市場投入製品)
VS5: マーケティング・販売
    ↓ (顧客認知・購買)
VS6: 顧客体験提供
    ↓ (体験・フィードバック)
VS7: 価値の振り返りと進化
    ↓ (学習・改善)
    → VS0-VS6へフィードバック
```

### リードタイム
```yaml
新製品開発サイクル:
  - コンセプト発想～市場投入: 12-24ヶ月
  - 内訳:
    • VS1-VS2: 3-6ヶ月
    • VS3: 6-12ヶ月
    • VS4: 2-4ヶ月（生産立ち上げ）
    • VS5-VS6: 1-2ヶ月（市場導入）

継続改善サイクル:
  - 四半期レビュー（VS7 → VS1-VS6フィードバック）
  - 年次戦略レビュー（VS0更新）
```

### 価値創造の測定
```yaml
バリューストリームKPI統合:
  VS0: 戦略浸透度、ESG評価
  VS1: 機会発見数、市場カバレッジ
  VS2: コンセプトテスト評価、差別化認知
  VS3: 新製品数、開発効率、特許数
  VS4: 生産量、品質率、環境効率
  VS5: シェア、ブランド認知、マーケティングROI
  VS6: NPS、満足度、ロイヤルティ
  VS7: インサイト数、改善実施率

統合成果指標:
  - 売上高成長率
  - 営業利益率
  - ROE
  - 顧客満足度
  - ブランド価値
  - ESGスコア
```
