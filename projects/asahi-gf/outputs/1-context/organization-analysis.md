## 2. コア・コンピテンシー

### 技術的強み

#### 2.1 フリーズドライ技術（アマノフーズ）
- **概要**: 真空凍結乾燥技術による食品保存
- **特徴**:
  - 素材本来の味・香り・栄養を保持
  - 長期保存が可能
  - 軽量で携帯性に優れる
- **応用**: 味噌汁、スープ、カレー、パスタ等

#### 2.2 ビール酵母技術（エビオス）
- **概要**: 1929年から続くビール酵母活用技術
- **特徴**:
  - 天然由来の栄養素（ビタミンB群、アミノ酸、食物繊維）
  - 整腸作用
  - 長年の研究蓄積
- **応用**: エビオス錠、健康食品原料

#### 2.3 育児栄養研究（和光堂）
- **概要**: 1906年創業以来100年超の育児研究
- **特徴**:
  - 乳児の栄養・発達に関する深い知見
  - 安全性への徹底したこだわり
  - 世代を超えた信頼
- **応用**: ベビーフード、粉ミルク、育児用品

#### 2.4 栄養設計力
- **概要**: ライフステージに応じた栄養設計能力
- **特徴**:
  - 科学的エビデンスに基づく製品開発
  - 機能性表示食品の開発実績
  - 多様なニーズへの対応力

## 3. 組織文化・価値観

### 企業理念（アサヒグループ共通）
- **Our Mission**: 期待を超えるおいしさ、楽しい生活文化の創造
- **Our Vision**: 高付加価値ブランドを核として成長する"グローカルな価値創造企業"
- **Our Values**: 挑戦と革新、最高の品質、感謝と誠実

### 組織文化の特徴
1. **品質第一**: 食品安全への徹底したコミットメント
2. **顧客起点**: 消費者ニーズを起点とした製品開発
3. **科学的アプローチ**: エビデンスに基づく製品設計
4. **長期視点**: ブランド価値の長期的な育成

## 4. 人材・能力

### 推定される主要人材領域
1. **研究開発人材**: 食品科学、栄養学、生化学の専門家
2. **品質管理人材**: HACCP、ISO認証等の専門知識
3. **マーケティング人材**: ブランド管理、消費者インサイト
4. **生産技術人材**: 製造プロセス最適化、自動化

### 組織的強み
- 統合前の各社（和光堂、天野実業等）からの専門知識継承
- アサヒグループ全体でのナレッジ共有
- 長年蓄積された製品開発ノウハウ

## 5. SWOT分析サマリー

### Strengths（強み）
- 100年超の歴史を持つ老舗ブランド群
- 独自技術（フリーズドライ、ビール酵母）
- ライフステージ全体をカバーする製品ポートフォリオ
- アサヒグループのリソースとシナジー

### Weaknesses（弱み）
- 複数事業統合による組織的複雑性
- 国内市場依存度の高さ
- ブランド間のカニバリゼーションリスク

### Opportunities（機会）
- 健康志向の高まり
- 高齢化社会でのシニア食品需要
- 海外市場への展開可能性
- EC・D2Cチャネルの成長

### Threats（脅威）
- 原材料価格の上昇
- 少子化によるベビーフード市場縮小
- PB商品との競争激化
- 新規参入者の増加
