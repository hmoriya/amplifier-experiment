# クロスバリューストリーム分析：共通ケイパビリティ抽出

**作成日**: 2025-12-11
**プロジェクト**: アサヒグループ食品株式会社
**Phase**: 3 - Capabilities Decomposition
**分析対象**: VS1-VS4のクロス分析

---

## 1. エグゼクティブサマリー

### 1.1 分析結果概要

アサヒグループ食品の4つの主要バリューストリームを分析した結果、以下の共通ケイパビリティを抽出：

1. **技術基盤（Core）**: 複数VSで活用される差別化技術
2. **品質管理基盤（Supporting）**: 全VS共通の品質保証体制
3. **データ分析基盤（Supporting）**: 顧客理解と意思決定支援
4. **SCM・物流基盤（Supporting）**: 効率的な供給体制
5. **管理機能（Generic）**: 標準的な事業支援

### 1.2 戦略的インプリケーション

- **技術シナジーの最大化**: Core技術のVS間共有で競争力強化
- **基盤機能の効率化**: Supporting領域の統合・標準化
- **デジタル変革の加速**: データ基盤を活用した価値創造

## 6. 設計判断の記録

**Q: なぜ技術基盤を「共通ケイパビリティ」として抽出したか？**

A: 各技術（FD、酵母、栄養、製剤）は単一VSだけでなく、複数VSでの活用可能性がある。技術シナジーを最大化するためには、共通基盤として管理・展開することが効果的。

**Q: Generic領域の外部化タイミングは？**

A: 即座に実施可能。特にITインフラや一般管理機能は、成熟したクラウドサービスが存在し、移行リスクも低い。

**Q: データ分析基盤の優先度が高い理由は？**

A: デジタル変革の中核であり、全VSでの価値創造に直結する。特にパーソナライズ提案や需要予測は、今後の競争優位性の源泉となる可能性が高い。
