## 2. 主活動（Primary Activities）

### 2.1 研究開発活動

#### 活動概要
| 項目 | 内容 |
|------|------|
| 目的 | 新製品・新技術の創出、既存製品の改良 |
| 組織 | 研究開発部門（推定150名） |
| 投資 | 売上高比約2-3%（推定） |

#### 主要活動

| 活動 | 内容 | アウトプット |
|------|------|-------------|
| 基礎研究 | 栄養科学、食品科学の研究 | 研究論文、特許 |
| 応用研究 | 製品化に向けた技術開発 | 技術シーズ |
| 製品開発 | 新製品の処方・設計 | 製品仕様書 |
| 品質研究 | 安全性・安定性の検証 | 品質データ |

### 2.2 製品設計・開発活動

#### 活動概要
| 項目 | 内容 |
|------|------|
| 目的 | 研究成果の製品化、市場ニーズへの対応 |

### 2.3 調達・物流活動

#### 調達活動

| 項目 | 内容 |
|------|------|
| 目的 | 高品質原材料の安定調達、コスト最適化 |
| 調達先 | 国内外サプライヤー |
| 方針 | 品質優先、持続可能性配慮 |

#### 主要調達品目

| カテゴリ | 品目例 | 調達方針 |
|----------|--------|----------|
| 農産物 | 野菜、果物、穀物 | 産地指定、契約栽培 |
| 乳製品 | 乳原料 | 品質基準厳格化 |
| 機能性原料 | ビタミン、ミネラル | 認証取得品 |
| 包装資材 | 容器、フィルム | 環境配慮素材シフト |

#### 物流活動

| 活動 | 内容 | KPI |
|------|------|-----|
| 入荷物流 | 原材料の工場への輸送 | リードタイム |
| 工場間物流 | 製造拠点間の移動 | 効率性 |
| 出荷物流 | 製品の配送センターへの輸送 | コスト |
| 配送 | 小売店舗への配送 | 配送頻度、精度 |

## 6. 活動の最適化方向性

### 6.1 強化すべき活動

1. **デジタルマーケティング**: 消費者直接接点の強化
2. **データ活用**: 意思決定の高度化
3. **サステナビリティ**: 環境対応の加速

### 6.2 効率化すべき活動

1. **間接業務**: RPA・自動化の推進
2. **調達**: 共同購買、契約最適化
3. **物流**: 配送効率化

### 6.3 外部活用検討

1. **製造**: 一部OEM活用
2. **IT**: クラウド活用
3. **物流**: 3PL活用拡大
