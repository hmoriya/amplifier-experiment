# アサヒグループ食品 - 価値定義

## 1. 企業価値の定義

### 1.1 ミッション（存在意義）

> **「人生のあらゆるステージに寄り添い、おいしさと健康で人々の毎日を豊かにする」**

#### ミッションの解説
- **人生のあらゆるステージ**: ベビー（和光堂）から成人（ミンティア、ディアナチュラ）、シニア（バランス献立）まで
- **おいしさと健康**: 単なる栄養補給ではなく、食の楽しみと健康価値の両立
- **毎日を豊かに**: 日常に溶け込む製品で、持続的な価値提供

### 1.2 ビジョン（目指す姿）

> **「ライフステージ・ニュートリションのリーディングカンパニー」**

#### 2030年に向けた目標像
1. **国内トップクラスのライフステージ対応力**: 各年代向け製品でシェアトップ3
2. **科学的エビデンスに基づく信頼**: 機能性表示食品のラインナップ拡充
3. **サステナブルな事業モデル**: 環境負荷低減と事業成長の両立

### 1.3 バリュー（価値観・行動指針）

| バリュー | 定義 | 具体的行動 |
|----------|------|-----------|
| **品質第一** | 妥協なき品質へのこだわり | 厳格な品質基準、トレーサビリティ確保 |
| **顧客起点** | 消費者ニーズからの発想 | 消費者調査、VOC活用、製品改善 |
| **科学的根拠** | エビデンスに基づく製品開発 | 研究開発投資、学術連携 |
| **挑戦と革新** | 新しい価値の創造 | 新製品開発、新市場開拓 |
| **誠実と信頼** | ステークホルダーとの信頼関係 | 透明性、コンプライアンス |

## 3. 提供価値の3軸

### 3.1 Trust（信頼）
- **老舗の信頼**: 和光堂100年超、エビオス90年超の歴史
- **品質保証**: 厳格な品質管理体制
- **科学的根拠**: 研究開発に基づく製品設計
- **トレーサビリティ**: 原材料から製品までの追跡可能性

### 3.2 Taste（おいしさ）
- **素材の味**: フリーズドライ技術による風味保持
- **食べやすさ**: 各ライフステージに適した形態
- **嗜好性**: 継続摂取のための味の追求
- **多様性**: 豊富なフレーバー・バリエーション

### 3.3 Health（健康価値）
- **栄養設計**: ライフステージに応じた栄養バランス
- **機能性**: エビデンスに基づく健康機能
- **継続性**: 毎日続けられる製品設計
- **予防的価値**: 健康維持・増進への貢献
