## 2. 主要バリューストリーム

### 2.1 VS1: ベビー・シニア食品バリューストリーム

**対象ブランド**: 和光堂、バランス献立

```
[市場・顧客理解] → [製品企画] → [研究開発] → [製造] → [品質保証] → [販売] → [顧客サポート]
      │               │            │           │          │          │          │
      ▼               ▼            ▼           ▼          ▼          ▼          ▼
   育児/介護      ニーズを       安全性と     厳格な     徹底した    専門店     相談窓口
   トレンド把握   製品仕様化     栄養設計     衛生管理   品質検査    チャネル   対応
```

#### プロセス詳細

| ステージ | 活動 | インプット | アウトプット | 価値貢献 |
|----------|------|-----------|-------------|----------|
| 市場理解 | 育児/介護トレンド調査、VOC収集 | 市場データ、顧客声 | インサイトレポート | 顧客ニーズ把握 |
| 製品企画 | コンセプト開発、仕様策定 | インサイト、技術シーズ | 製品企画書 | 価値提案設計 |
| 研究開発 | 処方開発、安全性試験 | 企画書、原材料 | 製品仕様 | 品質・安全性 |
| 製造 | 原材料調達、製造、包装 | 仕様書、原材料 | 完成品 | 安定供給 |
| 品質保証 | 検査、認証、トレーサビリティ | 製品サンプル | 品質証明 | 信頼性担保 |
| 販売 | 営業、販促、店頭展開 | 完成品、販促物 | 売上 | 顧客アクセス |
| サポート | 相談対応、情報提供 | 顧客問い合わせ | 顧客満足 | 信頼構築 |

#### KPI
- 新製品開発リードタイム: 18ヶ月以内
- 品質クレーム率: 0.001%以下
- 顧客満足度: 85%以上

### 2.2 VS2: 健康食品バリューストリーム

**対象ブランド**: ディアナチュラ、エビオス、スリムアップスリム

```
[研究開発] → [エビデンス取得] → [製品設計] → [製造(GMP)] → [機能性表示] → [販売] → [継続購入]
     │              │               │             │             │           │          │
     ▼              ▼               ▼             ▼             ▼           ▼          ▼

  技術開発      文献レビュー                   GMP遵守                   チャネル   リピート促進

#### プロセス詳細

| ステージ | 活動 | インプット | アウトプット | 価値貢献 |
|----------|------|-----------|-------------|----------|
| 研究開発 | 有効成分研究、技術開発 | 学術情報、技術シーズ | 有効成分候補 | 科学的価値 |
| エビデンス | 試験実施、文献調査 | 成分候補 | エビデンスデータ | 信頼性 |
| 製品設計 | 処方設計、剤形開発 | エビデンス、ニーズ | 製品仕様 | 製品価値 |
| 製造 | GMP製造、品質管理 | 仕様書、原料 | 完成品 | 品質保証 |
| 表示対応 | 機能性表示届出 | エビデンス、製品情報 | 届出受理 | 訴求力 |
| 販売 | チャネル展開、販促 | 製品、販促物 | 売上 | 市場浸透 |
| CRM | 継続購入促進、情報提供 | 顧客データ | LTV向上 | 顧客価値 |

```

## 5. デジタル化による価値向上

### 5.1 現状と機会

| 領域 | 現状 | デジタル化機会 |
|------|------|----------------|
| 消費者理解 | 調査ベース | SNS分析、購買データ |
| 製品開発 | 経験ベース | AI活用、シミュレーション |
| 製造 | 自動化進行 | IoT、予知保全 |
| 販売 | 店頭中心 | EC強化、D2C |
| CRM | 限定的 | パーソナライズ |

### 5.2 デジタル投資優先度

1. **高**: EC/D2C基盤、消費者データ活用
2. **中**: 製造IoT、サプライチェーン可視化
3. **低**: 先端AI活用（段階的に）
