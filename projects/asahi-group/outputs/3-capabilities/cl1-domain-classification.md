## 2. Core ドメイン詳細

### VS1 研究開発

**分類理由**: 発酵プラットフォームの源泉。100年以上の酵母・発酵技術の蓄積は模倣困難な競争優位。

**CL1ケーパビリティ**:
| ID | ケーパビリティ | 説明 | 戦略的重要性 |
|----|---------------|------|--------------|
| VS1-C1 | 発酵プラットフォーム研究 | 酵母育種、発酵プロセス最適化、微生物学研究 | 最高 |
| VS1-C2 | 機能性素材研究 | 乳酸菌、機能性成分、健康効果研究 | 高 |
| VS1-C3 | 原料研究 | 大麦、ホップ、水、副原料の品質研究 | 高 |
| VS1-C4 | 基礎研究・産学連携 | 長期的研究テーマ、大学・研究機関との協働 | 高 |
| VS1-C5 | 知的財産管理 | 特許戦略、技術秘匿、ライセンス管理 | 高 |

### VS2 製品開発

**分類理由**: 「期待を超えるおいしさ」を具現化。VS1の技術を製品価値に変換する重要プロセス。

**CL1ケーパビリティ**:
| ID | ケーパビリティ | 説明 | 戦略的重要性 |
|----|---------------|------|--------------|
| VS2-C1 | プレミアムビール開発 | Asahi Super Dry、地域プレミアムブランドの処方設計 | 最高 |
| VS2-C2 | クラフト・イノベーション開発 | 限定品、実験的製品、新カテゴリ開拓 | 高 |
| VS2-C3 | ノンアルコール製品開発 | ノンアルビール、機能性飲料、健康志向製品 | 高 |
| VS2-C4 | スピリッツ・RTD開発 | ウイスキー、RTD、リキュール開発 | 中 |
| VS2-C5 | 官能評価・品質設計 | 味覚・香り設計、品質基準設定、製品評価 | 高 |

## 5. ドメイン間依存関係

### クリティカルパス

```
VS1（研究開発）→ VS2（製品開発）→ VS3（ブランド）

発酵技術 → 製品具現化 → 価値伝達
```

### 依存関係マトリクス

| From | To | タイプ | 説明 |
|------|-----|--------|------|
| VS1 | VS2 | Critical | 発酵技術・処方が製品開発の基盤 |
| VS2 | VS3 | Critical | 製品特性がブランドストーリーを決定 |
| VS3 | VS5 | Important | ブランド価値が営業力を増幅 |
| VS2 | VS4 | Important | 製品仕様が製造・物流要件を決定 |
| VS4 | VS6 | Important | 製造プロセスが品質管理要件を規定 |
| VS5 | VS1 | Feedback | 市場インサイト・VOCが研究開発にフィードバック |
| VS7 | VS1/2/3 | Extension | M&Aによるケイパビリティ獲得・強化 |
| VS0 | ALL | Foundation | 企業基盤が全VSの運営を支える |

## 6. 設計ストーリー

### なぜこの構成が最適か

1. **発酵プラットフォーム思考の適用**
   - VS1を全価値創造の源泉と位置づけ
   - 100年の技術蓄積が模倣困難な競争優位

2. **VL1との整合性**
   - 「おいしさ」= 技術（VS1/VS2）
   - 「楽しい生活文化」= 体験（VS3）
   - 両方がCoreとして不可欠

3. **投資の優先順位**
   - VS1→VS2→VS3のクリティカルパスに集中
   - Supporting領域は「確実に届ける」役割に特化

### 検討した代替案

| 代替案 | 却下理由 |
|--------|----------|
| VS4（SCM）をCoreに | SCM効率は差別化要因にならない |
| VS7（事業開発）をGenericに | M&A実行力はアサヒの強み |
| VS6（品質）をCoreに | 品質は前提条件であり差別化要因ではない |
| VS3（ブランド）をSupportingに | VL1「楽しい生活文化」に直結 |
