## 2. セグメント構造

### パターン判定

**質問1**: 顧客は何種類いますか？
- [x] 3種類以上 → **多面ステークホルダー型**

**質問2**: 顧客間に相互依存関係がありますか？
- **Yes**: 業務用顧客（飲食店）の売上が消費者の購買行動に影響
- 小売業者の品揃えが消費者のアクセスに影響

**質問3**: 内部ステークホルダー（従業員等）はサービス品質に直接影響しますか？
- **Yes**: 醸造技術者、研究者の専門性が製品品質に直結

### 判定結果

**パターン**: 多面ステークホルダー型

**理由**:
1. 4つの異なる顧客セグメントが存在
2. BtoC、BtoB、BtoBtoCの複合構造
3. グローバルとローカルの多層構造
4. 技術者・研究者の専門性が価値創造の源泉

## 3. セグメント定義

### セグメント一覧

| ID | セグメント名 | 説明 | 主要ニーズ | 収益貢献 |
|----|--------------|------|------------|----------|
| A | 一般消費者 | 家庭用ビール・飲料の購入者 | 品質、ブランド、価格、健康 | 直接（売上の大部分） |
| B | 業務用顧客 | 飲食店、ホテル、イベント会場 | 安定供給、品質、サポート | 直接 |
| C | 小売業者 | コンビニ、スーパー、量販店 | 売れ筋商品、マージン、販促支援 | 間接（流通チャネル） |
| D | BtoB顧客 | 酵母エキス、食品原料の納入先 | 品質、安定供給、技術サポート | 直接 |

### セグメント間の関係

```
┌──────────────┐
│  一般消費者   │◄──────────────────┐
└──────┬───────┘                   │
       │購買                        │ブランド認知
       ▼                           │
┌──────────────┐      販売      ┌──────────────┐
│  小売業者    │◄──────────────│  アサヒグループ │
└──────────────┘                └───────┬───────┘
                                        │
       ▲                                │供給
       │流通                            ▼
┌──────────────┐      サービス   ┌──────────────┐
│ 業務用顧客   │◄──────────────│  BtoB顧客    │
└──────────────┘   (一部)       └──────────────┘
       │                               │
       └───────────────────────────────┘
              間接的な価値連鎖
```

### ネットワーク効果

| 起点 | 終点 | 効果 |
|------|------|------|
| 消費者需要↑ | 小売取扱↑ | 人気商品は棚スペース拡大 |
| 小売取扱↑ | 消費者アクセス↑ | 購入機会の増加 |
| ブランド認知↑ | 業務用採用↑ | 飲食店でのメニュー採用 |
| 技術力↑ | BtoB信頼↑ | 素材供給先の拡大 |

### VS設計の方向性

#### Layer 1: セグメント別VS
- VS-Consumer: 一般消費者向け価値
- VS-BtoB: 業務用・小売向け価値
- VS-Material: BtoB素材向け価値

#### Layer 2: 統合VS
- VS-Brand: ブランド価値創造（全セグメント横断）
- VS-Innovation: 技術革新（発酵技術、酵母研究）
- VS-Sustainability: サステナビリティ（ESG対応）

## 5. 備考

### 業種固有の特徴
- **季節性**: ビール需要は夏季にピーク
- **規制依存**: 酒税法改正の影響大（2026年税率一本化）
- **ブランド重視**: 長期的なブランド資産構築が競争優位
- **技術蓄積**: 発酵技術は参入障壁として機能

### 注意事項
- 海外事業比率が高く（売上の約50%）、為替影響を受けやすい
- プレミアム化戦略は景気動向に左右される
- 健康志向の高まりによるアルコール離れへの対応が必要
