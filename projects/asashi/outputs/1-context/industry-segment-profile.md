## 2. セグメント構造

### パターン判定

**質問1**: 顧客は何種類いますか？
- [x] 1種類（チャネルは複数だが本質的に単一の消費者市場） → **標準型**
- [ ] 2種類（相互依存） → 両面市場型
- [ ] 3種類以上 → 多面ステークホルダー型

**質問2**: 顧客間に相互依存関係がありますか？
- **回答**: いいえ
- 消費者と取引先（小売店・飲食店）は「相互依存」ではなく「バリューチェーン上の上下関係」
- 取引先が増えても消費者が直接増えるわけではない（間接的な流通拡大）
- 人材派遣のような「スタッフが増えるとクライアントが増える」双方向効果は存在しない

**質問3**: 内部ステークホルダー（従業員等）はサービス品質に直接影響しますか？
- **回答**: 間接的影響
- 製造業では工場作業員・研究者が品質に影響するが、サービス業のような直接的顧客接点は少ない
- 品質は製品・プロセスに埋め込まれ、個人の対応力が価値を左右する割合は低い

### 判定結果

**パターン**: **標準型**

**理由**:
1. 最終的な価値受取者は「消費者」という単一セグメント
2. B2B取引先（小売・飲食）は流通チャネルであり、独立した価値創造対象ではない
3. 従業員は間接的貢献者であり、地方銀行のような直接的サービス提供者ではない

## 3. セグメント定義

### セグメント一覧

| ID | セグメント名 | 説明 | 主要ニーズ | 収益貢献 |
|----|--------------|------|------------|----------|
| A | **消費者（B2C）** | 最終消費者（家庭内消費・外食時消費） | 品質・味・健康・利便性・価格 | **直接** |
| - | 取引先（B2B）* | 小売店・飲食店・卸売業者 | 品揃え・利益率・サポート・物流 | 間接（チャネル） |

*取引先はセグメントではなくチャネルとして扱う

### セグメント間の関係

```
┌──────────────────────────────────────────────────┐
│                  消費者（B2C）                    │
│    家庭内消費者 ← → 外食・レジャー消費者          │
└──────────────────────────────────────────────────┘
                         ↑
                         │ 製品・ブランド価値
                         │
┌──────────────────────────────────────────────────┐
│              Asahi Group Holdings               │
│  酒類事業 | 飲料事業 | 食品事業                  │
└──────────────────────────────────────────────────┘
                         │
                         │ 流通・販売
                         ↓
┌──────────────────────────────────────────────────┐
│              取引先チャネル（B2B）               │
│  コンビニ | スーパー | 飲食店 | 卸売             │
└──────────────────────────────────────────────────┘
```

### ネットワーク効果

| 起点 | 終点 | 効果 |
|------|------|------|
| 消費者認知↑ | 取引先採用↑ | ブランド力が棚確保を促進 |
| 取引先網↑ | 消費者接点↑ | 流通拡大が購入機会を増加 |

※ただし、これは両面市場の「ネットワーク効果」ではなく、通常のバリューチェーン効果

## 5. 備考

### 業種固有の特徴

1. **ブランド価値の重要性**: 消費財では機能差別化が困難なため、ブランド・情緒的価値が競争優位
2. **グローバル展開**: 日本市場の成熟により、海外M&A（Peroni、Grolsch、Pilsner Urquell）が成長ドライバー
3. **健康志向対応**: 低アルコール・ノンアルコール・機能性飲料への需要シフト
4. **サステナビリティ**: 環境負荷低減（CO2削減、水使用効率、リサイクル）が経営課題

### 注意事項

1. **B2Bは別セグメントとしない**: 取引先は流通チャネルであり、独立した価値創造対象ではない
2. **地域別分析の必要性**: 日本/欧州/オセアニア/アジアで市場特性が異なる
3. **事業ポートフォリオ**: 酒類・飲料・食品の3事業を横断的に捉える必要あり

### 既存Phase 2との整合性

既存の `outputs/2-value/` 成果物（VS0-VS7）は標準型パターンで作成されており、本プロファイルと整合。追加の再構成は不要。
