# Strategic Domain Classification (CL1)

**プロジェクト:** asashi (アサヒグループホールディングス)
**作成日:** 2025-11-26
**ステータス:** Phase 3a 完了
**分析モード:** zen-architect (ANALYZE mode)

---

## エグゼクティブサマリー

アサヒグループホールディングスの8つのValue Streams（VS0-VS7）を戦略的重要性・競争優位性の観点から分析し、Core/Supporting/Genericに分類しました。

**分類結果:**
- **Core Domains（2）**: VS2, VS3 - 競争優位の源泉、最優先投資
- **Supporting Domains（3）**: VS1, VS4, VS5 - ビジネス運営の基盤
- **Generic Domains（3）**: VS0, VS6, VS7 - 汎用機能、外部活用推奨

**投資配分推奨:** Core 50% / Supporting 35% / Generic 15%

---

## ドメイン分類サマリー

### Core Domains（競争優位の源泉）

| VS | 名称 | 分類理由 |
|----|------|----------|
| **VS2** | 製品開発・イノベーション | 100年以上の醸造技術・酵母研究が競争優位の根幹。模倣困難な独自技術資産。 |
| **VS3** | ブランディング・マーケティング | プレミアムブランドポートフォリオと「グローカル戦略」が収益性を牽引。 |

### Supporting Domains（ビジネス支援）

| VS | 名称 | 分類理由 |
|----|------|----------|
| **VS1** | 市場機会発見・インサイト獲得 | Core Domains (VS2/VS3)への重要インプット。市場理解が製品・ブランド戦略を支える。 |
| **VS4** | 販売・流通・サプライチェーン | ビジネス運営に必須。効率性が収益性に直結するが、業界標準プラクティスで対応可能。 |
| **VS5** | カスタマーサポート・品質保証 | 食品・飲料業界では品質・安全が信頼の基盤。法規制遵守は必須。 |

### Generic Domains（汎用機能）

| VS | 名称 | 分類理由 |
|----|------|----------|
| **VS0** | ビジョン策定・戦略立案 | 全業界共通の経営機能。標準的な戦略フレームワーク・外部コンサル活用可能。 |
| **VS6** | データ活用・DX | 全業界共通のIT/DX課題。クラウドサービス・SaaS活用が効率的。 |
| **VS7** | 業務改善・オペレーショナルエクセレンス | リーン・Six Sigma等の標準手法。業界横断でベストプラクティス適用可能。 |

## 詳細分析

### VS0: ビジョン策定・戦略立案

**分類:** Generic Domain

#### 5軸評価

| 評価軸 | スコア | 根拠 |
|--------|--------|------|
| 戦略的重要性 | ★★★☆☆ | 方向性決定に重要だが、日常業務への直接影響は限定的 |
| 競争優位性 | ★★☆☆☆ | 戦略立案能力自体は差別化要因にならない |
| 複雑度 | ★★★☆☆ | 標準的な戦略フレームワークで対応可能 |
| カスタマイズ必要性 | ★★☆☆☆ | 業界共通のアプローチ適用可能 |
| 外部サービス代替可能性 | ★★★★☆ | 外部コンサル・戦略ツール活用可能 |

#### 推奨アプローチ
- 外部コンサルティングファームの活用
- 標準的な戦略策定ツール（BIダッシュボード等）の導入
- 経営陣の意思決定に集中し、分析作業は外部化

### VS2: 製品開発・イノベーション

**分類:** Core Domain ★★★★★

#### 5軸評価

| 評価軸 | スコア | 根拠 |
|--------|--------|------|
| 戦略的重要性 | ★★★★★ | 「期待を超えるおいしさ」実現の中核。全ての価値の源泉 |
| 競争優位性 | ★★★★★ | 100年以上の醸造技術・独自酵母株は模倣困難 |
| 複雑度 | ★★★★★ | 発酵技術・酵母研究は高度な専門性が必要 |
| カスタマイズ必要性 | ★★★★★ | 完全内製・自社固有の実装が必須 |
| 外部サービス代替可能性 | ★☆☆☆☆ | 外部代替不可。競争優位の核心 |

#### 競争優位性の根拠

**技術的優位性:**
- 100年以上の醸造技術蓄積
- 独自の酵母株・発酵プロセス
- 「Asahi Super Dry」辛口製法（唯一無二）
- ノンアルコールビール技術（業界リード）

**知的財産:**
- 発酵・酵母関連特許
- 製法ノウハウ（暗黙知含む）
- 長年の研究データ蓄積

#### 推奨アプローチ
- **フルカスタム開発**: 自社R&D部門による完全内製
- **最高品質投資**: R&D投資比率3%以上維持
- **企業秘密管理**: 技術・ノウハウの厳重管理
- **人材確保**: 発酵技術・食品科学の専門人材育成

### VS3: ブランディング・マーケティング

**分類:** Core Domain ★★★★★

#### 5軸評価

| 評価軸 | スコア | 根拠 |
|--------|--------|------|
| 戦略的重要性 | ★★★★★ | ブランド価値が売上・利益率に直結。プレミアム戦略の実現手段 |
| 競争優位性 | ★★★★★ | グローカル戦略・プレミアムブランドポートフォリオが差別化 |
| 複雑度 | ★★★★☆ | グローバル×ローカルのバランス調整は高度 |
| カスタマイズ必要性 | ★★★★★ | 自社ブランドの表現は完全カスタム必須 |
| 外部サービス代替可能性 | ★★☆☆☆ | 広告制作は外部活用可、戦略・ブランド管理は内製必須 |

#### 競争優位性の根拠

**ブランドポートフォリオ:**
- **グローバルブランド**: Asahi Super Dry
- **欧州プレミアム**: Pilsner Urquell, Peroni, Grolsch
- **地域ブランド**: 各国ローカル人気ブランド

**グローカル戦略:**
- ローカル市場での独自ブランド育成
- グローバルブランドの現地適応
- 3地域（日本・東アジア、欧州、アジアパシフィック）での展開

#### 推奨アプローチ
- **ブランド戦略**: 内製（CMO/ブランドマネージャー主導）
- **クリエイティブ制作**: 外部エージェンシー活用（但し戦略は内製）
- **デジタルマーケティング**: ハイブリッド（戦略内製+運用一部外部）
- **消費者エンゲージメント**: 自社プラットフォーム構築

## 投資配分推奨

### 推奨投資比率

| ドメイン | 投資比率 | 対象VS | 投資方針 |
|----------|----------|--------|----------|
| **Core** | 50% | VS2, VS3 | フルカスタム開発、最高品質投資 |
| **Supporting** | 35% | VS1, VS4, VS5 | ベストプラクティス+カスタマイズ |
| **Generic** | 15% | VS0, VS6, VS7 | SaaS/パッケージ活用、外部サービス |

### 投資比率の根拠

**Core Domain（50%）への集中投資理由:**
1. **競争優位の源泉**: VS2の醸造技術・VS3のブランド力がアサヒグループの差別化要因
2. **模倣困難性**: 100年以上の蓄積は短期では追いつけない
3. **収益性への直結**: プレミアム戦略がマージン向上を牽引
4. **長期的投資効果**: 技術・ブランドは時間とともに価値が蓄積

**Supporting Domain（35%）への適切な投資:**
1. **Core Domain支援**: VS1のインサイトがVS2/VS3の品質を向上
2. **運営効率**: VS4の効率がコスト競争力に影響
3. **信頼基盤**: VS5の品質保証が食品・飲料業界での必須条件

**Generic Domain（15%）での効率化:**
1. **コスト最適化**: 外部サービス活用で固定費削減
2. **スピード**: SaaS活用で導入期間短縮
3. **リソース集中**: 汎用機能に人材を割かず、Core Domainに集中

### 予算配分イメージ（例: IT投資100億円の場合）

| ドメイン | 配分額 | 主要用途 |
|----------|--------|----------|
| Core (50%) | 50億円 | R&Dシステム、ブランド管理、デジタルマーケティング基盤 |
| Supporting (35%) | 35億円 | CRM、サプライチェーン最適化、品質管理システム |
| Generic (15%) | 15億円 | クラウド基盤、SaaSライセンス、業務改善ツール |

## 戦略的推奨事項

### 1. Core Domain（VS2/VS3）への最優先投資

**推奨施策:**
- R&D投資比率を売上高の3%以上に維持・拡大
- デジタルツール活用による製品開発サイクル短縮
- グローバルブランドマネジメントプラットフォーム構築
- 消費者エンゲージメント強化（D2C、CDP活用）

**期待効果:**
- 新製品開発サイクル20%短縮
- ブランドエクイティスコア向上
- プレミアム製品比率向上による収益性改善

### 2. Supporting Domainの効率化と高度化

**推奨施策:**
- AI需要予測導入（VS4）による在庫最適化
- 品質トレーサビリティのデジタル化（VS5）
- 市場インサイトのリアルタイム化（VS1）

**期待効果:**
- 在庫コスト10%削減
- 品質クレーム対応時間50%短縮
- 市場機会の早期発見率向上

### 3. Generic Domainの外部サービス活用

**推奨施策:**
- クラウドファーストポリシーの徹底（VS6）
- 業務改善のSaaS化・RPA導入（VS7）
- 戦略立案ツールの標準化（VS0）

**期待効果:**
- IT運用コスト30%削減
- システム導入期間50%短縮
- 社内IT人材のCore Domain支援へのシフト

### 4. ドメイン間連携の強化

**推奨施策:**
- VS1→VS2→VS3のデータ連携強化（消費者インサイト→製品開発→マーケティング）
- VS5→VS2のフィードバックループ構築（品質クレーム→製品改良）
- VS6をイネーブラーとして全VSのデジタル化支援

**期待効果:**
- 市場投入までの時間（Time to Market）短縮
- 顧客満足度向上
- 組織横断の意思決定スピード向上

### 5. 人材配置の最適化

**推奨施策:**
- Core Domain（VS2/VS3）への専門人材集中配置
- Generic Domain（VS6/VS7）は外部パートナー・SIer活用
- Supporting Domainは社内ベテラン+外部ハイブリッド

**期待効果:**
- 専門性の深化（Core Domain）
- コスト効率（Generic Domain）
- ナレッジ継承（Supporting Domain）
