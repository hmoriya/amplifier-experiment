## 業種・セグメント情報

**業種:** 製造業・消費財（食品・飲料業界）
**セグメントパターン:** 標準型
**参照:** `outputs/1-context/industry-segment-profile.md`

### パターン選定理由

アサヒグループは最終消費者（B2C）を主要顧客とする消費財メーカーです。
B2B取引先（小売・飲食）は流通チャネルであり、人材派遣や不動産のような
「両面市場」ではありません。そのため、標準的なVS0-VS7パターンを適用します。

### VSパターン構成

```
標準型: VS0-VS7（8つのValue Stream）
├── VS0: ビジョン策定・戦略立案 (Generic)
├── VS1: 市場機会発見・インサイト獲得 (Supporting)
├── VS2: 製品開発・イノベーション (Core) ★
├── VS3: ブランディング・マーケティング (Core) ★
├── VS4: 販売・流通・サプライチェーン (Supporting)
├── VS5: カスタマーサポート・品質保証 (Supporting)
├── VS6: データ活用・DX (Generic)
└── VS7: 業務改善・オペレーショナルエクセレンス (Generic)

★ = Core Domain（競争優位性の源泉）
```

## 1. ビジョン（将来像）

### グローバルビジョン

**「ビール中心の酒類、飲料、食品を世界で展開するリーディングカンパニー」**

### ビジョンの要素

#### 1. グローバルリーダーシップ
- 世界100カ国・地域以上での事業展開
- グローバルTop 5を目指す飲料企業
- 各地域でのNo.1またはTop 3ポジション

#### 2. グローカル戦略の実践
- **ローカルプレミアム化**: 各国で競争力の高い独自ブランドを展開
- **グローバルブランド拡大**: Asahi Super Dry等の世界展開
- 地域特性とグローバル標準の最適バランス

#### 3. 多角的ポートフォリオ
- ビールを中核としつつ、多様な製品カテゴリーへ展開
- 酒類、ノンアルコール飲料、食品・サプリメント
- 健康志向・機能性製品への注力

#### 4. サステナブルな成長
- 環境・社会課題の解決と事業成長の両立
- カーボンニュートラルの実現
- サーキュラーエコノミーの推進

## 2. ミッション（使命）

### コアミッション

**「期待を超えるおいしさ、楽しい生活文化の創造」**

### ミッションの意味

#### 「期待を超えるおいしさ」
- 単なる製品提供ではなく、驚きと感動を届ける
- 独自の発酵技術と醸造技術による差別化
- 継続的なイノベーションと品質向上
- 地域の嗜好に合わせた製品開発

#### 「楽しい生活文化の創造」
- 製品を通じた豊かな生活体験の提供
- 社交、くつろぎ、祝祭などの場面の演出
- 責任ある飲酒文化の醸成
- コミュニティとのつながり創出

### ミッションの実践

**製品・サービス:**
- 最高品質の製品開発
- 多様なニーズへの対応
- 継続的なイノベーション

**顧客体験:**
- 期待を超える満足度
- 記憶に残る体験の提供
- ブランドとの情緒的つながり

**社会貢献:**
- 適正飲酒の啓発
- 地域文化の尊重と振興
- 環境・社会課題への取り組み

## 3. 企業スローガン

**「Make the world shine～おいしさと楽しさで世界に輝きを～」**

### スローガンの意味

#### 「Make the world shine」
- 世界をポジティブに変革する意志
- 明るさ、喜び、希望をもたらす
- グローバル企業としての責任と影響力

#### 「おいしさと楽しさ」
- 製品価値の2つの柱
- 感覚的満足（おいしさ）と情緒的満足（楽しさ）
- 両立による総合的な顧客価値

#### 「世界に輝きを」
- グローバル規模での価値提供
- 社会全体への貢献
- 持続可能な未来への寄与

## 4. コアバリュー（価値観）

### 1. 企業理念の実践（Asahi Group Philosophy）

**基本姿勢:**
グループ活動の根幹に企業理念（AGP）を据え、すべての意思決定と行動の基準とする

**実践内容:**
- 企業理念に基づく経営判断
- 全従業員への理念の浸透
- 理念と日常業務の連動
- ステークホルダーとの価値共有

### 2. サステナビリティと経営の統合

**基本姿勢:**
サステナビリティを経営戦略の中核に位置づけ、社会・環境価値と経済価値を同時に創出

**実践内容:**
- 環境目標の設定と達成（CO2削減、水資源管理等）
- 社会課題解決型ビジネスの推進
- ESG経営の実践
- ステークホルダーとの協働

### 3. 社会課題解決への貢献

**基本姿勢:**
事業活動を通じて社会課題を解決し、社会・事業の双方にプラスインパクトを創出

**実践内容:**
- 責任ある飲酒の啓発
- 地域コミュニティとの共生
- ダイバーシティ&インクルージョン
- 健康・ウェルネスへの貢献

### 4. イノベーションと品質へのこだわり

**基本姿勢:**
「期待を超えるおいしさ」を実現するための継続的なイノベーションと妥協なき品質追求

**実践内容:**
- R&D投資の継続
- 独自技術の開発（発酵、醸造等）
- 品質保証体制の強化
- 顧客フィードバックの製品開発への反映

### 5. 多様性の尊重と人材育成

**基本姿勢:**
「社員一人ひとりが個性を発揮しながら、互いに尊重し合うカルチャー」の実現

**実践内容:**
- ダイバーシティ&インクルージョンの推進
- グローバル人材の育成
- キャリア開発支援
- 働きがいのある職場づくり

### 6. ステークホルダーとの対話

**基本姿勢:**
透明性の高い情報開示と、すべてのステークホルダーとの建設的な対話

**実践内容:**
- 統合報告書での包括的情報開示
- ステークホルダーダイアログの実施
- 双方向コミュニケーション
- フィードバックの経営への反映
