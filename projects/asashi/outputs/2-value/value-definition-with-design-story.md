## 🎯 設計ストーリー：なぜこの構造なのか

本ドキュメントは、価値分解・バックキャスティング・バリューステージ設計の**設計判断の理由**を明示したV5形式の価値定義です。

## 1. 価値分解の理由（VL1→VL2→VL3）

### 1.1 VL1（最上位価値）の選定理由

**選定した価値**: 「期待を超えるおいしさと楽しい生活文化の創造」

**なぜこの価値なのか**:

| 観点 | 理由 |
|------|------|
| **企業理念との整合** | コーポレートスローガン「Cheers to a Brighter Tomorrow」の本質を表現 |
| **上位目的** | 単なる「ビールを売る」ではなく、**生活の豊かさを提供する**という価値創造企業としての定義 |
| **歴史的強み** | 100年以上の醸造技術の蓄積が「おいしさ」という言葉に凝縮 |
| **差別化ポイント** | 「期待を超える」はスーパードライの「キレ」の原点。競合との差別化の核心 |

---

### 1.2 VL2グループ分けの理由

消費者が「おいしい」「この会社の製品を選びたい」と感じる要素を逆算して4軸に分解しました。

```
消費者が「アサヒを選ぶ」瞬間
    ↑ 何が影響している？

┌─────────────────────────────────────────────────────────┐
│ VL2-1: 製品イノベーション価値                           │
│   「新しい味覚体験がある」「他社にない製品がある」      │
│   → 選ぶ理由の核心。Coreドメインの根拠                  │
├─────────────────────────────────────────────────────────┤
│ VL2-2: 品質価値                                         │
│   「いつも同じ味」「安心して飲める」                    │
│   → 選び続ける理由。Table Stakes                        │
├─────────────────────────────────────────────────────────┤
│ VL2-3: ブランド体験価値                                 │
│   「このブランドが好き」「ライフスタイルに合う」        │
│   → 感情的なつながり。Coreドメインの根拠                │
├─────────────────────────────────────────────────────────┤
│ VL2-4: 持続可能性価値                                   │
│   「罪悪感なく楽しめる」「環境に配慮している」          │
│   → 現代消費者の必須要件。今後の成長領域                │
└─────────────────────────────────────────────────────────┘
```

**分解軸の選定理由**:

| 軸 | 選定理由 | 戦略的位置づけ |
|----|----------|----------------|
| 製品イノベーション | 飲料・食品業界では「味」が最大の差別化要因 | **Core**（投資集中） |
| 品質 | 信頼の基盤。これがないと選ばれない | Supporting（維持） |
| ブランド体験 | 感情的価値がリピートを生む | **Core**（投資集中） |
| 持続可能性 | 若年層の価値観に対応。将来のCore候補 | Supporting（育成） |

---

### 1.3 VL3詳細化の理由

各VL2を「どうやって実現するか」の視点で分解しました。

#### VL2-1（製品イノベーション）→ VL3への分解

| VL3 | なぜこの分解か | ドメイン示唆 |
|-----|----------------|--------------|
| **VL3-1-1: 発酵技術** | アサヒの**100年の強み**。スーパードライの独自酵母はここから生まれた。他社には真似できない | **Core** |
| **VL3-1-2: 素材研究** | 「キレ」は素材選定から生まれた。原料調達と研究の両面が必要 | **Core** |
| VL3-1-3: 健康価値 | ノンアル、低糖質、機能性飲料は**成長市場**。将来のCore昇格候補 | Supporting→Core |

**分解の深さの判断基準**:
1. 組織構造と一致するか（研究所の部門構成と対応）
2. 投資判断の単位として適切か
3. チーム編成の境界として機能するか

## 2. MSバックキャスティングの理由

### 2.1 MS5（理想状態）の設定理由

**設定した状態**: 「製品イノベーションリーダー」（18ヶ月後）

**なぜ18ヶ月後にこの状態を目指すのか**:

```
外部環境分析からの逆算:
┌─────────────────────────────────────────────────────────┐
│ 競合動向                                               │
│ ・キリン: 健康価値でリード（プラズマ乳酸菌）           │
│ ・サントリー: ウイスキー・RTDでグローバル展開          │
│ → 差別化には「新しい価値創造」が必要                   │
├─────────────────────────────────────────────────────────┤
│ 市場トレンド                                           │
│ ・プレミアムビール市場は価格競争ではなく価値競争       │
│ ・若年層のビール離れ → 新カテゴリ創造が急務            │
│ → イノベーション力が生き残りの鍵                       │
├─────────────────────────────────────────────────────────┤
│ 内部ケイパビリティ                                     │
│ ・発酵研究の蓄積は十分                                 │
│ ・デジタル化による研究加速が可能                       │
│ → 18ヶ月で成果を出せる土台がある                       │
└─────────────────────────────────────────────────────────┘
```

**成功基準の設定理由**:

| 指標 | 目標値 | 設定理由 |
|------|--------|----------|
| 新製品売上比率 | 30%以上 | 現状20%からの伸長。イノベーション企業の業界標準 |
| プレミアムセグメントシェア | 1位 | 既に強い領域をさらに強化。選択と集中 |
| 顧客NPS | +50以上 | 推奨意向がブランド価値を測る最良指標 |

---

### 2.2 逆算の論理（MS5→MS1）

```
MS5「イノベーションリーダー」（18ヶ月後）
    │   成功基準: 新製品売上比率30%、シェア1位、NPS+50
    │
    │ ← 成功製品がグローバル展開されている必要
    ↓
MS4「パイプライン充実」（12ヶ月後）
    │   成功基準: 年間新製品10品、開発リードタイム50%短縮
    │
    │ ← 研究成果を製品化する仕組みが必要
    ↓
MS3「研究知見の蓄積」（9ヶ月後）
    │   成功基準: 酵母ライブラリ500株、研究DB構築完了
    │
    │ ← 体系的な研究基盤が必要
    ↓
MS2「新プロセス稼働」（6ヶ月後）
    │   成功基準: パイロット5件完了、習熟率80%
    │
    │ ← チームの習熟と初期成果が必要
    ↓
MS1「基盤整備」（3ヶ月後）
    成功基準: 研究データ管理システム稼働、チーム編成完了
```

**なぜこの順序か**:
- 研究開発は「インフラ→プロセス→知見→成果→展開」という段階を踏む
- 各段階を飛ばすと「砂上の楼閣」になる
- 3ヶ月単位は四半期レビューと整合し、経営判断のタイミングに合う

## 3. MS→VS変換の理由

### 3.1 組織視点から顧客視点への変換理由

**MSは「組織が達成すべき状態」、VSは「消費者にとって価値ある状態」**

この変換が重要な理由:
- 組織の進捗（研究基盤構築）は消費者には見えない
- 消費者が感じるのは「新しい製品が選べるようになった」という結果
- VS視点で設計することで、**消費者価値を見失わない**

| MS（組織視点） | VS（消費者視点） | 変換の意図 |
|----------------|------------------|------------|
| MS1「研究基盤確立」 | VS1-2「新製品の選択肢が増え始める」 | 消費者は基盤には興味がないが、新しい選択肢には価値を感じる |
| MS3「研究成熟」 | VS4「革新的製品を体験できる」 | 研究の成熟は、革新的製品として消費者に届く |
| MS5「リーダー」 | VS7「環境配慮製品を選択できる」 | リーダー企業は社会的責任も果たしていると期待される |

---

### 3.2 VS粒度の決定理由

8つのVS（VS0-VS7）を採用した理由:

| VS | 対応するMS | 粒度の判断理由 |
|----|-----------|----------------|
| VS0 | 基盤（戦略） | 企業の方向性を理解する。投資家・パートナー向けの価値 |
| VS1 | MS1前半 | 消費者インサイト。ニーズ把握の段階 |
| VS2 | MS1後半-MS2前半 | 製品開発。選択肢が増える段階。**Coreドメイン** |
| VS3 | MS2中盤 | ブランディング。愛着形成の段階。**Coreドメイン** |
| VS4 | MS2後半-MS3前半 | サプライチェーン。安定供給の段階 |
| VS5 | MS3後半-MS4前半 | R&D・イノベーション。革新を体験する段階 |
| VS6 | MS4後半 | 品質・規制。信頼確立の段階 |
| VS7 | MS5 | サステナビリティ。社会的価値の段階 |

**なぜ8つか**:
- 多すぎると管理が複雑、少なすぎると粗い
- 各VSがチーム境界（サブドメイン）と対応可能な粒度
- 四半期ごとの進捗管理に適した数

## 価値定義サマリー

### 価値宣言（VL1）

```yaml
価値宣言:
  ビジョン: 期待を超えるおいしさと楽しい生活文化を創造する

  提供価値:
    顧客への価値: 高品質で革新的な飲料・食品による豊かな生活体験
    社会への価値: 醸造技術を活かした持続可能な食文化の発展
    組織への価値: グローバルプレミアムブランドとしての成長

  価値の独自性:
    差別化要因: 100年以上の醸造技術と発酵研究の蓄積
    競争優位性: プレミアムビール市場でのブランド力
    価値の持続性: 継続的なR&D投資と技術革新
```

### 価値分解構造

```
VL1: 期待を超えるおいしさと楽しい生活文化の創造
│
├── VL2-1: 製品イノベーション価値 [Core]
│   ├── VL3-1-1: 発酵技術による味の革新
│   ├── VL3-1-2: 素材による差別化
│   └── VL3-1-3: 健康価値の付加
│
├── VL2-2: 品質価値 [Supporting]
│   ├── VL3-2-1: 製造品質の一貫性
│   ├── VL3-2-2: 鮮度管理
│   └── VL3-2-3: トレーサビリティ
│
├── VL2-3: ブランド体験価値 [Core]
│   ├── VL3-3-1: プレミアム体験
│   ├── VL3-3-2: 地域・文化との結びつき
│   └── VL3-3-3: 顧客コミュニティ
│
└── VL2-4: 持続可能性価値 [Supporting]
    ├── VL3-4-1: 環境負荷低減
    ├── VL3-4-2: 地域社会への貢献
    └── VL3-4-3: 責任ある飲酒推進
```
