**プロジェクト:** asashi (アサヒグループホールディングス)
**作成日:** 2025-11-25
**ステータス:** 詳細化完了

---

## 1. Value Stream 概要

### 1.1 目的

独自技術を活かした製品開発と継続的イノベーションを通じて、
「期待を超えるおいしさ」を実現し、競争優位性を確立する。

### 1.2 Core Domain としての位置づけ

**なぜCore Domainなのか:**

1. **競争優位性の源泉**
   - 100年以上の醸造技術蓄積
   - 独自の酵母株・発酵プロセス
   - 競合が容易に模倣できない技術資産

2. **差別化の根幹**
   - 「Asahi Super Dry」の辛口製法
   - プレミアムビールの品質
   - ノンアルコール技術

3. **投資優先度: 最高**
   - 自社開発・内製化を推奨
   - 企業秘密として厳重管理

### 1.3 戦略的重要性

```
VS2の成功 → ブランド価値向上(VS3) → 売上・シェア拡大(VS4) → 企業価値向上(VS0)
```

## 5. KPI・測定指標

### 5.1 イノベーション指標

| KPI | 説明 | 目標例 | 測定頻度 |
|-----|------|--------|----------|
| 新製品開発サイクルタイム | コンセプトから発売までの期間 | 12ヶ月以内 | プロジェクト毎 |

| KPI | 説明 | 目標例 | 測定頻度 |
|-----|------|--------|----------|

| 製品回収件数 | リコール発生件数 | 0件 | 年次 |
| 官能評価スコア | 社内評価基準達成率 | 95%以上 | 製品毎 |

## 7. アサヒグループの競争優位性

### 7.1 技術的強み

| 領域 | 強み | 競合比較 |
|------|------|----------|
| 酵母技術 | 100年以上の研究蓄積 | 差別化要因 |
| 発酵技術 | 独自の発酵プロセス | 模倣困難 |
| 辛口製法 | Asahi Super Dryの製法 | 唯一無二 |
| ノンアル技術 | 本格的な味わい実現 | 業界リード |

### 7.2 製品ポートフォリオ

**酒類:**
- **グローバルブランド**: Asahi Super Dry
- **欧州プレミアム**: Pilsner Urquell, Peroni, Grolsch
- **クラフト/地域**: 各国ローカルブランド
- **ノンアルコール**: ドライゼロ等

**飲料・食品:**
- 清涼飲料水
- 乳製品・乳酸菌飲料
- 機能性飲料
- 健康食品・サプリメント

### 7.3 研究開発体制

**研究領域:**
1. **基盤技術**: 分析、生産、容器・包装、環境
2. **素材探索**: 酵母、乳酸菌、枯草菌、その他微生物
3. **商品開発**: 新製品、既存品改良、プロセス革新

**推定R&D投資:** 売上高の約2-3%（業界平均並み）

2. **オープンイノベーション**
