## 1. 分類フレームワーク

### 1.1 Core/Supporting/Generic定義

| 分類 | 定義 | 投資方針 | 内製/外注 |
|------|------|----------|-----------|
| **Core** | 競争優位の源泉、差別化に直接貢献 | 積極投資 | 内製重視 |
| **Supporting** | Coreを支援、事業運営に必須 | 効率化投資 | 状況に応じて |
| **Generic** | 汎用的、外部調達可能 | 最適化 | 外注可能 |

### 1.2 サントリー固有の分類基準

### 2.2 A/B/C判定との対応関係

| ドメイン | A/B/C判定 | Core/Supporting/Generic | 理由 |
|----------|-----------|-------------------------|------|
| R&D基盤研究 | C判定（45%） | **Core** | 水科学・発酵は競争優位の源泉 |
| ブランド統括 | - | **Core** | 「水と生きる」は差別化の中核 |
| マーケティング | B判定（30%） | **Core** | 事業ごとに顧客接点で差別化 |
| 製造管理 | B判定（25%） | **Core** | 製造技術が品質・コスト競争力 |
| サステナビリティ | A判定（85%） | **Core** | ESG経営の差別化要因 |
| サプライチェーン | C判定（55%） | Supporting | 効率化重要だが差別化ではない |
| 顧客管理 | C判定（40%） | Supporting | CRM/CDPはCore支援 |
| コンプライアンス | C判定（60%） | Supporting | 必須だが差別化要因ではない |
| R&D応用研究 | C判定（45%） | Supporting | 製品開発は事業支援 |
| IT基盤 | A判定（90%） | Generic | 統合効率が重要、差別化ではない |
| 財務・経理 | A判定（85%） | Generic | 標準化・効率化優先 |
| 人事・組織 | A判定（80%） | Generic | 制度統一が主目的 |
