### 2.2 各軸の詳細評価

#### 【軸1】VL1統一可能性（3/3点）

**評価**: 全事業で「水と生きる」を統一価値として設定可能

| スコア | 基準 | サントリーの状況 |
|--------|------|------------------|
| 0点 | 全く異なる価値訴求 | - |
| 1点 | 抽象化すれば共通可能 | - |
| 2点 | 自然に共通価値を設定可能 | - |
| **3点** | **既に共通価値が企業理念として浸透** | **「水と生きる」が全社で浸透済み** |

**根拠**:

```
「水と生きる SUNTORY」
  │
  ├── 自然との共生
  │   ・天然水の森（全国21,000ha）
  │   ・水源涵養活動
  │   ・生物多様性保全
  │
  ├── 社会への貢献
  │   ・水育（みずいく）活動
  │   ・地域社会との共創
  │   ・文化・芸術支援
  │
  └── 人材の活躍
      ・「やってみなはれ」精神
      ・ダイバーシティ推進
      ・グローバル人材育成
```

**事業別の「水と生きる」適用**:

| 事業 | VL1との接続 | 統一性 |
|------|------------|--------|
| **酒類（ウイスキー）** | 「水と生きる」が品質の源泉（仕込み水） | 自然 |
| **酒類（ビール/RTD）** | 天然水仕込み、醸造への水の重要性 | 自然 |
| **飲料** | 天然水そのもの、「水」が製品価値 | 完全一致 |
| **健康食品** | 生命の源である水と健康 | 自然 |
| **外食** | 水と食の融合、食文化創造 | 自然 |

**スコア: 3点** - 企業理念として既に全社浸透しており、最高評価

#### 【軸4】価値トレーサビリティ複雑度（2/3点）

**評価**: Operation→VS→VL1への追跡は中程度の複雑性

| スコア | 基準 | サントリーの状況 |
|--------|------|------------------|
| 0点 | シンプルに追跡可能 | - |
| 1点 | 一部複雑だが管理可能 | - |
| **2点** | **複雑で分割管理が必要** | **4事業領域で中程度に複雑** |
| 3点 | 極めて複雑で完全分離が必要 | - |

**根拠**:


### 3.2 適合/不適合判定

| 観点 | 判定 | 詳細 |
|------|------|------|
| **VL1統一性** | **適合** | 「水と生きる」は既に企業理念として浸透、全事業統合に適する |
| **ジャーニー統一性** | **不適合** | VS4以降の顧客ジャーニーが事業で大きく異なる |
| **Cap活用** | **部分適合** | 共通技術基盤は強いが、応用は事業固有 |
| **管理可能性** | **部分適合** | 単一VStrでの管理は複雑、階層化が必要 |
| **評価統一性** | **不適合** | 事業固有KPIが多く、統一評価体系は困難 |

### 3.3 総合判定

**判定: 部分適合（Partially Fit）**

価値軸パターン単独での採用は推奨しない。以下の理由による：

1. **強み: VL1の統一性は極めて高い**
   - 「水と生きる」は全事業を貫く共通価値として機能
   - 企業ブランドとしての訴求力が高い
   - ESG/サステナビリティとの整合性も優れる

2. **弱み: 事業固有性への対応が困難**
   - ウイスキー（長期熟成、プレミアム）と飲料（即時消費、大量販売）の性質差
   - 健康食品（機能訴求、継続購入）と外食（来店体験）の性質差
   - 単一VStrでは事業固有の価値創造プロセスが埋もれる

## 5. 推奨事項

### 5.1 価値軸パターン単独採用の場合

**条件付きで採用可能**（ただし以下の対応が必要）:

1. **事業サブVStrの設計**
   - 価値軸VS0-VS7の下に、事業固有のサブステージを設計
   - 例: VS4（ブランド・マーケティング）配下に「ウイスキー向けVS4」「飲料向けVS4」を定義

2. **評価体系の二層化**
   - 全社共通: VL1貢献度（サステナビリティKPI、ブランド想起）
   - 事業固有: 事業KPI（売上、シェア、継続率等）

3. **ガバナンス設計**
   - VS0/VS1/VS7は全社統括
   - VS2-VS6は事業部主導（全社方針との整合確認）
