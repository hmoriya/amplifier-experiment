# サントリーホールディングス - 価値定義

## パーパス（Purpose）

> 「人と自然と響きあい、豊かな生活文化を創造し、『人間の生命の輝き』をめざす」

## コーポレートメッセージ

> 「水と生きる SUNTORY」

## 創業精神（DNA）

### やってみなはれ
失敗を恐れることなく、新しい価値の創造をめざし、あきらめずに挑み続ける

### 利益三分主義
事業で得た利益は：
1. 自社への再投資
2. お客様へのサービス向上
3. 社会への還元
