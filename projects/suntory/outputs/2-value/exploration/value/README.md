# 価値軸探索 - サントリー

## 目的

全社統合型のバリューストリーム（1本）を設計する。

## 設計方針

- **VStr数**: 1本
- **目的**: ブランド統一・全社一貫性
- **適用場面**: 強力なブランド統一が必要な場合

## 作業手順

```bash
# このworktreeで作業開始
/parasol:2-value VS0  # まずVS0から設計
```

## 統合VStrの構造

```
サントリーグループ統合VStr
    │
    VS0: ビジョン・戦略
    VS1: 市場分析
    VS2: 顧客体験
    VS3: オペレーション
    VS4: イノベーション
    VS5: リスク・コンプライアンス
    VS6: パートナーシップ
    VS7: サステナビリティ
```
