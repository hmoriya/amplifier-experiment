## 3. 軸別の主要発見

### 3.1 価値軸（統合型）の発見

**6ステージ構造**:
```
認知 → 検討 → 購入 → 体験 → 継続 → 推奨
```

**L1優先ケーパビリティ**:
- CDP/CRM基盤（最優先）- 全事業顧客ID統合
- デジタルマーケティング基盤
- EC/D2C基盤

**強み**: ブランド統一性、クロスセル最大化
**課題**: 意思決定の遅延、事業特性の希薄化

### 3.2 事業部軸（独立型）の発見

**5事業の構成比**:
```
飲料: 49%  スピリッツ: 25%  ビール: 12%  健康食品: 5%  ワイン: 4%
```

**事業別特性**:
| 事業 | 製造特性 | チャネル特性 | リードタイム |
|------|----------|-------------|-------------|
| 飲料 | 大量自動化 | CVS/SM/自販機 | 短 |
| スピリッツ | 蒸留・熟成 | 酒販店/免税 | 5-30年 |
| ビール | 醸造・鮮度 | 飲食店/小売 | 中 |
| ワイン | 醸造・熟成 | 専門店 | 長 |
| 健康 | GMP製造 | D2C/EC | 中 |

**強み**: 事業最適化、意思決定速度
**課題**: シナジー喪失、重複投資

### 3.3 フュージョン型の発見

**「水と生きる」価値統合軸**:
- 水の恵み: 全事業に共通する水資源
- 発酵・醸造の知恵: 技術的シナジー
- 健康への貢献: ウェルネス視点

**4つの共通ケーパビリティ**:
1. R&D基盤（水科学、発酵研究）
2. ブランド統括（「水と生きる」）
3. サステナビリティ（2050目標）
4. IT/DX基盤（CDP、共通PF）

**強み**: シナジーと独立性の両立
**課題**: 調整コスト、責任分界

### 3.4 ケーパビリティ統合型の発見

**A/B/C判定結果**:
- **A判定（統合）**: IT基盤(90%)、財務経理(85%)、サステナ(85%)、人事(80%)
- **B判定（個別）**: マーケティング(30%)、製造管理(25%)
- **C判定（ハイブリッド）**: コンプラ(60%)、SCM(55%)、R&D(45%)、CRM(40%)

**強み**: 理論的最適、DX投資効率化
**課題**: 初期設計コスト、判定の客観化

### 4.2 推奨理由

| # | 推奨理由 | 詳細 |
|---|----------|------|
| 1 | 組織整合性 | サントリーHDの実態（HD+事業会社）と自然に整合 |
| 2 | シナジー効果 | 「水と生きる」を軸にR&D・ブランド・サステナを共有 |
| 3 | 事業独立性 | マーケ・製造は事業特性を最大化 |
| 4 | 段階的実装 | A判定から優先的にDX推進、段階移行可能 |
| 5 | 変革適応力 | 新規事業追加、M&A統合が容易 |
