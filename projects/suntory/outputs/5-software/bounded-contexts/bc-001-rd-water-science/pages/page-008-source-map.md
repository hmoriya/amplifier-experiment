## コンポーネント

| コンポーネント | 種類 | 説明 |
|----------------|------|------|
| 地図 | Map (OSM/Google) | インタラクティブ地図 |
| マーカー | Marker | 水源位置表示 |
| クラスター | MarkerCluster | ズームアウト時の集約 |
| レイヤー切替 | LayerControl | 表示レイヤー選択 |
| フィルタ | FilterPanel | 種別・状態フィルタ |
| 情報パネル | InfoPanel | 水源詳細ポップアップ |
| ズームコントロール | ZoomControl | ズーム操作 |
| 凡例 | Legend | マーカー凡例 |

## データ表示

| データ項目 | ソース | 表示形式 |
|------------|--------|----------|
| 水源位置 | WaterSource[] | マーカー |
| 水源詳細 | WaterSource Entity | ポップアップ |
| 最新検査 | WaterQualityTest | テキスト |
| 涵養面積 | 集計値 | テキスト |
| 涵養活動 | ヒートマップレイヤー | 色分け表示 |

## アクション

| アクション | トリガー | 結果 |
|------------|----------|------|
| ズーム/パン | マウス/タッチ | 地図移動 |
| マーカークリック | クリック | 情報パネル表示 |
| レイヤー切替 | トグル | 表示レイヤー変更 |
| フィルタ適用 | 選択 | マーカーフィルタ |
| 詳細を見る | リンククリック | 水源詳細画面遷移 |
| 全画面 | ボタンクリック | フルスクリーン表示 |
