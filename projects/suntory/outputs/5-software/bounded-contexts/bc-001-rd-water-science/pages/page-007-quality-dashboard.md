## コンポーネント

| コンポーネント | 種類 | 説明 |
|----------------|------|------|
| KPIカード | StatCard x 4 | 主要指標表示 |
| 期間フィルタ | Dropdown | 表示期間選択 |
| トレンドグラフ | LineChart | 合格率推移 |
| カテゴリ別グラフ | BarChart | カテゴリ別検査数 |

| レポート出力 | Button | PDF/Excel出力 |

## データ表示

| データ項目 | ソース | 表示形式 |
|------------|--------|----------|
| 検査合格率 | 集計値 | パーセント+増減 |
| 月間検査数 | 集計値 | 件数+増減 |
| アラート数 | 集計値 | 件数 |
| トレンド | 時系列集計 | 折れ線グラフ |
| カテゴリ別 | グループ集計 | 棒グラフ |
| アラート | Alert[] | テーブル |

| アクション | トリガー | 結果 |
