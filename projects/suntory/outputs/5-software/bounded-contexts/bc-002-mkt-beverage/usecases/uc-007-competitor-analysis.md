# UC-007 競合分析 AnalyzeCompetitors

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-007 |
| 名前 | 競合分析 |
| アクター | 市場調査アナリスト |
| 所属Operation | OP-008 競合分析 |

## 説明

競合他社の動向を分析し、自社の競争ポジションを評価する。

## 事前条件

- 競合情報が収集されている
- 分析対象競合が定義されている

## 事後条件

- 競合分析レポートが作成される
- 競合プロファイルが更新される

## 基本フロー

1. アナリストが競合分析画面を開く
2. アナリストが分析対象競合を選択する
3. システムが収集済み競合情報を表示する
4. アナリストが製品ポートフォリオを比較する
5. アナリストが価格戦略を分析する
6. アナリストがマーケティング活動を分析する
7. アナリストがSWOT分析を実施する
8. システムが競争ポジションマップを生成する
9. アナリストが推奨アクションを記述する
10. アナリストがレポートを作成・共有する

## ビジネスルール適用

- BR-014: 公開情報のみを使用
- BR-015: 主要競合5社は必須分析
