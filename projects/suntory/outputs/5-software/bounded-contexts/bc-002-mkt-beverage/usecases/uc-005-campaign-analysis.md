# UC-005 キャンペーン効果分析 AnalyzeCampaignEffect

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-005 |
| 名前 | キャンペーン効果分析 |
| アクター | マーケティングアナリスト |
| 所属Operation | OP-005 キャンペーン分析 |

## 説明

キャンペーン終了後に効果を分析し、ROIを算出してレポートを作成する。

## 事前条件

- キャンペーンが終了している（または中間分析の場合は実行中）
- パフォーマンスデータが収集されている

## 事後条件

- 効果分析レポートが作成される
- インサイトがナレッジベースに蓄積される

## 基本フロー

1. アナリストがキャンペーン分析画面を開く
2. システムが分析対象キャンペーン一覧を表示する
3. アナリストが対象キャンペーンを選択する
4. システムが収集済みデータを表示する
5. アナリストが分析期間と比較基準を設定する
6. システムがKPI実績を計算・表示する
7. アナリストがチャネル別分析を実行する
8. システムがアトリビューション分析結果を表示する
9. アナリストがROIを算出する
10. アナリストがインサイトと推奨事項を記述する
11. アナリストがレポートを生成・共有する

## 代替フロー

### 4a. データ欠損

4a1. システムが欠損データを警告する
4a2. アナリストが利用可能データで分析続行を選択する
4a3. 基本フロー5に進む

## ビジネスルール適用

- BR-010: マルチタッチアトリビューションモデル適用
- BR-011: 過去キャンペーンとの比較必須
