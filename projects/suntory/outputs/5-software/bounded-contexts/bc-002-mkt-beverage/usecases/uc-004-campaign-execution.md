# UC-004 キャンペーン実行管理 ManageCampaignExecution

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-004 |
| 名前 | キャンペーン実行管理 |
| アクター | マーケティング担当者、キャンペーンオペレーター |
| 所属Operation | OP-004 キャンペーン実行 |

## 説明

承認済みキャンペーンの実行を管理し、リアルタイムでパフォーマンスを監視する。

## 事前条件

- キャンペーン計画が承認済み
- クリエイティブ素材が準備完了
- メディア発注が完了

## 事後条件

- キャンペーンが実行される
- パフォーマンスデータが収集される

## 基本フロー

1. オペレーターがキャンペーンダッシュボードを開く
2. システムが実行予定キャンペーン一覧を表示する
3. オペレーターが対象キャンペーンを選択する
4. システムが計画詳細と準備状況を表示する
5. オペレーターが最終確認を行い実行開始する
6. システムが各チャネルへ配信を開始する
7. システムがリアルタイムでメトリクスを収集・表示する
8. オペレーターがパフォーマンスを監視する
9. オペレーターが必要に応じて調整を行う
10. キャンペーン終了時、システムが最終レポートを生成する

## 代替フロー

### 6a. 配信エラー

6a1. システムがエラーアラートを表示する
6a2. オペレーターが問題を調査する
6a3. オペレーターが修正または代替策を実行する

### 9a. 緊急停止

9a1. オペレーターが緊急停止をリクエストする
9a2. システムが確認ダイアログを表示する
9a3. オペレーターが確認する
9a4. システムが全チャネルの配信を停止する

## ビジネスルール適用

- BR-008: 予算消化80%でアラート
- BR-009: パフォーマンス異常時は自動アラート
