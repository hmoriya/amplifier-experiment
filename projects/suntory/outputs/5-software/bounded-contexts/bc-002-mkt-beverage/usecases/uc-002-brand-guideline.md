# UC-002 ブランドガイドライン作成 CreateBrandGuideline

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-002 |
| 名前 | ブランドガイドライン作成 |
| アクター | ブランドマネージャー、デザイナー |
| 所属Operation | OP-002 ブランドガイドライン管理 |

## 説明

ブランドガイドラインを作成し、ロゴ、カラー、タイポグラフィ等のルールを定義する。

## 事前条件

- ブランド戦略が承認済み
- ブランド資産（ロゴ等）が準備されている

## 事後条件

- ブランドガイドラインが公開される
- 関係者に通知が送信される

## 基本フロー

1. ブランドマネージャーが新規ガイドライン作成を開始する
2. システムがテンプレートを表示する
3. ブランドマネージャーがブランドエッセンスを入力する
4. デザイナーがロゴ使用ルールを定義する
5. デザイナーがカラーパレットを設定する
6. デザイナーがタイポグラフィを設定する
7. ブランドマネージャーがトーン&マナーを記述する
8. ブランドマネージャーが使用例を追加する
9. システムがプレビューを生成する
10. ブランドマネージャーが公開を承認する
11. システムがガイドラインを公開し関係者に通知する

## 代替フロー

### 4a. 既存ガイドラインの更新

4a1. システムが前バージョンを読み込む
4a2. ブランドマネージャーが変更箇所を編集する
4a3. 基本フロー9に進む

## ビジネスルール適用

- BR-003: ガイドラインはバージョン管理される
- BR-004: 必須セクション（ロゴ、カラー、タイポ、トーン）は必ず含む
