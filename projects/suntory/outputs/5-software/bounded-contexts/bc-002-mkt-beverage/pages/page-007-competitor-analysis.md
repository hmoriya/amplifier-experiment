# PAGE-007 競合分析 CompetitorAnalysis

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-007 |
| 日本語名 | 競合分析 |
| 英語名 | CompetitorAnalysis |
| URL | /market/competitors |
| 所属UseCase | UC-007 競合分析 |

## 説明

競合他社の動向を分析し、自社の競争ポジションを評価する画面。

## アクセス権限

| ロール | 権限 |
|--------|------|
| 市場調査アナリスト | 編集・分析 |
| マーケティングマネージャー | 閲覧 |
| ブランドマネージャー | 閲覧 |

## 画面構成

### ヘッダー

- カテゴリ選択
- 期間選択
- レポート出力ボタン

### 競合一覧

| カラム | 説明 |
|--------|------|
| 企業/ブランド名 | 競合名 |
| 市場シェア | カテゴリ内シェア |
| シェア推移 | 前期比 |
| 主力製品 | 主要製品リスト |
| 直近動向 | 最新ニュース |
| 詳細 | プロファイルへ |

### 市場シェアチャート

| チャート | 内容 |
|----------|------|
| シェア推移 | 月次/四半期のシェア推移 |
| シェア構成 | 現在のシェア構成（円グラフ） |
| カテゴリ別 | 製品カテゴリ別シェア |

### ポジショニングマップ

| 軸 | 選択可能項目 |
|----|--------------|
| X軸 | 価格帯/品質/機能性等 |
| Y軸 | ブランド力/イノベーション等 |
| バブルサイズ | 市場シェア/売上 |

### SWOT分析パネル

| セクション | 内容 |
|------------|------|
| 強み | 自社の強み |
| 弱み | 自社の弱み |
| 機会 | 市場機会 |
| 脅威 | 競合からの脅威 |

### サイドバー

- 競合ニュースフィード
- ウォッチリスト
- アラート設定

## アクション

| アクション | 説明 |
|------------|------|
| 競合追加 | 新規競合を追加 |
| プロファイル編集 | 競合情報を更新 |
| レポート生成 | 競合分析レポート生成 |
| アラート設定 | 動向アラート設定 |
