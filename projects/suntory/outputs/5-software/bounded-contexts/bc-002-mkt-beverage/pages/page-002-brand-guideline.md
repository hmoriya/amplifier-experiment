# PAGE-002 ブランドガイドライン編集 BrandGuidelineEditor

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-002 |
| 日本語名 | ブランドガイドライン編集 |
| 英語名 | BrandGuidelineEditor |
| URL | /brands/{brandId}/guideline/edit |
| 所属UseCase | UC-002 ブランドガイドライン作成 |

## 説明

ブランドガイドラインを作成・編集する画面。ロゴ、カラー、タイポグラフィ等のルールを定義する。

## アクセス権限

| ロール | 権限 |
|--------|------|
| ブランドマネージャー | 編集・公開 |
| デザイナー | 編集 |
| マーケティング担当者 | 閲覧 |

## 画面構成

### ヘッダー

- ブランド名
- バージョン番号
- ステータス（編集中/公開済み）
- 保存・公開ボタン

### メインコンテンツ（タブ構成）

#### タブ1: ロゴ

| 要素 | タイプ | 説明 |
|------|--------|------|
| プライマリロゴ | 画像アップロード | メインロゴ |
| ロゴバリエーション | 画像グリッド | 各種バリエーション |
| 使用ルール | リッチテキスト | 使用ルール説明 |
| 禁止例 | 画像グリッド | NG例 |
| クリアスペース | 図示ツール | 余白ルール |

#### タブ2: カラー

| 要素 | タイプ | 説明 |
|------|--------|------|
| プライマリカラー | カラーピッカー | メインカラー |
| セカンダリカラー | カラーピッカー | サブカラー |
| アクセントカラー | カラーピッカー | アクセント |
| カラーコード表 | テーブル | HEX/RGB/CMYK値 |
| 使用比率 | チャート | 推奨使用比率 |

#### タブ3: タイポグラフィ

| 要素 | タイプ | 説明 |
|------|--------|------|
| 見出しフォント | フォント選択 | 見出し用 |
| 本文フォント | フォント選択 | 本文用 |
| サイズ規定 | テーブル | サイズ階層 |
| 使用例 | プレビュー | サンプル表示 |

#### タブ4: トーン&マナー

| 要素 | タイプ | 説明 |
|------|--------|------|
| ブランドパーソナリティ | タグ | 性格キーワード |
| ボイストーン | テキストエリア | 語りかけの調子 |
| NGワード | リスト | 使用禁止語 |
| コピー例 | テーブル | OK/NG例 |

### サイドバー

- セクション目次
- 完成度インジケーター
- ダウンロードリンク

## アクション

| アクション | 説明 |
|------------|------|
| 保存 | ドラフト保存 |
| プレビュー | PDF形式でプレビュー |
| 公開 | ガイドライン公開 |
| 資産パッケージ生成 | ダウンロード用ZIP生成 |
