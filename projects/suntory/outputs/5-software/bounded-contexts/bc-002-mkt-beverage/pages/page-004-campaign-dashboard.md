# PAGE-004 キャンペーンダッシュボード CampaignDashboard

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-004 |
| 日本語名 | キャンペーンダッシュボード |
| 英語名 | CampaignDashboard |
| URL | /campaigns/dashboard |
| 所属UseCase | UC-004 キャンペーン実行管理 |

## 説明

実行中・予定・完了キャンペーンを一覧表示し、パフォーマンスを監視するダッシュボード。

## アクセス権限

| ロール | 権限 |
|--------|------|
| マーケティング担当者 | 閲覧・操作 |
| マーケティングマネージャー | 閲覧・承認 |
| キャンペーンオペレーター | 閲覧・実行操作 |

## 画面構成

### ヘッダー

- 期間フィルター（今週/今月/四半期）
- ブランドフィルター
- ステータスフィルター
- 新規作成ボタン

### KPIサマリーカード

| カード | 内容 |
|--------|------|
| 実行中キャンペーン数 | アクティブな件数 |
| 本日の配信数 | 当日の配信ボリューム |
| 予算消化率 | 全体予算の消化状況 |
| 平均パフォーマンス | 目標達成率 |

### メインコンテンツ

#### キャンペーン一覧テーブル

| カラム | 説明 |
|--------|------|
| キャンペーン名 | 名称（詳細へのリンク） |
| ブランド | 対象ブランド |
| ステータス | 予定/実行中/終了/停止 |
| 期間 | 開始日〜終了日 |
| 予算 | 予算額と消化率 |
| パフォーマンス | 主要KPI達成率 |
| アクション | 操作ボタン |

#### パフォーマンスチャート

| チャート | 内容 |
|----------|------|
| 日次推移 | 配信数・反応数の日次推移 |
| チャネル別 | チャネル別パフォーマンス |
| 予算消化 | 予算消化の推移 |

### サイドバー

- 今日のタスク
- アラート一覧
- 承認待ち一覧

## アクション

| アクション | 説明 |
|------------|------|
| 詳細表示 | キャンペーン詳細へ |
| 開始/停止 | キャンペーンの開始・停止 |
| 編集 | 計画の編集 |
| レポート | レポート画面へ |

## リアルタイム更新

- パフォーマンス数値: 5分ごと自動更新
- アラート: リアルタイムプッシュ
