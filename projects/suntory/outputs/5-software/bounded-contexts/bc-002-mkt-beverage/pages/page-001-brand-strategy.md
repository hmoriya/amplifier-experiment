# PAGE-001 ブランド戦略編集 BrandStrategyEditor

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-001 |
| 日本語名 | ブランド戦略編集 |
| 英語名 | BrandStrategyEditor |
| URL | /brands/{brandId}/strategy/edit |
| 所属UseCase | UC-001 ブランド戦略策定 |

## 説明

ブランド戦略を作成・編集するための画面。ビジョン、ポジショニング、ターゲット等を入力する。

## アクセス権限

| ロール | 権限 |
|--------|------|
| ブランドマネージャー | 編集・保存・承認申請 |
| マーケティング部長 | 閲覧・承認 |
| マーケティング担当者 | 閲覧のみ |

## 画面構成

### ヘッダー

- ブランド名表示
- 戦略バージョン表示
- ステータス表示（ドラフト/レビュー中/承認済み）
- 保存・承認申請ボタン

### メインコンテンツ

#### セクション1: ブランドエッセンス

| 要素 | タイプ | 説明 |
|------|--------|------|
| ビジョン | テキストエリア | ブランドのビジョン |
| ミッション | テキストエリア | ブランドのミッション |
| バリュー | タグ入力 | ブランドの価値観（複数） |

#### セクション2: ターゲット定義

| 要素 | タイプ | 説明 |
|------|--------|------|
| プライマリターゲット | ペルソナ選択 | 主要ターゲット |
| セカンダリターゲット | ペルソナ選択 | 副次ターゲット |
| ターゲット特性 | テーブル | 人口統計・心理特性 |

#### セクション3: ポジショニング

| 要素 | タイプ | 説明 |
|------|--------|------|
| ポジショニングステートメント | テキストエリア | ポジショニング文 |
| 差別化ポイント | リスト入力 | 差別化要素 |
| ポジショニングマップ | キャンバス | 視覚的ポジション |

#### セクション4: 価値提案

| 要素 | タイプ | 説明 |
|------|--------|------|
| 機能的価値 | テキストエリア | 機能的ベネフィット |
| 情緒的価値 | テキストエリア | 情緒的ベネフィット |
| 自己表現価値 | テキストエリア | 自己表現ベネフィット |

### サイドバー

- 市場データサマリー
- 競合ポジション表示
- 関連ドキュメントリンク

## アクション

| アクション | 説明 |
|------------|------|
| 保存 | ドラフト保存 |
| プレビュー | 戦略書プレビュー |
| 承認申請 | 承認ワークフロー開始 |
| 履歴表示 | 過去バージョン表示 |

## バリデーション

| フィールド | ルール |
|------------|--------|
| ビジョン | 必須、500文字以内 |
| ミッション | 必須、300文字以内 |
| ターゲット | 最低1つ必須 |
| ポジショニング | 必須 |
