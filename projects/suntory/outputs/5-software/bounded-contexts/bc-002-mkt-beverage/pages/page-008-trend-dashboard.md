# PAGE-008 トレンドダッシュボード TrendDashboard

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-008 |
| 日本語名 | トレンドダッシュボード |
| 英語名 | TrendDashboard |
| URL | /market/trends |
| 所属UseCase | UC-008 トレンド分析 |

## 説明

消費者トレンドと市場動向を可視化し、分析するためのダッシュボード。

## アクセス権限

| ロール | 権限 |
|--------|------|
| 市場調査アナリスト | 閲覧・分析 |
| マーケティング全般 | 閲覧 |
| 経営層 | 閲覧 |

## 画面構成

### ヘッダー

- カテゴリフィルター
- 期間選択
- トレンドアラート設定

### トレンドサマリー

| カード | 内容 |
|--------|------|
| 注目トレンド | 今最も注目すべきトレンド |
| 新興トレンド | 新たに検出されたトレンド |
| 衰退トレンド | 勢いが落ちているトレンド |
| 自社関連 | 自社に関連するトレンド |

### メインコンテンツ

#### トレンドマップ

| 要素 | 説明 |
|------|------|
| バブルチャート | トレンドの規模と成長率 |
| カテゴリ色分け | カテゴリ別色分け |
| クリックで詳細 | 詳細パネル表示 |

#### トレンド推移

| チャート | 内容 |
|----------|------|
| 検索ボリューム | 検索トレンド推移 |
| SNS言及 | ソーシャル言及推移 |
| 売上相関 | 売上との相関表示 |

#### トレンド一覧

| カラム | 説明 |
|--------|------|
| トレンド名 | キーワード/テーマ |
| カテゴリ | 分類 |
| 成長率 | 前期比成長率 |
| 規模 | 市場規模/言及量 |
| 自社関連度 | 関連度スコア |
| ステータス | 上昇/安定/下降 |

### サイドバー

- カテゴリツリー
- 保存済み分析
- レポートリンク

## アクション

| アクション | 説明 |
|------------|------|
| 詳細分析 | トレンド詳細分析へ |
| ウォッチ登録 | トレンドをウォッチリストに追加 |
| レポート生成 | トレンドレポート生成 |
| アラート設定 | 変動アラート設定 |

## データ更新

| データソース | 更新頻度 |
|--------------|----------|
| 検索トレンド | 日次 |
| SNSデータ | 時間次 |
| 売上データ | 日次 |
| 調査データ | 調査完了時 |
