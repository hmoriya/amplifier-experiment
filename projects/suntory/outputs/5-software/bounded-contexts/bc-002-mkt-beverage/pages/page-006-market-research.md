# PAGE-006 市場調査管理 MarketResearchManager

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-006 |
| 日本語名 | 市場調査管理 |
| 英語名 | MarketResearchManager |
| URL | /research/projects |
| 所属UseCase | UC-006 市場調査実施 |

## 説明

市場調査プロジェクトを管理し、調査の計画から結果管理までを行う画面。

## アクセス権限

| ロール | 権限 |
|--------|------|
| 市場調査アナリスト | 作成・編集・実行 |
| マーケティングマネージャー | 閲覧・承認 |
| マーケティング担当者 | 閲覧 |

## 画面構成

### ヘッダー

- 期間フィルター
- ステータスフィルター
- 新規調査作成ボタン

### プロジェクト一覧

| カラム | 説明 |
|--------|------|
| 調査名 | プロジェクト名称 |
| 調査タイプ | 定量/定性/混合 |
| ステータス | 計画中/実施中/分析中/完了 |
| 期間 | 調査期間 |
| 担当者 | 主担当 |
| 進捗 | 進捗率 |
| アクション | 操作ボタン |

### クイックサマリー

| カード | 内容 |
|--------|------|
| 実施中調査 | 現在進行中の件数 |
| 今月完了予定 | 今月完了予定件数 |
| 調査予算 | 年間予算と消化率 |
| 最新レポート | 直近公開レポート |

### サイドバー

- 調査カレンダー
- 調査テンプレート
- 外部調査会社リスト

## アクション

| アクション | 説明 |
|------------|------|
| 新規作成 | 新規調査プロジェクト作成 |
| 詳細表示 | プロジェクト詳細へ |
| レポート閲覧 | 完了調査のレポート閲覧 |
| 複製 | 過去調査をテンプレートとして複製 |
