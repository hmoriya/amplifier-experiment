# PAGE-005 キャンペーン分析 CampaignAnalysis

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-005 |
| 日本語名 | キャンペーン分析 |
| 英語名 | CampaignAnalysis |
| URL | /campaigns/{campaignId}/analysis |
| 所属UseCase | UC-005 キャンペーン効果分析 |

## 説明

キャンペーンの効果を詳細分析し、ROIを算出するための画面。

## アクセス権限

| ロール | 権限 |
|--------|------|
| マーケティングアナリスト | 分析・レポート作成 |
| マーケティングマネージャー | 閲覧 |
| マーケティング担当者 | 閲覧 |

## 画面構成

### ヘッダー

- キャンペーン名・期間
- 分析期間選択
- 比較対象選択（前回/前年同期/平均）
- レポート出力ボタン

### KPIサマリー

| カード | 内容 |
|--------|------|
| 総リーチ | 到達人数と目標比 |
| エンゲージメント | 反応数とレート |
| コンバージョン | 成果数と率 |
| ROI | 投資対効果 |

### メインコンテンツ（タブ構成）

#### タブ1: パフォーマンス推移

| 要素 | タイプ | 説明 |
|------|--------|------|
| 日次推移チャート | 折れ線グラフ | 各指標の日次推移 |
| 期間比較 | 棒グラフ | 比較対象との比較 |
| 指標選択 | チェックボックス | 表示指標選択 |

#### タブ2: チャネル分析

| 要素 | タイプ | 説明 |
|------|--------|------|
| チャネル別パフォーマンス | テーブル | 各チャネルの成果 |
| コスト効率 | 棒グラフ | CPA/CPC比較 |
| アトリビューション | サンキーダイアグラム | コンバージョン経路 |

#### タブ3: オーディエンス分析

| 要素 | タイプ | 説明 |
|------|--------|------|
| セグメント別反応 | テーブル | セグメント別成果 |
| 新規vs既存 | 円グラフ | 顧客タイプ別 |
| 地域別 | 地図 | 地域別パフォーマンス |

#### タブ4: ROI分析

| 要素 | タイプ | 説明 |
|------|--------|------|
| 投資内訳 | 円グラフ | コスト構成 |
| 売上貢献 | 数値 | アトリビューション売上 |
| ROI計算 | テーブル | ROI詳細計算 |
| 予算効率 | 指標 | 予算あたり成果 |

### サイドバー

- キャンペーン基本情報
- 関連キャンペーン
- インサイトメモ入力

## アクション

| アクション | 説明 |
|------------|------|
| レポート生成 | PDF/PPTレポート生成 |
| データエクスポート | CSV/Excelエクスポート |
| インサイト保存 | 分析メモを保存 |
| 共有 | レポートを共有 |
