# PAGE-003 キャンペーン企画 CampaignPlanner

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-003 |
| 日本語名 | キャンペーン企画 |
| 英語名 | CampaignPlanner |
| URL | /campaigns/new |
| 所属UseCase | UC-003 キャンペーン企画 |

## 説明

キャンペーンを企画・設計するための画面。目的、ターゲット、予算、スケジュール等を設定する。

## アクセス権限

| ロール | 権限 |
|--------|------|
| マーケティング担当者 | 作成・編集 |
| マーケティングマネージャー | 承認 |
| メディアプランナー | 閲覧・編集（メディア部分） |

## 画面構成

### ステップウィザード形式

#### ステップ1: 基本情報

| 要素 | タイプ | 説明 |
|------|--------|------|
| キャンペーン名 | テキスト | キャンペーン名称 |
| ブランド選択 | ドロップダウン | 対象ブランド |
| 目的 | ラジオボタン | 認知/検討/購入/ロイヤルティ |
| 目的詳細 | テキストエリア | 具体的な目的 |
| キャンペーンタイプ | セレクト | 新製品/季節/イベント等 |

#### ステップ2: ターゲット設定

| 要素 | タイプ | 説明 |
|------|--------|------|
| セグメント選択 | マルチセレクト | CDPセグメント |
| リーチ見込み | 数値表示 | 推定リーチ人数 |
| ペルソナ表示 | カード | 選択セグメントのペルソナ |
| カスタム条件 | フィルタービルダー | 追加絞り込み |

#### ステップ3: スケジュール・予算

| 要素 | タイプ | 説明 |
|------|--------|------|
| 開始日 | 日付ピッカー | キャンペーン開始日 |
| 終了日 | 日付ピッカー | キャンペーン終了日 |
| 総予算 | 数値入力 | 総予算額 |
| 予算配分 | スライダー | チャネル別配分 |
| カレンダー表示 | ガントチャート | スケジュール表示 |

#### ステップ4: チャネル・メディア

| 要素 | タイプ | 説明 |
|------|--------|------|
| チャネル選択 | チェックボックス | 利用チャネル |
| メディアミックス | 配分チャート | メディア配分 |
| 予算シミュレーション | グラフ | 予算消化予測 |
| リーチ・フリークエンシー | 数値表示 | 到達予測 |

#### ステップ5: KPI設定

| 要素 | タイプ | 説明 |
|------|--------|------|
| KPI選択 | チェックボックス | 測定するKPI |
| 目標値設定 | 数値入力 | 各KPIの目標値 |
| ベンチマーク表示 | 参考値 | 過去実績 |

#### ステップ6: 確認・申請

| 要素 | タイプ | 説明 |
|------|--------|------|
| サマリー表示 | 読み取り専用 | 入力内容サマリー |
| ガイドラインチェック | アラート | 違反警告 |
| 承認者選択 | ドロップダウン | 承認者指定 |

## アクション

| アクション | 説明 |
|------------|------|
| ドラフト保存 | 途中保存 |
| 次へ/戻る | ステップ移動 |
| 承認申請 | 承認ワークフロー開始 |
