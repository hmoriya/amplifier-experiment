# BC-002 MKT-Beverage 飲料マーケティング

## 概要

飲料製品のマーケティング戦略立案・実行、ブランド管理、キャンペーン管理を担当するBounded Context。

## スコープ

### 含むもの

- ブランドポートフォリオ管理
- マーケティングキャンペーン企画・実行
- 市場分析・競合分析
- 製品ライフサイクル管理
- プロモーション管理
- 広告メディア管理
- マーケティングROI分析

### 含まないもの

- 顧客データ管理（→ BC-003 CRM-CDP）
- 販売チャネル管理（→ BC-007 SCM-Distribution）
- 製品開発R&D（→ BC-001 RD-WaterScience等）

## サブドメイン分類

| 分類 | 理由 |
|------|------|
| Core Domain | 飲料事業の差別化・競争優位の源泉 |

## コンテキストマップ

### 上流（依存先）

| BC | 関係 | 受け取るデータ |
|----|------|----------------|
| BC-001 RD-WaterScience | Conformist | 製品技術情報、品質特性 |
| BC-003 CRM-CDP | Customer-Supplier | 顧客セグメント、顧客インサイト |
| BC-015 FIN-Management | Customer-Supplier | 予算情報、コスト基準 |

### 下流（依存元）

| BC | 関係 | 提供するデータ |
|----|------|----------------|
| BC-007 SCM-Distribution | Customer-Supplier | キャンペーン情報、需要予測 |
| BC-003 CRM-CDP | Partnership | キャンペーン反応データ要求 |

## L3 Capabilities

| ID | 名前 | 説明 |
|----|------|------|
| CAP-001 | ブランド管理 BrandManagement | ブランドポートフォリオの戦略策定と実行 |
| CAP-002 | キャンペーン管理 CampaignManagement | マーケティングキャンペーンの企画から実行・分析 |
| CAP-003 | 市場分析 MarketAnalysis | 市場動向・競合分析とインサイト導出 |

## 主要サービス

| サービス | 責務 |
|----------|------|
| BrandService | ブランドライフサイクル管理 |
| CampaignService | キャンペーン企画・実行・評価 |
| MarketResearchService | 市場調査・分析 |
| MediaService | 広告メディア計画・実行 |
| PromotionService | プロモーション施策管理 |
| ROIAnalysisService | マーケティング投資対効果分析 |

## 主要ユーザー

| ユーザー | 役割 |
|----------|------|
| ブランドマネージャー | ブランド戦略の策定・実行責任者 |
| マーケティング担当者 | キャンペーン・プロモーション実務 |
| 市場調査アナリスト | 市場分析・レポート作成 |
| メディアプランナー | 広告メディア計画策定 |

## 成功指標

| 指標 | 目標値 |
|------|--------|
| ブランド認知度 | 対象市場で80%以上 |
| キャンペーンROI | 投資対効果150%以上 |
| 市場シェア | カテゴリ別1-2位維持 |
| 新製品成功率 | 60%以上（発売後1年） |
