# OP-003 キャンペーン企画 PlanCampaign PLAN_CAMPAIGN

## 概要

| 項目 | 値 |
|------|-----|
| ID | OP-003 |
| 日本語名 | キャンペーン企画 |
| 英語名 | PlanCampaign |
| 定数名 | PLAN_CAMPAIGN |
| 所属Capability | CAP-002 キャンペーン管理 |

## 説明

マーケティングキャンペーンの目的、ターゲット、メッセージ、チャネル、予算、
スケジュールを策定する。

## トリガー

| トリガー | 説明 |
|----------|------|
| 年間計画策定 | 年度マーケティング計画に基づく |
| 新製品発売 | 新製品ローンチキャンペーン |
| 季節イベント | 季節・イベントに合わせた企画 |
| 競合対応 | 競合施策への対抗企画 |

## 入力

| 入力 | 型 | 説明 |
|------|-----|------|
| campaignBrief | CampaignBrief | キャンペーン概要 |
| budget | Budget | 利用可能予算 |
| targetSegment | CustomerSegment[] | ターゲットセグメント |
| brandGuideline | BrandGuideline | ブランドガイドライン |

## 出力

| 出力 | 型 | 説明 |
|------|-----|------|
| campaignPlan | CampaignPlan | キャンペーン計画書 |
| mediaBreakdown | MediaBreakdown | メディア配分計画 |
| timeline | CampaignTimeline | スケジュール |
| kpiTargets | KPITarget[] | KPI目標値 |

## ビジネスルール

| ルール | 説明 |
|--------|------|
| 予算上限 | 承認済み予算枠内で計画 |
| ブランド準拠 | ガイドラインへの準拠必須 |
| KPI設定 | 測定可能なKPI必須設定 |
| リードタイム | 実行開始の4週間前までに承認 |

## 例外処理

| 例外 | 対応 |
|------|------|
| 予算超過 | 規模縮小または追加予算申請 |
| ターゲット不明確 | CDP連携で再セグメント |
