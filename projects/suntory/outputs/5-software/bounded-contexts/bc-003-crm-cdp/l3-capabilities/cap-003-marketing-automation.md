# CAP-003 マーケティングオートメーション MarketingAutomation MARKETING_AUTOMATION

## 概要

| 項目 | 値 |
|------|-----|
| ID | CAP-003 |
| 日本語名 | マーケティングオートメーション |
| 英語名 | MarketingAutomation |
| 定数名 | MARKETING_AUTOMATION |
| 所属BC | BC-003 CRM-CDP |

## 説明

顧客行動やセグメントに基づき、自動化されたコミュニケーションを配信する。
カスタマージャーニーに沿った最適なタイミング・チャネルでの接触を実現する。

## 提供する価値

- コミュニケーションの自動化
- パーソナライズされた顧客体験
- 適切なタイミングでの接触
- オペレーション効率化
