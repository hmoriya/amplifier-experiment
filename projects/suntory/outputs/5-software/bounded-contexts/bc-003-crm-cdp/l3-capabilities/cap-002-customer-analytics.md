# CAP-002 顧客分析 CustomerAnalytics CUSTOMER_ANALYTICS

## 概要

| 項目 | 値 |
|------|-----|
| ID | CAP-002 |
| 日本語名 | 顧客分析 |
| 英語名 | CustomerAnalytics |
| 定数名 | CUSTOMER_ANALYTICS |
| 所属BC | BC-003 CRM-CDP |

## 説明

統合された顧客データを分析し、セグメンテーション、行動予測、インサイト導出を行う。
機械学習を活用した高度な分析機能を提供する。

## 提供する価値

- 顧客行動の深い理解
- 精度の高いセグメンテーション
- 予測分析による先回り施策
- データドリブンな意思決定支援

| OP-006 | 予測モデル実行 RunPrediction | 予測モデルの実行・スコアリング |
