# PAGE-002 顧客プロファイル CustomerProfile

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-002 |
| 日本語名 | 顧客プロファイル |
| 英語名 | CustomerProfile |
| URL | /customers/{customerId} |
| 所属UseCase | UC-001 顧客プロファイル閲覧 |

## 説明

顧客の360度ビューを表示する画面。属性、行動、取引履歴を統合表示。

## アクセス権限

| ロール | 権限 |
|--------|------|
| CRMマネージャー | 全情報閲覧 |
| データアナリスト | 全情報閲覧 |
| カスタマーサポート | 限定情報閲覧 |

## 画面構成

### ヘッダー

- 顧客名、顧客ID
- ステータスバッジ
- 主要セグメントタグ

### タブ構成

#### タブ1: 基本情報

| 要素 | 説明 |
|------|------|
| 属性情報 | 名前、連絡先、住所等 |
| 統合ID一覧 | 各ソースのID |
| 同意状況 | マーケティング同意 |

#### タブ2: 行動履歴

| 要素 | 説明 |
|------|------|
| タイムライン | 時系列行動表示 |
| チャネル別 | チャネル別行動 |
| イベント詳細 | 各イベントの詳細 |

#### タブ3: 取引履歴

| 要素 | 説明 |
|------|------|
| 購買一覧 | 購買履歴テーブル |
| 購買サマリー | 累計金額、回数等 |
| トレンド | 購買トレンドグラフ |

#### タブ4: スコア・セグメント

| 要素 | 説明 |
|------|------|
| 予測スコア | 各種予測スコア |
| 所属セグメント | セグメント一覧 |
| スコア推移 | スコアのトレンド |

#### タブ5: コミュニケーション

| 要素 | 説明 |
|------|------|
| 配信履歴 | 過去の配信 |
| 反応履歴 | 開封、クリック等 |
| 予定配信 | 今後の予定 |

### サイドバー

- クイックアクション
- 関連顧客（世帯等）
- 担当者情報
