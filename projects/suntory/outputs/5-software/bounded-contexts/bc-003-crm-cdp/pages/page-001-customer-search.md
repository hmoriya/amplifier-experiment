# PAGE-001 顧客検索 CustomerSearch

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-001 |
| 日本語名 | 顧客検索 |
| 英語名 | CustomerSearch |
| URL | /customers/search |
| 所属UseCase | UC-001 顧客プロファイル閲覧 |

## 説明

顧客を検索し、プロファイル画面へ遷移するための画面。

## アクセス権限

| ロール | 権限 |
|--------|------|
| CRMマネージャー | 全顧客検索可 |
| データアナリスト | 全顧客検索可 |
| カスタマーサポート | 担当顧客のみ |

## 画面構成

### 検索フォーム

| 要素 | タイプ | 説明 |
|------|--------|------|
| キーワード | テキスト | 名前、メール、電話で検索 |
| 顧客ID | テキスト | 統合顧客IDで検索 |
| セグメント | ドロップダウン | セグメントで絞り込み |
| 登録期間 | 日付範囲 | 登録日で絞り込み |

### 検索結果

| カラム | 説明 |
|--------|------|
| 顧客ID | 統合顧客ID |
| 名前 | 顧客名 |
| メール | メールアドレス（マスク） |
| 登録日 | 初回登録日 |
| セグメント | 所属セグメント |
| アクション | 詳細リンク |

## アクション

| アクション | 説明 |
|------------|------|
| 検索 | 検索実行 |
| クリア | 条件クリア |
| 詳細表示 | プロファイル画面へ |
| エクスポート | 検索結果CSV出力 |
