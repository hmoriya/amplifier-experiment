# PAGE-005 配信ダッシュボード DeliveryDashboard

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-005 |
| 日本語名 | 配信ダッシュボード |
| 英語名 | DeliveryDashboard |
| URL | /delivery/dashboard |
| 所属UseCase | UC-004 配信モニタリング |

## 説明

コミュニケーション配信の状況をリアルタイムで監視するダッシュボード。

## アクセス権限

| ロール | 権限 |
|--------|------|
| MAオペレーター | 閲覧・操作 |
| CRMマネージャー | 閲覧 |

## 画面構成

### KPIサマリー

| カード | 内容 |
|--------|------|
| 本日配信数 | 当日の総配信数 |
| 配信成功率 | 成功率（%） |
| 平均開封率 | 開封率（%） |
| アクティブジャーニー | 実行中ジャーニー数 |

### 配信一覧

| カラム | 説明 |
|--------|------|
| 配信名 | コミュニケーション名 |
| チャネル | メール/Push/SMS |
| ステータス | 実行中/完了/エラー |
| 送信数 | 送信件数 |
| 成功率 | 配信成功率 |
| 開封率 | 開封率 |
| 開始時刻 | 配信開始時刻 |

### リアルタイムチャート

| チャート | 内容 |
|----------|------|
| 配信推移 | 時間帯別配信数 |
| チャネル別 | チャネル別内訳 |
| エラー推移 | エラー発生状況 |

### アラートパネル

| 要素 | 説明 |
|------|------|
| アクティブアラート | 未対応アラート |
| アラート履歴 | 過去のアラート |

## アクション

| アクション | 説明 |
|------------|------|
| 詳細表示 | 配信詳細へ |
| 一時停止 | 配信を一時停止 |
| 再開 | 停止中配信を再開 |
| レポート | レポート画面へ |

## リアルタイム更新

- 配信メトリクス: 1分ごと自動更新
- アラート: リアルタイムプッシュ
