# PAGE-003 セグメントビルダー SegmentBuilder

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-003 |
| 日本語名 | セグメントビルダー |
| 英語名 | SegmentBuilder |
| URL | /segments/new |
| 所属UseCase | UC-002 セグメント作成 |

## 説明

条件を組み合わせて顧客セグメントを作成するビジュアルビルダー。

## アクセス権限

| ロール | 権限 |
|--------|------|
| データアナリスト | 作成・編集 |
| CRMマネージャー | 作成・編集・承認 |
| マーケティング担当者 | 閲覧・複製 |

## 画面構成

### ヘッダー

- セグメント名入力
- 説明入力
- 保存・プレビューボタン

### 条件ビルダー

| 要素 | タイプ | 説明 |
|------|--------|------|
| 条件グループ | コンテナ | AND/OR切り替え |
| 条件行 | ビルダー | 属性・演算子・値 |
| 属性選択 | ツリー選択 | 利用可能属性 |
| 演算子 | ドロップダウン | =, !=, >, <, contains等 |
| 値入力 | 動的 | 属性に応じた入力 |

### リアルタイムプレビュー

| 要素 | 説明 |
|------|------|
| 対象者数 | 条件に該当する顧客数 |
| サンプル顧客 | ランダムサンプル表示 |
| 属性分布 | 主要属性の分布 |

### 設定パネル

| 要素 | 説明 |
|------|------|
| 更新頻度 | リアルタイム/日次/手動 |
| 有効期間 | 開始日・終了日 |
| 通知設定 | サイズ変動時の通知 |

## アクション

| アクション | 説明 |
|------------|------|
| 条件追加 | 新規条件行追加 |
| グループ追加 | 条件グループ追加 |
| プレビュー | 対象者プレビュー |
| 保存 | セグメント保存 |
