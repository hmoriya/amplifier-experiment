# PAGE-004 ジャーニービルダー JourneyBuilder

## 概要

| 項目 | 値 |
|------|-----|
| ID | PAGE-004 |
| 日本語名 | ジャーニービルダー |
| 英語名 | JourneyBuilder |
| URL | /journeys/new |
| 所属UseCase | UC-003 ジャーニー設計 |

## 説明

ドラッグ&ドロップでカスタマージャーニーを設計するビジュアルビルダー。

## アクセス権限

| ロール | 権限 |
|--------|------|
| MAオペレーター | 作成・編集 |
| CRMマネージャー | 作成・編集・有効化 |

## 画面構成

### ヘッダー

- ジャーニー名
- ステータス（ドラフト/有効/停止）
- 保存・有効化ボタン

### ツールパレット

| カテゴリ | ノードタイプ |
|----------|--------------|
| トリガー | セグメント入場、イベント発生、スケジュール |
| アクション | メール送信、Push送信、SMS送信、Webhook |
| 条件 | 分岐、待機、フィルター |
| 制御 | 終了、別ジャーニーへ |

### キャンバス

| 要素 | 説明 |
|------|------|
| ノード | ジャーニーステップ |
| コネクター | ステップ間の接続線 |
| 分岐 | 条件分岐の表現 |
| ズーム | キャンバスのズーム |

### プロパティパネル

| 要素 | 説明 |
|------|------|
| ノード設定 | 選択ノードの詳細設定 |
| コンテンツ選択 | 配信コンテンツ選択 |
| 条件設定 | 分岐条件の設定 |
| 待機設定 | 待機時間・条件 |

### 統計パネル（実行中の場合）

| 要素 | 説明 |
|------|------|
| 各ステップ通過数 | ノードごとの通過人数 |
| 現在待機中 | 各ステップの待機人数 |
| 完了数 | ジャーニー完了人数 |

## アクション

| アクション | 説明 |
|------------|------|
| ノード追加 | パレットからドラッグ |
| 接続 | ノード間を線で接続 |
| 検証 | ジャーニーの妥当性チェック |
| 保存 | ドラフト保存 |
| 有効化 | ジャーニー開始 |
