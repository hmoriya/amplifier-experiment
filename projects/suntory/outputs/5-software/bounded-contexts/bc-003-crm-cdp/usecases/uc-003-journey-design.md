# UC-003 ジャーニー設計 DesignJourney

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-003 |
| 名前 | ジャーニー設計 |
| アクター | MAオペレーター、CRMマネージャー |
| 所属Operation | OP-007 ジャーニー設計 |

## 説明

カスタマージャーニーを視覚的に設計し、自動配信フローを構築する。

## 事前条件

- セグメントが作成されている
- コンテンツテンプレートが準備されている

## 事後条件

- ジャーニーが作成される
- ジャーニーが実行可能状態になる

## 基本フロー

1. ユーザーがジャーニー作成画面を開く
2. ユーザーがジャーニー名と目的を入力する
3. ユーザーがエントリー条件（トリガー）を設定する
4. ユーザーがキャンバスにステップを配置する
5. ユーザーが各ステップのアクションを設定する
6. ユーザーが分岐条件を設定する
7. ユーザーが終了条件を設定する
8. システムがジャーニーを検証する
9. ユーザーがジャーニーを保存・有効化する

## 代替フロー

### 8a. 検証エラー

8a1. システムがエラー箇所をハイライト表示する
8a2. ユーザーが該当箇所を修正する
