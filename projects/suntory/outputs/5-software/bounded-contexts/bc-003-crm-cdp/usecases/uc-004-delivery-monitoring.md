# UC-004 配信モニタリング MonitorDelivery

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-004 |
| 名前 | 配信モニタリング |
| アクター | MAオペレーター |
| 所属Operation | OP-008 配信実行 |

## 説明

コミュニケーション配信の状況をリアルタイムで監視し、問題発生時に対応する。

## 事前条件

- 配信ジョブが実行中または完了している

## 事後条件

- 配信状況が把握される
- 必要に応じて対応アクションが実行される

## 基本フロー

1. オペレーターが配信ダッシュボードを開く
2. システムが実行中・完了配信の一覧を表示する
3. オペレーターが対象配信を選択する
4. システムがリアルタイムメトリクスを表示する
5. オペレーターが配信成功率、開封率等を確認する
6. オペレーターがエラーレコードを確認する

## 代替フロー

### 5a. 異常検知

5a1. システムがアラートを表示する
5a2. オペレーターが詳細を確認する
5a3. オペレーターが配信停止または継続を判断する
