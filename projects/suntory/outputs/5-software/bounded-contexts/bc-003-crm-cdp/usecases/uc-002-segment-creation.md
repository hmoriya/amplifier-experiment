# UC-002 セグメント作成 CreateSegment

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-002 |
| 名前 | セグメント作成 |
| アクター | データアナリスト、マーケティング担当者 |
| 所属Operation | OP-004 セグメント作成 |

## 説明

顧客属性・行動条件に基づいてセグメントを定義・作成する。

## 事前条件

- ユーザーがセグメント作成権限を持つ
- 顧客データが統合されている

## 事後条件

- セグメントが作成される
- セグメントメンバーが計算される

## 基本フロー

1. ユーザーがセグメント作成画面を開く
2. ユーザーがセグメント名と説明を入力する
3. ユーザーが条件ビルダーで抽出条件を設定する
4. システムがリアルタイムで対象者数を表示する
5. ユーザーが条件を調整する
6. ユーザーが更新頻度を設定する
7. ユーザーがセグメントを保存する
8. システムがセグメントメンバーを計算する

## 代替フロー

### 4a. 対象者数が少なすぎる

4a1. システムが警告を表示する
4a2. ユーザーが条件を緩和する
