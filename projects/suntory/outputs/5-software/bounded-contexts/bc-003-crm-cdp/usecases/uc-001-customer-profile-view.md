# UC-001 顧客プロファイル閲覧 ViewCustomerProfile

## 概要

| 項目 | 値 |
|------|-----|
| ID | UC-001 |
| 名前 | 顧客プロファイル閲覧 |
| アクター | CRMマネージャー、データアナリスト |
| 所属Operation | OP-003 プロファイル構築 |

## 説明

統合された顧客プロファイル（360度ビュー）を閲覧し、顧客理解を深める。

## 事前条件

- ユーザーが閲覧権限を持つ
- 対象顧客のプロファイルが存在する

## 事後条件

- プロファイル閲覧ログが記録される

## 基本フロー

1. ユーザーが顧客検索を実行する
2. システムが検索結果を表示する
3. ユーザーが対象顧客を選択する
4. システムが統合プロファイルを表示する
5. ユーザーが各セクション（属性、行動、取引）を確認する
6. システムがタイムライン形式で行動履歴を表示する
