# OP-007 ジャーニー設計 DesignJourney DESIGN_JOURNEY

## 概要

| 項目 | 値 |
|------|-----|
| ID | OP-007 |
| 日本語名 | ジャーニー設計 |
| 英語名 | DesignJourney |
| 定数名 | DESIGN_JOURNEY |
| 所属Capability | CAP-003 マーケティングオートメーション |

## 説明

顧客の状態遷移に基づくカスタマージャーニーを設計する。

## トリガー

| トリガー | 説明 |
|----------|------|
| ユーザー操作 | 新規ジャーニー作成 |
| 既存ジャーニー複製 | テンプレートからの作成 |

## 入力

| 入力 | 型 | 説明 |
|------|-----|------|
| journeyName | String | ジャーニー名 |
| entryCondition | Condition | エントリー条件 |
| steps | JourneyStep[] | ジャーニーステップ |

## 出力

| 出力 | 型 | 説明 |
|------|-----|------|
| journey | Journey | ジャーニー定義 |
| validationResult | ValidationResult | 検証結果 |
| estimatedReach | Long | 推定対象者数 |

## ビジネスルール

| ルール | 説明 |
|--------|------|
| 無限ループ防止 | 循環参照の検出・防止 |
| 終了条件必須 | 明確な終了条件設定 |
| 待機時間制限 | 最大待機時間の制限 |
