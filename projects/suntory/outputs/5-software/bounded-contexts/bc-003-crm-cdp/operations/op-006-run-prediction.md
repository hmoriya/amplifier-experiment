# OP-006 予測モデル実行 RunPrediction RUN_PREDICTION

## 概要

| 項目 | 値 |
|------|-----|
| ID | OP-006 |
| 日本語名 | 予測モデル実行 |
| 英語名 | RunPrediction |
| 定数名 | RUN_PREDICTION |
| 所属Capability | CAP-002 顧客分析 |

## 説明

機械学習モデルを実行し、顧客の将来行動を予測・スコアリングする。

## トリガー

| トリガー | 説明 |
|----------|------|
| 日次バッチ | 全顧客スコアリング |
| リアルタイム | 個別顧客スコアリング |
| モデル更新後 | 新モデル適用 |

## 入力

| 入力 | 型 | 説明 |
|------|-----|------|
| model | PredictionModel | 予測モデル |
| customers | CustomerProfile[] | 対象顧客 |
| features | FeatureSet | 特徴量セット |

## 出力

| 出力 | 型 | 説明 |
|------|-----|------|
| scores | CustomerScore[] | 顧客スコア |
| predictions | Prediction[] | 予測結果 |

## ビジネスルール

| ルール | 説明 |
|--------|------|
| モデル検証 | 本番適用前の精度検証必須 |
| スコア正規化 | 0-100の範囲に正規化 |
| 履歴保持 | スコア履歴を保持 |

## 関連UseCase

| UseCase | 関係 |
|---------|------|
| UC-009 予測スコアリング | 主UseCase |
| UC-010 モデル管理 | モデル運用 |
