# OP-004 セグメント作成 CreateSegment CREATE_SEGMENT

## 概要

| 項目 | 値 |
|------|-----|
| ID | OP-004 |
| 日本語名 | セグメント作成 |
| 英語名 | CreateSegment |
| 定数名 | CREATE_SEGMENT |
| 所属Capability | CAP-002 顧客分析 |

## 説明

顧客属性・行動に基づくセグメントを定義し、対象顧客を抽出する。

## トリガー

| トリガー | 説明 |
|----------|------|
| ユーザー操作 | 新規セグメント作成 |
| スケジュール | 定期的な再計算 |
| イベント | トリガー条件成立時 |

## 入力

| 入力 | 型 | 説明 |
|------|-----|------|
| segmentDefinition | SegmentDefinition | セグメント定義 |
| conditions | SegmentCondition[] | 抽出条件 |

## 出力

| 出力 | 型 | 説明 |
|------|-----|------|
| segment | Segment | セグメント |
| memberCount | Long | 対象顧客数 |
| sampleMembers | CustomerProfile[] | サンプル顧客 |

## ビジネスルール

| ルール | 説明 |
|--------|------|
| 条件検証 | 条件の論理的妥当性検証 |
| サイズ制限 | 最小10名以上（個人特定防止） |
| 更新頻度 | セグメントタイプに応じた更新 |
