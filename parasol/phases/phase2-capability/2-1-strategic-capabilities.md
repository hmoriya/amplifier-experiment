## 入力：価値定義からの要求

### ビジョンKPI（北極星指標）
- ワクワク創造指数（WCI）: 95ポイント目標（2030年）
  - 顧客価値: 40%
  - 環境価値: 30%
  - 未来価値: 30%

### 価値の3つの柱
1. **顧客価値「ワクワク」**: 日常に小さな幸せを提供
2. **環境価値「ありがとう」**: 持続可能なビジネスモデル
3. **未来価値「約束」**: 次世代への責任

## タスク：L1戦略的能力の定義

### 実行ステップ

#### Step 1: 価値から能力への変換
価値定義の各要素を実現するために必要な最上位の組織能力を特定します。

```
価値要素 → 必要な能力
----------------------------------------
ワクワク（顧客価値） → 顧客体験革新能力
ありがとう（環境価値） → サステナビリティ実現能力
約束（未来価値） → 長期価値創造能力
```

#### Step 2: L1能力の構造化

以下の5つのL1戦略的能力を定義：

### L1-001: 持続可能価値創造エクセレンス
```
定義: 環境・社会・経済の三側面で同時に価値を創造する統合的能力
重要性: ビジョンの中核を実現
成功指標:
  - WCI: 95ポイント
  - ESG評価: AAA
  - 企業価値: 2兆円
```

### L1-002: グローバル市場リーダーシップ
```
定義: 国内外市場でカテゴリーリーダーとなる市場創造・獲得能力
重要性: 成長エンジンとして機能
成功指標:
  - グローバル売上比率: 40%
  - 新製品成功率: 50%+
  - ブランド価値: Top10
```

### L1-003: 革新的エコシステム構築
```
定義: 多様なステークホルダーとの共創による新価値創出能力
重要性: イノベーション基盤
成功指標:
  - エコシステム参加: 2,000社
  - 共創プロジェクト: 年100件
  - オープンイノベーション: 売上20%
```

### L1-004: デジタル変革リーダーシップ
```
定義: デジタル技術によるビジネスモデル変革能力
重要性: 変革推進力
成功指標:
  - デジタル売上比率: 30%
  - AI活用率: 80%
  - データ価値創出: 500億円/年
```

### L1-005: レジリエント経営基盤
```
定義: 環境変化に適応し持続的に成長する経営基盤
重要性: 持続性確保
成功指標:
  - 危機対応: 24時間以内
  - BCP達成率: 100%
  - 変化適応: 業界最速
```

#### Step 3: 能力間の相互依存性

```
L1-001 → L1-002: 価値創造が市場優位性を生む
L1-002 → L1-003: 市場展開がエコシステムを拡大
L1-003 → L1-004: 共創がデジタル化を加速
L1-004 → L1-005: デジタルがレジリエンスを強化
L1-005 → L1-001: 安定基盤が価値創造を支える
```

#### Step 4: 成熟度評価と開発計画

現在の成熟度レベル（1-5スケール）:
