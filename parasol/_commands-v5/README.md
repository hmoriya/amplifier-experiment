# Parasol V5 統合コマンド体系

## 概要

Parasol V5は、価値トレーサビリティシステムと完全統合した新しいコマンド体系を提供します。すべてのコマンドが価値駆動で設計され、構造的必然性を自動検証し、想像の設計を完全に排除します。

## 核心思想

### 1. 価値を中心とした設計
- すべてのコマンドが価値トレースを自動記録
- 各操作が価値にどう貢献するかを可視化
- 価値の流れを追跡可能

### 2. 構造的必然性の保証
- コマンド実行時に自動的に必然性をチェック
- 想像の設計を検出してブロック
- 価値に基づいた判断を強制

### 3. マイルストーン駆動の実行
- MS1-MS5の順序を自動的に保証
- 品質ゲートによる自動検証
- 段階的な価値実現を支援

### 4. 統合的な可視化
- 価値実現状況をリアルタイムで把握
- 構造的健全性を継続的にモニタリング
- 包括的なレポート機能

## コマンドカテゴリ

### 1. 価値管理コマンド群
価値の定義、追跡、検証を行うコマンド群

### 2. マイルストーン管理コマンド群
MS1-MS5の進行管理と品質保証を行うコマンド群

### 3. 統合実行コマンド群
複数のフェーズを統合的に実行する高速実行コマンド群

### 4. 分析・レポートコマンド群
プロジェクトの健全性と価値実現状況を分析・報告するコマンド群

## 主な特徴

### 自動化された価値管理
- 価値トレースの自動記録
- 価値継承の自動チェック
- 価値実現の進捗追跡

### インテリジェントな検証
- 構造的必然性の自動検証
- 想像の設計の自動検出・ブロック
- リアルタイムフィードバック

### 統合的なワークフロー
- ワンコマンドで複数フェーズを実行
- 依存関係の自動解決
- エラーからの自動回復

### 包括的な可視化
- 価値フローの可視化
- マイルストーン進捗の表示
- 健全性ダッシュボード

## はじめ方

```bash
# クイックスタート（MS1まで自動実行）
amplifier parasol:quick-start --project my-project

# 完全設計（MS1-MS3を一括実行）
amplifier parasol:full-design --project my-project

# プロジェクト健全性診断
amplifier parasol:project-health --project my-project
```
