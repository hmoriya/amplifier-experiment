# CL2 Sub-capability Design（サブケイパビリティ設計）

**プロジェクト**: サントリーホールディングス / SCM
**Parasol Version**: V5
**作成日**: 2025-01-15
**フェーズ**: Phase 3-2 CL2 Sub-capability Design

---

## 概要

CL1ドメイン分類で特定した5ドメインを、TVDC分類に基づきサブケイパビリティ（CL2）に分解します。

### CL2分解方針（CL1からの継承）

| CL1ドメイン | TVDC分類 | CL2分解方針 | 目標CL2数 |
|------------|---------|-------------|----------|
| demand-management | Core | 機能別に分解 | 3-4 |
| logistics-optimization | Core | 機能別に分解 | 4-5 |
| quality-assurance | VCI | 規制要件別に分解 | 2-3 |
| sustainability | Core/Supporting | Core（水源）とSupporting（ESG）を分離 | 3-4 |
| order-management | Generic | 最小構成 | 1-2 |

## CL1-01: demand-management（需要管理）[Core]

### ドメイン概要

顧客需要の予測と在庫・生産計画の最適化を担う中核ドメイン。サントリーの安定供給価値を実現する最重要領域。

### CL2サブケイパビリティ

#### CL2-01-01: demand-forecasting（需要予測）[Core]

| 項目 | 内容 |
|------|------|
| **定義** | AI/ML活用による需要予測の実行と精度向上 |
| **価値トレース** | VL2-1（安定供給価値）→ VL3-1-1（欠品ゼロ） |
| **TVDC分類** | Core（差別化★★★、価値必然性★★★） |
| **投資配分** | 10% |
| **差別化要因** | 「計画性」重視、AI予測精度での競合優位 |

**主要機能**:
- 販売実績データ分析
- 気象・イベント外部データ連携
- 季節・プロモーション需要予測
- 予測精度のフィードバック改善

**サントリー固有要件**:
- 飲料/酒類の季節変動対応（夏季飲料需要、年末酒類需要）
- 新製品発売時の予測モデル構築
- グループ内製造拠点間の需要分散最適化

---

#### CL2-01-02: inventory-optimization（在庫最適化）[Core]

| 項目 | 内容 |
|------|------|
| **定義** | 全拠点の適正在庫水準維持と最適配置 |
| **価値トレース** | VL2-2（効率化価値）→ VL3-2-1（物流コスト最適化） |
| **TVDC分類** | Core（差別化★★☆、価値必然性★★★） |
| **投資配分** | 8% |
| **差別化要因** | 倉庫約300拠点の統合最適化による効率化 |

**主要機能**:
- 安全在庫水準の自動計算
- 拠点間在庫移動の最適化
- 賞味期限管理・先入先出
- 在庫回転率の改善分析

**サントリー固有要件**:
- 60工場＋300倉庫の在庫可視化
- 飲料の賞味期限管理（約1年）
- 酒類の長期熟成在庫管理

---

#### CL2-01-03: production-planning（生産計画）[Core]

| 項目 | 内容 |
|------|------|
| **定義** | 需要予測に基づく製造計画の立案と調整 |
| **価値トレース** | VL2-1（安定供給価値）→ VL3-1-2（リードタイム短縮） |
| **TVDC分類** | Core（差別化★★☆、価値必然性★★★） |
| **投資配分** | 7% |
| **差別化要因** | 酒類・飲料統合による生産効率化 |

**主要機能**:
- 製造スケジュール立案
- 原材料所要量計算（MRP）
- 製造ライン能力管理
- 委託製造工場連携

**サントリー固有要件**:
- 国内約60工場の統合計画
- 委託製造工場（OEM）との連携
- 季節製品の生産集中期対応

## CL1-02: logistics-optimization（物流最適化）[Core]

### ドメイン概要

2024年問題対応と輸送効率化を実現する中核ドメイン。サントリーの「スマートロジスティクス」戦略を支える差別化領域。

### CL2サブケイパビリティ

#### CL2-02-01: transport-planning（輸送計画）[Core]

| 項目 | 内容 |
|------|------|
| **定義** | 輸送計画の立案と最適化、2024年問題対応 |
| **価値トレース** | VL2-2（効率化価値）→ VL3-2-2（輸送能力確保） |
| **TVDC分類** | Core（差別化★★★、価値必然性★★★） |
| **投資配分** | 8% |
| **差別化要因** | 「スマートロジスティクス」による計画性重視 |

**主要機能**:
- 配車計画の自動作成
- 積載効率の最適化
- ドライバー労働時間管理（960時間上限）
- 荷待ち時間の削減管理

**サントリー固有要件**:
- パレット効率化（35→45ケース、29%向上）
- 異業種共同輸送（大王グループ連携）
- 中継拠点活用による長距離対応

## CL1-03: quality-assurance（品質保証）[VCI]

### ドメイン概要

食品安全・トレーサビリティを確保するVCIドメイン。サントリーブランドへの信頼を支える基盤的品質機能。

### CL2サブケイパビリティ

#### CL2-03-01: food-safety-compliance（食品安全コンプライアンス）[VCI]

| 項目 | 内容 |
|------|------|
| **定義** | 食品衛生法・HACCP準拠の品質管理体制 |
| **価値トレース** | VL2-3（品質保証価値）→ VL3-3-1（HACCP準拠） |
| **TVDC分類** | VCI（差別化★☆☆、価値必然性★★★） |
| **投資配分** | 10% |
| **VCI理由** | 規制要件必須、失敗は致命的だが差別化困難 |

**主要機能**:
- HACCP管理計画の運用
- 重要管理点（CCP）のモニタリング
- 逸脱時の是正措置
- 記録・文書管理

**規制要件**:
- 食品衛生法（厚生労働省）
- HACCP義務化（2021年完全施行）
- ISO 22000（任意だが推奨）

**サントリー固有要件**:
- 飲料製造の殺菌工程管理
- 酒類製造の発酵工程管理
- 委託製造工場の品質監査

## 設計判断と根拠

### 1. Core領域への重点投資（56%）

**判断**: demand-management + logistics-optimization + water-source-management を Core として最重点投資

**根拠**:
- サントリーの「スマートロジスティクス」戦略との整合
- 2024年問題対応での先行優位構築
- 「天然水の森」による持続可能性差別化

### 2. VCI領域の品質重視（26%）

**判断**: quality-assurance + warehouse-management を VCI として確実性最優先

**根拠**:
- 食品安全は失敗許容度ゼロ
- リコール対応のトレーサビリティは事業継続に直結
- 倉庫管理は物流全体の品質基盤

### 3. Generic領域のコスト最適化（10%）

**判断**: order-management を Generic として標準化・外部調達活用

**根拠**:
- 受発注は業界標準EDIで十分
- パートナー連携もAPI標準化で対応
- 内製する差別化理由なし
