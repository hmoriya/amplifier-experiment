## 2. セグメント構造

### パターン判定

**質問1**: 顧客は何種類いますか？
- [x] 1種類 → 標準型
- [ ] 2種類（相互依存） → 両面市場型
- [ ] 3種類以上 → 多面ステークホルダー型

**質問2**: 顧客間に相互依存関係がありますか？
- SCMコンテキストでは、主要顧客は「流通（卸売・小売）」
- 最終消費者は間接顧客

**質問3**: 内部ステークホルダーはサービス品質に直接影響しますか？
- 製造・物流部門は製品品質・納期に直接影響
- 但し、SCMコンテキストでは外部向けVStrが主体

### 判定結果

**パターン**: **標準型**（SCMコンテキスト）

**理由**:
- SCMの価値提供先は主に流通チャネル（卸売・小売）
- 最終消費者への価値は製品を通じた間接提供
- B2B型のサプライチェーンが分析対象

## 4. SCM価値チェーン分析

### バリューチェーン構造

| 段階 | 主要活動 | 関連セグメント | 価値創造 |
|------|---------|--------------|---------|
| **調達** | 原材料購買、サプライヤー管理 | E（サプライヤー） | 品質確保、コスト最適化 |
| **製造** | 生産計画、品質管理 | 内部 | 製品品質、効率性 |
| **物流** | 倉庫管理、配送 | D（物流パートナー） | 納期遵守、コスト効率 |
| **販売** | 受注、供給 | A, B, C（流通） | 顧客満足、売上 |

### 業界特有の課題（2024年問題）

| 課題 | 影響 | サントリーの対応 |
|------|------|-----------------|
| **ドライバー不足** | 輸送能力23%減（2030年予測） | スマートロジスティクス |
| **待機時間** | 物流効率低下 | traevoによる可視化 |
| **積載効率** | コスト増加 | パレット改善（29%向上） |
| **長距離輸送** | リードタイム悪化 | 異業種共同輸送 |

## 8. 備考

### 業種固有の特徴

- **季節変動**: 夏季の飲料需要は通常の3倍
- **販促連動**: 小売の販促計画に連動した供給が必要
- **鮮度管理**: 賞味期限管理が重要
- **重量物**: ビール・飲料は重量物で積載効率が課題

### 注意事項

- SCMコンテキストでは「最終消費者」は間接ステークホルダー
- 流通（卸売・小売）への価値提供を主軸に設計
- 2024年問題への対応が喫緊の課題
