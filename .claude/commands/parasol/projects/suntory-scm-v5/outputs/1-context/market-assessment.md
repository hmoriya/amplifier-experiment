## 2. 市場動向

### 飲料業界トレンド

| トレンド | 説明 | SCMへの影響 |
|---------|------|------------|
| **健康志向** | 機能性飲料、糖質オフ需要 | 製品SKU増加、在庫管理複雑化 |
| **サステナビリティ** | 環境配慮、リサイクル | 包装資材変更、回収物流 |
| **EC拡大** | オンライン購買増加 | 小口配送増加、ラストマイル |
| **人口減少** | 国内市場縮小 | 効率化必須、海外展開 |

### 物流業界トレンド（2024年問題）

| 課題 | 現状 | 2030年予測 |
|------|------|-----------|
| **ドライバー不足** | 深刻化 | 輸送能力23%減 |
| **労働時間規制** | 年960時間上限 | 更なる規制強化 |
| **物流コスト** | 上昇傾向 | 20-30%上昇見込み |
| **CO2削減要請** | 強化 | カーボンニュートラル対応 |

## 4. サントリーのSCM競争力分析

### 強み（Strengths）

| 強み | 説明 | 競争優位性 |
|------|------|-----------|
| **水源管理** | 「天然水の森」21,000ha | 原材料調達の安定性 |
| **グループ一体運営** | 酒類・飲料の統合物流 | スケールメリット |
| **DX先行投資** | traevo、トレーサビリティ | 効率化の先行優位 |
| **異業種連携** | 大王グループとの共同輸送 | 輸送能力確保 |

### 弱み（Weaknesses）

| 弱み | 説明 | 改善方向 |
|------|------|---------|
| **自販機網** | コカ・コーラに劣後 | 小売チャネル強化 |
| **物流子会社** | 専業物流会社なし | パートナー活用 |
| **グローバルSCM** | 地域ごとに最適化 | 統合推進 |

### 機会（Opportunities）

| 機会 | 説明 | 活用方策 |
|------|------|---------|
| **物流共同化** | 業界横断の効率化 | 異業種連携拡大 |
| **AI・IoT活用** | 需要予測精度向上 | SCM DX加速 |
| **モーダルシフト** | 鉄道・船舶活用 | 長距離輸送代替 |
| **地産地消** | 製造拠点最適化 | リードタイム短縮 |

### 脅威（Threats）

| 脅威 | 説明 | 対応策 |
|------|------|--------|
| **物流コスト上昇** | 人件費・燃料費 | 効率化、値上げ転嫁 |
| **ドライバー不足深刻化** | 輸送能力減 | 自動化、共同輸送 |
| **気候変動** | 原材料リスク | 調達多様化 |
| **規制強化** | 環境規制 | サステナブルSCM |

## 7. Phase 2への示唆

### 市場から導出される価値

| 市場ニーズ | 対応する価値（VL候補） |
|-----------|---------------------|
| 物流効率化 | 輸送コスト最適化 |
| 安定供給 | 欠品ゼロの供給体制 |
| トレーサビリティ | 品質保証・リスク対応 |
| サステナビリティ | 環境配慮型物流 |

### 差別化の方向性

1. **DX活用**: traevo、トレーサビリティでの先行優位維持
2. **異業種連携**: 共同輸送の拡大による能力確保
3. **水源管理**: サステナビリティでの差別化
4. **グループシナジー**: 酒類・飲料統合SCMの深化
