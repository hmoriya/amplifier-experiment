# 制約条件

**プロジェクト**: サントリーホールディングス / SCM
**Parasol Version**: V5
**作成日**: 2025-01-15

---

## 1. 分析の制約条件

### 1.1 情報アクセス制約

| 制約 | 説明 | 対応 |
|------|------|------|
| **非公開情報** | 詳細な物流コスト、在庫水準、KPI | 公開情報から推測 |
| **内部システム** | SCMシステムの詳細仕様 | 一般的なアーキテクチャで想定 |
| **サプライヤー情報** | 個別取引条件、契約内容 | 業界標準で想定 |

### 1.2 利用する情報源

| 情報源 | 用途 | 信頼性 |
|--------|------|--------|
| サントリーHD IR資料 | 経営戦略、財務情報 | 高 |
| プレスリリース | SCM施策、提携情報 | 高 |
| 業界メディア | 物流戦略、取り組み詳細 | 中-高 |
| 業界レポート | 市場動向、競合分析 | 中 |

## 2. ビジネス制約

### 2.1 規制・法令制約

| 制約 | 説明 | 影響範囲 |
|------|------|---------|
| **食品衛生法** | 製造・保管・輸送の衛生管理 | 全サプライチェーン |
| **酒税法** | 酒類の製造・流通規制 | 酒類事業SCM |
| **働き方改革関連法** | ドライバー時間外労働960時間上限 | 物流全般 |
| **容器包装リサイクル法** | 容器の回収・リサイクル義務 | 包装・回収物流 |
| **道路運送法** | 輸送に関する規制 | 配送業務 |
| **倉庫業法** | 倉庫運営に関する規制 | 倉庫管理 |

### 2.2 業界標準・認証制約

| 標準・認証 | 必要性 | 対象領域 |
|-----------|--------|---------|
| HACCP | 必須 | 食品製造・保管 |
| ISO 22000 | 推奨 | 食品安全マネジメント |
| ISO 14001 | 推奨 | 環境マネジメント |
| グリーン物流認証 | 推奨 | 物流事業 |

## 3. 組織・リソース制約

### 3.1 人的リソース

| 制約 | 説明 | 影響 |
|------|------|------|
| **ドライバー不足** | 業界全体で深刻化、2030年に23%減 | 輸送能力 |
| **倉庫作業員不足** | 労働力確保困難 | 倉庫オペレーション |
| **DX人材** | SCM DX推進の専門人材 | デジタル化速度 |

### 3.2 インフラ制約

| 制約 | 説明 | 影響 |
|------|------|------|
| **倉庫容量** | 既存倉庫の物理的容量 | 在庫管理 |
| **配送網** | 既存の配送ルート・拠点 | リードタイム |
| **システム連携** | レガシーシステムとの統合 | DX推進速度 |

### 3.3 物流インフラ制約（2024年問題）

| 制約 | 現状 | 対応策 |
|------|------|--------|
| **年間時間外労働** | 960時間上限 | 効率化、人員増 |
| **連続運転時間** | 4時間以内 | 中継拠点活用 |
| **荷待ち時間** | 削減必須 | 予約システム導入 |

## 4. 技術制約

### 4.1 既存システム制約

| システム | 制約 | 影響 |
|---------|------|------|
| **基幹システム** | グループ各社で異なる | データ連携 |
| **WMS** | 拠点ごとに異なる | 在庫可視化 |
| **TMS** | 複数システム併存 | 配車最適化 |

### 4.2 データ制約

| 制約 | 説明 | 対応 |
|------|------|------|
| **データ粒度** | 拠点・品目で異なる | 標準化推進 |
| **リアルタイム性** | バッチ更新の領域あり | IoT活用 |
| **外部連携** | パートナーとのデータ連携 | API標準化 |

---

## 5. 財務制約

### 5.1 投資制約

| 領域 | 制約 | 考慮事項 |
|------|------|---------|
| **物流DX投資** | ROI必須 | 段階的投資 |
| **自動化設備** | 高額投資 | 優先順位付け |
| **システム刷新** | 既存資産活用 | 段階的移行 |

### 5.2 コスト制約

| 制約 | 説明 | 対応方向 |
|------|------|---------|
| **物流コスト上昇** | 人件費・燃料費増加 | 効率化で吸収 |
| **在庫コスト** | 適正在庫維持 | 需要予測精度向上 |
| **品質コスト** | 品質維持は必須 | コスト削減対象外 |

## 6. プロジェクト固有の制約

### 6.1 スコープ制約

| 含むもの（In Scope） | 含まないもの（Out of Scope） |
|---------------------|---------------------------|
| サントリーグループのSCM全体 | 個別製品の製造工程詳細 |
| 調達→製造→物流→販売の価値チェーン | 海外SCMの詳細 |
| 2024年問題への対応 | 個別システムの技術設計 |
| トレーサビリティ | マーケティング・販促 |

### 6.2 分析深度の制約

| 領域 | 分析深度 | 理由 |
|------|---------|------|
| 価値定義（VL） | 深い | コア分析対象 |
| VStream | 深い | コア分析対象 |
| SCMプロセス | 中程度 | 公開情報ベース |
| システム設計 | 概要 | 今回スコープ外 |

## 8. 制約への対応方針

### 8.1 2024年問題への対応

| 施策 | 内容 | 効果 |
|------|------|------|
| **スマートロジスティクス** | 全体最適・計画性重視 | 効率化 |
| **異業種共同輸送** | 大王グループとの連携 | 能力確保 |
| **パレット効率化** | 35→45ケース/パレット | 29%向上 |
| **車両位置可視化** | traevo導入 | 待機時間削減 |

### 8.2 DX制約への対応

| 施策 | 内容 |
|------|------|
| **段階的導入** | ROIを確認しながら拡大 |
| **パートナー活用** | 日立等との協創 |
| **標準化推進** | データ・プロセスの標準化 |

### 8.3 リソース制約への対応

| 施策 | 内容 |
|------|------|
| **自動化投資** | 倉庫・配送の自動化 |
| **パートナー連携** | 物流会社との戦略的提携 |
| **人材育成** | DX人材の内製化 |
