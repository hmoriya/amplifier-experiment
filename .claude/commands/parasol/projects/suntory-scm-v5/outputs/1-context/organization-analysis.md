## 5. コングロマリット判定

### スコアリング

| 基準 | スコア | 根拠 |
|------|--------|------|
| **市場独立性** | 1/3 | 酒類・飲料・食品は同一顧客基盤（小売・卸売）が多い |
| **技術独立性** | 1/3 | 飲料製造技術は共通基盤、発酵技術等を共有 |
| **財務独立性** | 2/3 | サントリー食品は上場、事業部別P/L管理 |
| **組織独立性** | 1/3 | グループ間人事異動あり、共通機能会社 |
| **M&A履歴** | 2/3 | Beam買収(2014)、オランジーナ買収等 |
| **合計** | **7/15点** | |

### 判定結果

**判定**: **関連多角化**（6-9点）

**特徴**:
- 飲料・酒類という関連性の高い事業ポートフォリオ
- 共通のサプライチェーン基盤を活用
- グループシナジーを積極的に追求

### 推奨軸（3軸選択）

| 優先順位 | 軸 | 理由 |
|---------|-----|------|
| 1 | **fusion-axis** | 酒類・飲料の共通SCM基盤 |
| 2 | **platform-axis** | 物流プラットフォーム共通化 |
| 3 | **capability-axis** | SCM能力の横断的活用 |

## 6. SCMコンテキストにおける組織の特徴

### サプライチェーンの特徴

| 特徴 | 説明 |
|------|------|
| **水源管理** | 自然水源の保全活動（「天然水の森」） |
| **製造拠点** | 国内工場約60拠点（清涼飲料） |
| **物流ネットワーク** | 倉庫約300拠点、トラック配送網 |
| **原材料調達** | 農産物（大麦、ホップ等）、包装資材 |

### 2024年問題への対応

| 施策 | 内容 |
|------|------|
| **スマートロジスティクス** | 全体最適・計画性重視の物流戦略 |
| **traevo導入** | 車両位置情報・作業ステータス可視化 |
| **パレット効率化** | 35ケース→45ケース/パレット（29%向上） |
| **異業種共同輸送** | 大王グループとの関東〜関西間混載 |

### トレーサビリティシステム

2024年1月、日立製作所と協創により、原材料入荷から製造、物流、倉庫保管までを一元管理するチェーントレーサビリティシステムを全拠点で運用開始。

### コンテキスト定義

| 項目 | 内容 |
|------|------|
| **分析対象** | サントリーグループのSCM領域 |
