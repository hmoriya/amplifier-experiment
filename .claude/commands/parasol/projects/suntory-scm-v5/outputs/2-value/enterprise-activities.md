# エンタープライズ活動（Enterprise Activities）

**プロジェクト**: サントリーホールディングス / SCM
**Parasol Version**: V5
**作成日**: 2025-01-15

---

## 概要

このドキュメントは、Value Streams（VS0-VS7）を実現するために必要なエンタープライズ活動（EA）を定義します。
各活動は価値階層（VL3）にマッピングされ、責任部門とKPIが明確化されています。

## 5. EA実行優先度マトリクス

### 価値必然性 × 差別化ポテンシャル

```
            高差別化ポテンシャル
                  │
    ┌─────────────┼─────────────┐
    │  差別化投資  │  戦略投資   │
    │ EA-009,010  │ EA-028,029  │
    │ EA-011,028  │ EA-030,031  │
────┼─────────────┼─────────────┼────
低   │             │             │ 高
必然性│  監視       │  基盤維持   │必然性
    │ EA-003,006  │ EA-016,017  │
    │ EA-007      │ EA-018,019  │
    │             │ EA-020,021  │
    └─────────────┼─────────────┘
                  │
            低差別化ポテンシャル
```

### 優先度分類

| 分類 | 説明 | 該当EA |
|------|------|--------|
| **戦略投資** | 高必然性・高差別化：競争優位の源泉 | EA-028, EA-029, EA-030, EA-031 |
| **差別化投資** | 低必然性・高差別化：選択的強化 | EA-009, EA-010, EA-011 |
| **基盤維持** | 高必然性・低差別化：VCI候補、安定運用 | EA-016, EA-017, EA-018, EA-019, EA-020, EA-021 |
| **監視** | 低必然性・低差別化：効率化対象 | EA-003, EA-006, EA-007 |
