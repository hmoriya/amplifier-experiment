## 2. 価値継承マトリクス

### 2.1 グループバリュー → SCM VL2 マッピング

| グループバリュー | SCM VL2 | 継承内容 | 継承強度 |
|------------------|---------|----------|----------|
| **Growing for Good** | VL2-1: 安定供給価値 | 流通パートナーの成長を支援する安定供給 | ★★★ |
| **Growing for Good** | VL2-2: 効率化価値 | 効率化を通じた顧客競争力の向上 | ★★☆ |
| **やってみなはれ** | VL2-2: 効率化価値 | スマートロジスティクスへの挑戦 | ★★★ |
| **やってみなはれ** | VL2-4: 持続可能性価値 | 異業種共同輸送への先駆的取り組み | ★★☆ |
| **利益三分主義** | VL2-2: 効率化価値 | 効率化の利益を顧客に還元 | ★★★ |
| **利益三分主義** | VL2-4: 持続可能性価値 | 社会へのサステナビリティ貢献 | ★★★ |
| **水と生きる** | VL2-4: 持続可能性価値 | 天然水の森、水源涵養活動 | ★★★ |
| **水と生きる** | VL2-3: 品質保証価値 | 水を原料とする製品の品質保証 | ★★☆ |

### 2.2 継承強度の定義

| 強度 | 意味 | 説明 |
|------|------|------|
| ★★★ | 直接継承 | グループバリューがそのまま反映される |
| ★★☆ | 間接継承 | グループバリューを解釈して適用 |
| ★☆☆ | 弱い継承 | 背景として存在するが直接的ではない |

## 3. 詳細継承分析

### 3.1 「Growing for Good」からの継承

```yaml
source:
  level: L1
  name: "Growing for Good"
  description: "成長を通じた社会貢献、継続的成長で社会を良くする力を大きくする"

inheritance:
  - target: VL2-1（安定供給価値）
    strength: "★★★"
    rationale: |
      流通パートナー（卸売・小売・飲食店）の事業成長を支える安定供給を実現。
      サントリーの成長と顧客の成長が連動するWin-Winの関係を構築。
    manifestation:
      - "欠品ゼロによる販売機会損失の防止"
      - "販促連動出荷による売上最大化支援"
      - "需要予測精度向上による在庫最適化"

  - target: VL2-2（効率化価値）
    strength: "★★☆"
    rationale: |
      物流効率化により顧客の競争力向上に貢献。
      コスト削減分を顧客価格に反映し、顧客の利益に貢献。
    manifestation:
      - "配送効率化によるコスト削減"
      - "EDI連携による発注業務効率化"
```

## 4. 継承の妥当性検証

### 4.1 カバレッジ検証

| VL2 | 継承元バリュー | カバレッジ | 評価 |
|-----|---------------|-----------|------|
| VL2-1: 安定供給価値 | Growing for Good | ★★★ | ✅ 十分 |
| VL2-2: 効率化価値 | やってみなはれ、利益三分主義 | ★★★★★ | ✅ 十分 |
| VL2-3: 品質保証価値 | 水と生きる | ★★☆ | ⚠️ 補強推奨 |
| VL2-4: 持続可能性価値 | 水と生きる、利益三分主義 | ★★★★★★ | ✅ 十分 |

### 4.2 ギャップ分析

**VL2-3（品質保証価値）の継承が弱い理由**:
- グループバリューに「品質」を直接示すものがない
- ただし、パーパスの「人間の生命の輝き」が品質保証の根拠となる

**補強案**:
```
パーパス「人間の生命の輝き」
    │
    ├── 健康に貢献する製品 → 品質保証が必須
    │
    └── 安全な製品の提供 → HACCP、トレーサビリティ
```

## 6. 継承検証チェックリスト

### 6.1 必須チェック項目

| # | チェック項目 | 結果 | 備考 |
|---|-------------|------|------|
| 1 | 全VL2がグループバリューから継承されているか | ✅ | VL2-3は補強推奨 |
| 2 | 継承理由が明確か | ✅ | 各継承に rationale 記載 |
| 3 | 継承強度が評価されているか | ✅ | ★★★/★★☆/★☆☆ で評価 |
| 4 | 想像の設計（根拠なき設計）がないか | ✅ | 全て企業理念に基づく |
| 5 | パーパスとの整合性があるか | ✅ | VL1がパーパスを具現化 |

### 6.2 推奨チェック項目

| # | チェック項目 | 結果 | 備考 |
|---|-------------|------|------|
| 1 | 差別化要素が継承に含まれているか | ✅ | 水と生きる→天然水の森 |
| 2 | コングロマリット特性が考慮されているか | ✅ | fusion-axis採用 |
| 3 | 競合との差異が継承で説明できるか | ✅ | サステナビリティ優位性 |
