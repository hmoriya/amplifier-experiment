## 2. 組織プロファイル

### 2.1 サントリーホールディングス（グループ本社）

| 項目 | 内容 |
|------|------|
| **正式名称** | サントリーホールディングス株式会社 |

| **本社** | 大阪府大阪市北区堂島浜2-1-40 |
| **連結売上高** | 約3兆1,000億円（2023年） |
| **従業員数** | 約40,000人（連結） |
| **グローバル展開** | 世界60以上の国と地域 |

### 2.2 主要事業会社

| 事業会社 | 主要事業 | SCMとの関係 |
|----------|---------|-------------|
| **サントリー食品インターナショナル** | 飲料事業 | 主要顧客（内部） |

| 項目 | 内容 |
|------|------|
| **位置づけ** | グループ横断のSCM最適化を担う |
| **主要機能** | 需要計画、在庫管理、物流最適化、品質保証 |
| **主要顧客** | 流通パートナー（卸売・小売・飲食店） |
| **差別化要素** | スマートロジスティクス、天然水の森、日立協創 |

## 3. コングロマリット評価

### 3.1 多角化スコア

| 評価項目 | スコア | 評価理由 |
|----------|--------|---------|
| 事業領域の多様性 | 3/5 | 飲料・酒類中心、関連多角化 |
| 地理的多様性 | 4/5 | グローバル60カ国展開 |
| 顧客セグメント多様性 | 3/5 | B2B（流通）中心 |
| 技術領域の多様性 | 2/5 | 飲料製造・物流に集中 |
| **合計** | **12/20** | 関連多角化型コングロマリット |

### 3.2 SCMスコープへの影響

- **軸パターン推奨**: fusion-axis（グループ統合価値の提供）
- **理由**: 飲料・酒類の共通物流基盤、品質管理の統一
