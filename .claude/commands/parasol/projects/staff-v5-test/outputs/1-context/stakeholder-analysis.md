# ステークホルダー分析 - スタッフサービス・ホールディングス

## 🎯 主要ステークホルダーマップ

### 1. プライマリステークホルダー（直接的利害関係者）

#### 1.1 求職者・派遣スタッフ
- **属性**: 20代～50代の多様な職種希望者
- **ニーズ**: 
  - 安定的な雇用機会
  - スキルアップ支援
  - ワークライフバランス
  - 公正な評価と報酬
- **構造的変化への影響**:
  - 若手層の減少により希少性が上昇
  - スキル要求の高度化
  - キャリア自律意識の向上

#### 1.2 企業顧客（派遣先企業）
- **属性**: 大手から中小まで幅広い業界・規模
- **ニーズ**:
  - 優秀な人材の迅速な確保
  - 人件費の変動費化
  - 専門スキル人材へのアクセス
  - コンプライアンス対応
- **構造的変化への影響**:
  - 人材獲得競争の激化
  - 正社員採用の困難化
  - DX人材需要の急増

#### 1.3 正社員（スタッフサービスグループ従業員）
- **属性**: 営業、コーディネーター、バックオフィス
- **ニーズ**:
  - 働きがいのある職場環境
  - キャリア成長機会
  - 適正な評価と報酬
  - デジタルツールによる業務効率化
- **構造的変化への影響**:
  - 営業手法のデジタル化必須
  - データ分析能力の要求

### 2. セカンダリステークホルダー（間接的利害関係者）

#### 2.1 株主・投資家
- **ニーズ**: 持続的な企業価値向上、安定配当
- **構造的変化への対応期待**: ビジネスモデルの進化

#### 2.2 政府・規制当局
- **関心事項**: 労働法規遵守、雇用創出、社会保障
- **新たな要求**: 働き方改革推進、同一労働同一賃金

#### 2.3 地域社会
- **期待**: 地域雇用創出、経済活性化
- **構造的課題**: 地方の人材不足深刻化

#### 2.4 教育機関
- **連携ニーズ**: 実践的スキル教育、キャリア支援
- **構造的変化**: リスキリング需要の増大

## 🔍 構造的必然性に基づく価値期待

### 求職者・派遣スタッフの価値期待（構造的必然性: 5.0）
1. **終身雇用崩壊への対応**
   - 複数企業での経験蓄積支援
   - ポータブルスキルの可視化
   - キャリア自律支援

2. **デジタル化対応**
   - オンライン登録・面談の完全対応
   - デジタルスキル習得支援
   - AIマッチングによる最適配置

### 企業顧客の価値期待（構造的必然性: 4.8）
1. **人材不足への対応**
   - 迅速な人材供給体制
   - 質の高い人材プール
   - 長期的パートナーシップ

2. **DX推進支援**
   - デジタル人材の供給
   - 業務プロセス改善提案
   - データ活用による最適配置
