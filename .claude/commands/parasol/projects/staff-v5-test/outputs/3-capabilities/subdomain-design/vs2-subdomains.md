# VS2: スキル開発・キャリア支援 - Subdomain Design (CL2)

プロジェクト: staff-v5-test
Value Stream: VS2 - スキル開発・キャリア支援
ドメインタイプ: Core Domain（競争優位の源泉）
作成日: 2025-12-14

## Value Stream 概要

**目的**: 人材の価値向上とキャリア形成支援を通じて、個人のキャリア自律と市場価値最大化を実現する

**価値の流れ（Phase 2より）**:
- A1: スキルギャップ分析
- A2: 学習プログラム設計
- A3: 研修・OJT実施
- A4: スキル認定
- A5: キャリアパス提示
- A6: 次期案件への活用

---

## サブドメイン一覧

### 1. vs2-skill-assessment
**目的**: 個人の現在スキルと市場要求スキルのギャップを正確に把握・分析

### 2. vs2-learning-management
**目的**: 効果的な学習プログラムの設計・提供・管理

### 3. vs2-career-planning
**目的**: 個人のキャリアパス設計と成長戦略の立案・支援

### 4. vs2-certification-management
**目的**: スキル認定・資格管理とデジタルバッジ発行

### 5. vs2-market-intelligence
**目的**: 労働市場動向分析と将来スキル需要予測

---

## サブドメイン詳細

### 1. vs2-skill-assessment

**目的**: 個人の現在スキルレベルを多面的に評価し、市場要求との差分を明確化する

**ビジネスオペレーション一覧**:
このサブドメインで実行する業務活動：
- **スキル診断実施**: オンラインアセスメント、実技テスト、360度評価
- **スキルギャップ分析**: 現在スキルvs要求スキルの差分算出
- **スキルポートフォリオ管理**: 個人のスキル履歴・証跡の一元管理
- **市場価値算定**: スキルレベルに基づく適正年収・市場価値の推定
- **定期的な再評価**: スキルの経年変化追跡と更新

**対応する活動（Phase 2のValue Flowより）**:
- A1: スキルギャップ分析

**データ所有範囲**:
- スキル評価結果データ
- アセスメント履歴
- スキルマスタ定義
- 評価基準・ルーブリック

**他サブドメインとの依存関係**:
- vs2-market-intelligence: スキル需要データの参照
- vs2-learning-management: 学習推奨のためのギャップ情報提供
- vs2-career-planning: キャリア設計のための現状スキル情報提供

**マイクロサービス候補評価**:
- 独立性: 高（評価ロジックは独立）
- スケーラビリティ: 高（大量の評価実施が想定される）
- チーム境界: 明確（評価専門チーム）
- データ境界: 明確（評価データは独立管理）

**推奨**: 独立マイクロサービス化 ★★★★★

---

### 2. vs2-learning-management

**目的**: 個別最適化された学習体験の設計・提供・効果測定を実現する

**ビジネスオペレーション一覧**:
このサブドメインで実行する業務活動：
- **学習プログラム設計**: カリキュラム作成、コンテンツ選定、学習パス設計
- **学習コンテンツ管理**: 教材作成・更新、外部コンテンツ統合
- **受講管理**: 申込受付、スケジューリング、進捗トラッキング
- **学習効果測定**: 理解度テスト、実務適用度評価、ROI測定
- **講師・メンター管理**: 講師アサイン、評価、フィードバック収集

**対応する活動（Phase 2のValue Flowより）**:
- A2: 学習プログラム設計
- A3: 研修・OJT実施

**データ所有範囲**:
- 学習プログラムマスタ
- コンテンツライブラリ
- 受講履歴・進捗データ
- 学習効果測定結果

**他サブドメインとの依存関係**:
- vs2-skill-assessment: スキルギャップ情報の参照
- vs2-certification-management: 修了証・認定連携
- vs1（人材獲得）: 受講者情報の参照

**マイクロサービス候補評価**:
- 独立性: 高（LMS機能として独立）
- スケーラビリティ: 高（動画配信等の負荷対応）
- チーム境界: 明確（教育専門チーム）
- データ境界: 明確（学習データは独立）

**推奨**: 独立マイクロサービス化 ★★★★★

---

### 3. vs2-career-planning

**目的**: 個人のキャリア目標達成に向けた戦略的な成長支援を提供する

**ビジネスオペレーション一覧**:
このサブドメインで実行する業務活動：
- **キャリアカウンセリング**: 1on1面談、キャリア相談、目標設定支援
- **キャリアパス設計**: 短期・中期・長期のキャリアロードマップ作成
- **成長戦略立案**: 必要スキル特定、学習計画策定、経験機会の設計
- **キャリアチェンジ支援**: 転職・異動に向けた準備支援
- **メンタリング・コーチング**: 継続的な成長支援とフィードバック

**対応する活動（Phase 2のValue Flowより）**:
- A5: キャリアパス提示
- A6: 次期案件への活用

**データ所有範囲**:
- キャリア目標データ
- 成長計画・ロードマップ
- カウンセリング記録
- キャリア達成度評価

**他サブドメインとの依存関係**:
- vs2-skill-assessment: 現在スキル情報の参照
- vs2-market-intelligence: 市場動向・将来需要の参照
- vs3（マッチング）: キャリア目標に基づく案件推奨

**マイクロサービス候補評価**:
- 独立性: 中（他サービスとの連携多い）
- スケーラビリティ: 中（相談業務は人的リソース依存）
- チーム境界: 明確（キャリアコンサルタントチーム）
- データ境界: 明確（キャリア情報は独立）

**推奨**: 独立マイクロサービス化 ★★★★☆

---

### 4. vs2-certification-management

**目的**: スキルの可視化と信頼性確保を通じて市場価値を証明する

**ビジネスオペレーション一覧**:
このサブドメインで実行する業務活動：
- **資格試験管理**: 試験実施、採点、合否判定
- **デジタルバッジ発行**: ブロックチェーン基盤の証明書発行
- **資格認定連携**: 外部資格団体との連携・相互認証
- **スキル証明書管理**: 発行履歴管理、真正性検証
- **有効期限管理**: 更新通知、再認定プロセス

**対応する活動（Phase 2のValue Flowより）**:
- A4: スキル認定

**データ所有範囲**:
- 認定基準・試験問題
- 資格・認定履歴
- デジタルバッジメタデータ
- 外部資格マッピング

**他サブドメインとの依存関係**:
- vs2-learning-management: 修了情報の参照
- vs2-skill-assessment: スキルレベル情報の参照
- 外部システム: 資格団体API連携

**マイクロサービス候補評価**:
- 独立性: 高（認定機能として独立）
- スケーラビリティ: 中（バッジ発行は高負荷想定）
- チーム境界: 明確（認定管理チーム）
- データ境界: 明確（認定データは独立）

**推奨**: 独立マイクロサービス化 ★★★★☆

---

### 5. vs2-market-intelligence

**目的**: 労働市場の動向把握と将来予測により戦略的なスキル開発を支援する

**ビジネスオペレーション一覧**:
このサブドメインで実行する業務活動：

- **スキルトレンド追跡**: 新技術・新スキルの早期発見

**データ所有範囲**:
- 市場調査データ

- 業界トレンドデータ

**他サブドメインとの依存関係**:
- vs2-skill-assessment: 需要スキル情報の提供

- vs5（企業サービス）: 企業ニーズ情報の収集

- 独立性: 高（分析機能として独立）
- スケーラビリティ: 高（大量データ処理）
- チーム境界: 明確（データ分析チーム）
- データ境界: 明確（市場データは独立）

**推奨**: 独立マイクロサービス化 ★★★★★

## マイクロサービス境界の推奨

### このVS内での優先順位

1. **vs2-skill-assessment**: スキル評価は全ての起点となる中核機能
2. **vs2-learning-management**: 学習体験の品質が差別化要因
3. **vs2-market-intelligence**: データ駆動の意思決定基盤
4. **vs2-certification-management**: スキル可視化による信頼性確保
5. **vs2-career-planning**: 人的サービスとシステムの融合領域

### 他VSとの統合ポイント

- VS1（人材獲得）の登録者情報: スキル評価の対象者データ連携
- VS3（マッチング）のスキル要件: キャリアプランに基づく案件推奨
- VS5（企業サービス）の人材ニーズ: 市場需要データの重要ソース

## 技術アーキテクチャ推奨

### 共通基盤
- **API Gateway**: 全サブドメインへの統一アクセスポイント
- **イベントバス**: サブドメイン間の非同期通信
- **認証・認可**: 統一的なアクセス制御

### サブドメイン別技術スタック
- **vs2-skill-assessment**: Node.js + PostgreSQL + Redis（高速評価処理）
- **vs2-learning-management**: Python + MongoDB + S3（大量コンテンツ管理）
- **vs2-career-planning**: Java + PostgreSQL（トランザクション重視）
- **vs2-certification-management**: Go + Blockchain（高信頼性・改ざん防止）
- **vs2-market-intelligence**: Python + BigQuery + ML Platform（大規模分析）

CL3: Bounded Context 定義

このVSのサブドメインのBounded Contextを定義：

```bash
# VS2のサブドメインから優先順に
/parasol:3-capabilities cl3 vs2-skill-assessment
/parasol:3-capabilities cl3 vs2-learning-management
/parasol:3-capabilities cl3 vs2-market-intelligence
/parasol:3-capabilities cl3 vs2-certification-management
/parasol:3-capabilities cl3 vs2-career-planning
```

**重要**: BC定義では以下を明確化します
- ビジネスオペレーションの詳細フロー
- ユビキタス言語（業務用語）の定義
- 集約、エンティティ、値オブジェクトの設計
- 他BCとのインターフェース仕様
