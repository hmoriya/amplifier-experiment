# Strategic Domain Classification (CL1)

プロジェクト: staff-v5-test  
会社: 株式会社スタッフサービス・ホールディングス  
作成日: 2025-01-21

## ドメイン分類サマリー

### Core Domains（競争優位の源泉）
1. **human-acquisition**: 人材獲得・エンゲージメント
2. **career-development**: スキル開発・キャリア支援
3. **matching-optimization**: マッチング・配置最適化
4. **enterprise-services**: 顧客企業サービス

### Supporting Domains（ビジネス支援）
1. **employment-management**: 就業管理・サポート
2. **compliance-risk**: コンプライアンス・リスク管理

### Generic Domains（汎用機能）
1. **governance-management**: 経営管理・ガバナンス
2. **billing-payment**: 請求・決済管理

## 詳細分析

### Core Domain: human-acquisition

**説明**: 求職者・派遣スタッフの獲得と関係構築の中核機能

**関連Value Stream**:
- VS1: 人材獲得・エンゲージメント

**評価**:
- 戦略的重要性: ★★★★★ (5/5)
- 競争優位性: ★★★★★ (5/5)
- 複雑度: ★★★★☆ (4/5)
- カスタマイズ必要性: ★★★★★ (5/5)

**分類理由**:
人材不足時代において、優秀な人材の獲得と継続的なエンゲージメントが事業の生命線。デジタル化による獲得チャネル革新が競争優位を生む。オムニチャネル登録、AIスキル評価、CRM/エンゲージメントプラットフォームなど、独自性の高いシステムが必要。

**推奨アプローチ**: フルカスタム開発、最高品質の投資

---

### Core Domain: career-development

**説明**: 人材の価値向上とキャリア形成支援の中核機能

**関連Value Stream**:
- VS2: スキル開発・キャリア支援

**評価**:
- 戦略的重要性: ★★★★★ (5/5)
- 競争優位性: ★★★★★ (5/5)
- 複雑度: ★★★★★ (5/5)
- カスタマイズ必要性: ★★★★★ (5/5)

**分類理由**:
終身雇用崩壊への必然的対応として、個人のキャリア資産形成支援が差別化の最重要要因。実務連動型スキル開発、デジタルバッジによる可視化、企業ニーズ予測に基づくプログラム設計など、高度にカスタマイズされた機能が必要。

**推奨アプローチ**: フルカスタム開発、継続的な機能拡張

---

### Core Domain: matching-optimization

**説明**: 人材と企業の最適マッチング実現の中核機能

**関連Value Stream**:
- VS3: マッチング・配置最適化

**評価**:
- 戦略的重要性: ★★★★★ (5/5)
- 競争優位性: ★★★★☆ (4/5)
- 複雑度: ★★★★★ (5/5)
- カスタマイズ必要性: ★★★★☆ (4/5)

**分類理由**:
AIによる精度向上が競争力に直結。マッチング精度30%向上、リードタイム14日→5日短縮、ミスマッチ率15%→5%削減という具体的な効果が見込まれる。独自のAIマッチングエンジンとリアルタイムスキルDBが必須。

**推奨アプローチ**: フルカスタム開発、AI/MLへの重点投資

---

### Core Domain: enterprise-services

**説明**: 企業顧客への総合的な人材戦略支援

**関連Value Stream**:
- VS5: 顧客企業サービス

**評価**:
- 戦略的重要性: ★★★★★ (5/5)
- 競争優位性: ★★★★☆ (4/5)
- 複雑度: ★★★★☆ (4/5)
- カスタマイズ必要性: ★★★★★ (5/5)

**分類理由**:
人材戦略パートナーへの進化が事業成長の鍵。データ分析による付加価値提供、労働市場分析レポート、人材需要予測など、B2B DXの推進が競争優位を生む。顧客ごとのカスタマイズが必要。

**推奨アプローチ**: フルカスタム開発、データ分析基盤への投資

---

### Supporting Domain: employment-management

**説明**: 派遣スタッフの就業中支援と管理

**関連Value Stream**:
- VS4: 就業管理・サポート

**評価**:
- 戦略的重要性: ★★★★☆ (4/5)
- 競争優位性: ★★☆☆☆ (2/5)
- 複雑度: ★★★☆☆ (3/5)
- カスタマイズ必要性: ★★★☆☆ (3/5)

**分類理由**:
法令遵守と品質維持に必須だが、差別化は限定的。業界標準的な機能で対応可能な部分が多い。一部の独自プロセスのみカスタマイズが必要。

**推奨アプローチ**: 既製品ベース + 部分カスタマイズ

---

### Supporting Domain: compliance-risk

**説明**: 法令遵守とリスク管理の体制

**関連Value Stream**:
- VS7: コンプライアンス・リスク管理

**評価**:
- 戦略的重要性: ★★★★☆ (4/5)
- 競争優位性: ★★☆☆☆ (2/5)
- 複雑度: ★★★★☆ (4/5)
- カスタマイズ必要性: ★★★☆☆ (3/5)

**分類理由**:
規制強化により重要性増大しているが、競争優位の直接的源泉ではない。業界共通の規制対応が主で、独自性は限定的。迅速な対応体制の構築が重要。

**推奨アプローチ**: 標準ソリューション + 業界特化カスタマイズ

---

### Generic Domain: governance-management

**説明**: グループ全体の統括・管理機能

**関連Value Stream**:
- VS0: 経営管理・ガバナンス

**評価**:
- 戦略的重要性: ★★★☆☆ (3/5)
- 競争優位性: ★☆☆☆☆ (1/5)
- 複雑度: ★★☆☆☆ (2/5)
- カスタマイズ必要性: ★★☆☆☆ (2/5)

**分類理由**:
必須だが差別化要因ではない。標準的な経営管理ツールやERPで対応可能。独自開発の必要性は低い。

**推奨アプローチ**: 既製ERP/経営管理ツールの活用

---

### Generic Domain: billing-payment

**説明**: 請求書発行と決済処理の管理

**関連Value Stream**:
- VS6: 請求・決済管理

**評価**:
- 戦略的重要性: ★★★☆☆ (3/5)
- 競争優位性: ★☆☆☆☆ (1/5)
- 複雑度: ★★☆☆☆ (2/5)
- カスタマイズ必要性: ★☆☆☆☆ (1/5)

**分類理由**:
財務プロセスの標準的な処理で、競争優位を生まない。成熟した外部サービスが多数存在し、独自開発のメリットは少ない。

**推奨アプローチ**: 外部SaaS利用（会計システム連携）

## 戦略的推奨事項

1. **Core Domain集中投資**: 
   - 人材獲得とキャリア開発に最優先で投資
   - AIマッチング技術の早期確立
   - B2B向けデータ分析サービスの強化

2. **Supporting Domain効率化**: 
   - 就業管理は既製品をベースに必要最小限のカスタマイズ
   - コンプライアンスは業界標準ソリューションを活用

3. **Generic Domain外注**: 
   - 経営管理はERP導入で対応
   - 請求・決済は外部SaaSを活用

4. **技術的負債回避**: 
   - Core Domainでの技術的負債は競争力を直接損なうため、品質を最優先
   - アジャイル開発と継続的改善の文化醸成

## 次のステップ

CL2: サブドメイン設計

Core Domainから優先的にサブドメイン分解を実施：

```bash
# Core Domain のVSから開始（優先度高）
/parasol:3-capabilities cl2 VS1    # 人材獲得・エンゲージメント
/parasol:3-capabilities cl2 VS2    # スキル開発・キャリア支援
/parasol:3-capabilities cl2 VS3    # マッチング・配置最適化
/parasol:3-capabilities cl2 VS5    # 顧客企業サービス

# Supporting Domain のVS
/parasol:3-capabilities cl2 VS4    # 就業管理・サポート
/parasol:3-capabilities cl2 VS7    # コンプライアンス・リスク管理

# Generic Domain のVS
/parasol:3-capabilities cl2 VS0    # 経営管理・ガバナンス
/parasol:3-capabilities cl2 VS6    # 請求・決済管理
```

人材サービス業界のDXにおいて、Core Domainへの集中投資と、AIを活用した差別化が成功の鍵となります。
