# Parasol V5.7 新機能提案: 組織階層価値継承（Organizational Value Inheritance）

**提案日**: 2025-01-14
**提案背景**: アサヒグループHD → アサヒ飲料 → カルピスブランド検証から発見

---

## 1. 課題認識

### V5.6の限界

現在のV5.6では、組織階層（HD → OpCo → Brand）は**コンテキスト情報**として記録されるが、価値定義との連動は明示的にサポートされていない。

```yaml
# 現在のV5.6: 組織スコープは単なるコンテキスト
organization_scope:
  holding_company: アサヒグループホールディングス
  operating_company: アサヒ飲料
  brand: カルピス

# 価値定義はブランドレベルのみで開始
value_hierarchy:
  VL1-1:
    description: "心身の健やかさを支える"
    # 上位組織の価値との関係が不明
```

### 実際のビジネスニーズ

企業グループでは、以下のような価値の継承・特殊化が行われている：

| 組織階層 | 価値の抽象度 | 顧客セグメント |
|----------|-------------|---------------|
| HD（持株会社） | 最も抽象的・普遍的 | 全世界・全顧客 |
| OpCo（事業会社） | 事業領域で特殊化 | 地域・カテゴリ限定 |
| Brand（ブランド） | 最も具体的 | 特定ターゲット |

この**価値の継承と特殊化**、**顧客セグメントの絞り込み**を明示的にモデル化する仕組みがない。

## 2. 提案: Organizational Value Inheritance (OVI)

### 2.1 概念図

```
┌─────────────────────────────────────────────────────────────────┐
│                    アサヒグループHD (VL0)                        │
│  ════════════════════════════════════════════════════════════   │
│  本質的価値: 食を通じて世界の人々の健康で豊かな生活に貢献       │
│  顧客セグメント: 全世界の消費者                                  │
│  価値次元: 健康(30%) + 豊かさ(30%) + 品質(40%)                  │
└───────────────────────────┬─────────────────────────────────────┘
                            │ 継承
                            │ inheritance_type: specialization
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│                    アサヒ飲料 (VL0-OpCo)                         │
│  ════════════════════════════════════════════════════════════   │
│  継承元: アサヒグループHD.VL0                                    │
│  特殊化: 「食」→「飲料」、「世界」→「日本」                     │
│  本質的価値: 飲料を通じて日本の生活者の健康と潤いに貢献         │
│  顧客セグメント: 日本の飲料消費者                                │
│  価値次元: 健康(35%) + 潤い(25%) + 品質(40%)                    │
│  セグメント変換: 全世界 → 日本市場                              │
└───────────────────────────┬─────────────────────────────────────┘
                            │ 継承
                            │ inheritance_type: specialization
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│                    カルピス (VL1)                                 │
│  ════════════════════════════════════════════════════════════   │
│  継承元: アサヒ飲料.VL0-OpCo                                     │
│  特殊化: 「飲料」→「乳酸菌発酵飲料」、「生活者」→「家族」      │
│  本質的価値: 乳酸菌発酵で心身の健やかさと家族の絆を支える       │
│  顧客セグメント: 健康意識のある家族層                            │
│  価値次元: 健康(40%) + 家族絆(30%) + 伝統(30%)                  │
│  セグメント変換: 日本市場 → 家族層                              │
└─────────────────────────────────────────────────────────────────┘
```

### 2.2 継承タイプ

| 継承タイプ | 説明 | 例 |
|-----------|------|-----|
| **specialization** | 抽象概念を具体化 | 「食」→「飲料」→「乳酸菌飲料」 |
| **localization** | 地域・市場を限定 | 「世界」→「日本」→「家族層」 |
| **extension** | 新しい価値次元を追加 | +「家族の絆」 |
| **refinement** | 既存価値の比重変更 | 健康30%→40% |

### 2.3 セグメント絞り込みルール

```yaml
segment_narrowing:
  rule: "子は親より広いセグメントを持てない"
  validation:
    - child_segment ⊆ parent_segment
    - segment_size: parent > child

  example:
    HD: "全世界消費者" (100%)
    OpCo: "日本の飲料消費者" (~2%)
    Brand: "健康意識のある家族層" (~0.3%)
```

## 3. YAML仕様案

### 3.1 HD価値定義

```yaml
# phase0-holding-value.yaml
holding_value_definition:
  organization: アサヒグループホールディングス
  level: VL0

  core_value:
    id: VL0-HD
    description: "食を通じて世界の人々の健康で豊かな生活に貢献"
    why: "人々の生活に不可欠な食を提供する企業グループとしての存在意義"

  customer_segment:
    scope: global
    description: "全世界の消費者"
    size_indicator: "100%"

  value_dimensions:
    健康: 30%
    豊かさ: 30%
    品質: 40%
```

### 3.2 OpCo価値定義（継承）

```yaml
# phase1-opco-value.yaml
operating_company_value_definition:
  organization: アサヒ飲料
  level: VL0-OpCo

  inheritance:
    parent: VL0-HD
    type: specialization
    transformations:
      - from: "食"
        to: "飲料"
        rationale: "飲料事業への特化"
      - from: "世界"
        to: "日本"
        rationale: "国内飲料市場への集中"

  core_value:
    id: VL0-OpCo
    description: "飲料を通じて日本の生活者の健康と潤いに貢献"
    inherited_from: VL0-HD
    specialization: "食→飲料、世界→日本"

  customer_segment:
    scope: japan_domestic
    description: "日本の飲料消費者"
    size_indicator: "~2%"
    parent_segment: "全世界の消費者"
    narrowing_rationale: "日本市場への集中戦略"

  value_dimensions:
    健康: 35%  # 継承: 30% → 35% (+5%)
    潤い: 25%  # 新規追加（豊かさから派生）
    品質: 40%  # 継承: 維持
```

### 3.3 Brand価値定義（継承）

```yaml
# phase2-brand-value.yaml
brand_value_definition:
  organization: カルピス
  level: VL1

  inheritance:
    parent: VL0-OpCo
    type: specialization
    transformations:
      - from: "飲料"
        to: "乳酸菌発酵飲料"
        rationale: "カルピスの製品特性"
      - from: "生活者"
        to: "家族"
        rationale: "家族での消費シーン"

  core_value:
    id: VL1-1
    description: "乳酸菌発酵で心身の健やかさと家族の絆を支える"
    inherited_from: VL0-OpCo
    specialization: "飲料→乳酸菌発酵、生活者→家族"

  customer_segment:
    scope: japan_family
    description: "健康意識のある家族層"
    size_indicator: "~0.3%"
    parent_segment: "日本の飲料消費者"
    narrowing_rationale: "ターゲット家族層への集中"
    demographics:
      - 子育て世代（30-45歳）
      - 健康意識が高い
      - 家族のコミュニケーションを重視

  value_dimensions:
    健康: 40%  # 継承: 35% → 40% (+5%)
    家族絆: 30%  # 新規追加（ブランド固有）
    伝統: 30%  # 新規追加（100年ブランド）
```

## 4. 継承検証ルール

### 4.1 必須検証

```yaml
inheritance_validation:
  rules:
    - name: "セグメント包含"
      check: child_segment ⊆ parent_segment
      error: "子セグメントは親セグメントの部分集合でなければならない"

    - name: "価値継承"
      check: child_value derives_from parent_value
      error: "子の価値は親の価値から派生しなければならない"

    - name: "特殊化方向"
      check: specificity(child) > specificity(parent)
      error: "子は親より具体的でなければならない"
```

### 4.2 警告レベル

```yaml
inheritance_warnings:
  - name: "価値次元の大幅変更"
    condition: |dimension_change| > 20%
    message: "価値次元の比重が大幅に変更されています"

  - name: "新規価値次元"
    condition: new_dimension_added
    message: "親にない価値次元が追加されています。継承との整合性を確認してください"
```

## 6. V5.6との差分

| 機能 | V5.6 | V5.7(提案) |
|------|------|-----------|
| 組織階層 | コンテキストのみ | 価値継承の起点 |
| 価値定義開始点 | ブランドレベル | HD/OpCoから選択可 |
| 顧客セグメント | ブランド固有 | 継承・絞り込み |
| 価値継承 | なし | 明示的サポート |
| 継承検証 | なし | 自動検証ルール |

## 7. 期待効果

### 7.1 戦略整合性

```
HD戦略 → OpCo戦略 → Brand戦略
   │          │          │
   └──────────┴──────────┘
         整合性を自動検証
```

### 7.2 投資判断の一貫性

- HD: グループ全体の価値投資方針
- OpCo: 事業領域での投資配分
- Brand: ターゲット固有の投資優先度

全階層で**価値の継承**が明確なため、投資判断の根拠が上位から下位まで一貫する。

### 7.3 M&A・事業再編への対応

新ブランド取得時：
1. HD価値との整合性を検証
2. OpCo価値への位置づけを決定
3. 継承関係を明示化

## 9. 結論

**V5.7 Organizational Value Inheritance (OVI)** は、企業グループにおける価値の継承と特殊化を明示的にモデル化する機能である。

これにより：
1. **HD→OpCo→Brand**の価値継承が追跡可能に
2. **顧客セグメント絞り込み**が論理的に検証可能に
3. **投資判断の一貫性**が組織階層全体で担保される

カルピス検証から発見された重要なギャップであり、Parasol V5.7の主要新機能として提案する。
