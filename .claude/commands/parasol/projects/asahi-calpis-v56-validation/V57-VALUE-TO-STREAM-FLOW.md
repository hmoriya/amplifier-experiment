## 2. カルピス事例での完全フロー

### Step 1: HD価値定義（最も抽象的）

```yaml
# Phase 0.5 - Step 1
holding_value:
  id: VL0-HD
  organization: アサヒグループホールディングス

  value_statement: "食を通じて世界の人々の健康で豊かな生活に貢献"

  abstraction_level: highest

  customer_scope:
    description: "全世界の食品・飲料消費者"
    size: "100%"
    characteristics:
      - 食品・飲料を消費するすべての人
      - 地域・年齢・性別を問わない
      - 文化・習慣の多様性を包含

  core_value_elements:
    - element: "食"
      abstraction: "人間の基本的ニーズ"
    - element: "健康"
      abstraction: "身体的・精神的wellbeing"
    - element: "豊かな生活"
      abstraction: "QOL向上全般"
    - element: "世界"
      abstraction: "グローバルスケール"
```

### Step 2: OpCo価値継承（事業領域特化）

```yaml
# Phase 0.5 - Step 2
opco_value:
  id: VL0-OpCo-Beverages
  organization: アサヒ飲料

  inheritance:
    parent: VL0-HD
    type: specialization

  transformations:
    - from: "食"
      to: "飲料"
      rationale: "飲料事業への領域特化"

    - from: "世界"
      to: "日本"
      rationale: "国内市場へのフォーカス"

    - from: "豊かな生活"
      to: "潤い"
      rationale: "飲料ならではの価値表現"

  value_statement: "飲料を通じて日本の生活者の健康と潤いに貢献"

  abstraction_level: high

  customer_scope:
    description: "日本の清涼飲料消費者"
    size: "~2%"
    narrowing_from: "全世界消費者"
    narrowing_dimension: "地域 × カテゴリ"
    characteristics:
      - 日本在住
      - 清涼飲料を購入・消費する
      - 年齢・性別は広範
```

### Step 3: Brand価値継承（セグメント特化）

```yaml
# Phase 0.5 - Step 3
brand_value:
  id: VL1-Calpis
  organization: カルピス

  inheritance:
    parent: VL0-OpCo-Beverages
    type: segmentation

  transformations:
    - from: "飲料"
      to: "乳酸菌発酵飲料"
      rationale: "カルピスの製品特性"

    - from: "生活者"
      to: "家族"
      rationale: "家族での消費シーン特化"

    - from: "健康"
      to: "心身の健やかさ"
      rationale: "より包括的な健康観"

    - add: "家族の絆"
      rationale: "カルピス固有の情緒的価値"

  value_statement: "乳酸菌発酵で心身の健やかさと家族の絆を支える"

  abstraction_level: medium

  customer_scope:
    description: "健康意識のある家族層"
    size: "~0.3%"
    narrowing_from: "日本の清涼飲料消費者"
    narrowing_dimension: "ライフスタイル × 家族構成"

    detailed_segment:
      demographics:
        age_primary: 30-45歳（購買決定者）
        family_structure: 子供あり
        income: 中〜中上
      psychographics:
        health_consciousness: 高
        family_orientation: 高
        tradition_value: 中〜高
      behaviors:
        purchase_occasion: 日常・季節イベント
        consumption_scene: 家庭内・家族一緒
```

### Step 4: VL1→VL2→VL3 分解（価値の具体化）

```yaml

value_decomposition:

  # VL1（ブランド本質的価値）
  VL1-1:
    description: "心身の健やかさを支える"
    inherited_from: Brand価値の健康側面

    # VL2（中核価値）への分解
    children:
      VL2-1-1:
        description: "乳酸菌による健康価値"
        value_type: functional

        # VL3（詳細価値）への分解
        children:
          VL3-1-1-1:
            description: "カルピス菌による独自発酵"
            vc_pattern: product_led  # VC1主導
            tangibility: high

          VL3-1-1-2:
            description: "腸内環境への好影響"
            vc_pattern: info_led  # VC5主導
            tangibility: medium

          VL3-1-1-3:
            description: "機能性成分（L-92乳酸菌等）"
            vc_pattern: product_led  # VC1主導
            tangibility: high

      VL2-1-2:
        description: "安心・信頼の品質"
        value_type: functional

        children:
          VL3-1-2-1:
            description: "100年以上の製造実績"
            vc_pattern: relation_led  # VC4主導
            tangibility: low

          VL3-1-2-2:
            description: "厳格な品質管理"
            vc_pattern: product_led  # VC1主導
            tangibility: medium

  VL1-2:
    description: "世代を超えた幸福な記憶を紡ぐ"
    inherited_from: Brand価値の家族の絆側面

    children:
      VL2-2-1:
        description: "懐かしさと新しさの共存"
        value_type: emotional

        children:
          VL3-2-1-1:
            description: "伝統の味わい"
            vc_pattern: experience_led  # VC3主導
            tangibility: high

          VL3-2-1-2:
            description: "季節限定フレーバー"
            vc_pattern: experience_led  # VC3主導
            tangibility: high

        description: "家族のコミュニケーション触媒"
        value_type: emotional

        children:
          VL3-2-2-1:
            description: "一緒に作る・割る体験"
            vc_pattern: relation_led  # VC4主導
            tangibility: high
```

### Step 5: VL3からValue Stream設計への接続

```yaml
# Phase 3 - Value Stream設計
value_stream_design:

  design_principle: |
    VL3で定義された詳細価値を、
    ターゲット顧客（健康意識のある家族層）に
    どのようなジャーニーで届けるかを設計する

  # VL3 → Value Stream マッピング
  vl3_to_vstream_mapping:

    # 認知・選好段階で活性化するVL3
    awareness_preference_stage:
      target_vs: VS0 → VS2
      value_stream: VStr-1（ブランド認知・選好）
      activated_vl3:
        - VL3-1-1-2: "腸内環境への好影響"  # 健康訴求
        - VL3-1-2-1: "100年以上の製造実績"  # 信頼訴求
        - VL3-2-1-1: "伝統の味わい"  # 懐かしさ訴求

    # 初回体験段階で活性化するVL3
    first_experience_stage:
      target_vs: VS2 → VS3
      value_stream: VStr-2（初回体験）
      activated_vl3:
        - VL3-1-1-1: "カルピス菌による独自発酵"  # 製品体験
        - VL3-2-1-1: "伝統の味わい"  # 味覚体験
        - VL3-2-2-1: "一緒に作る・割る体験"  # 調製体験

    # 継続・定着段階で活性化するVL3
    retention_stage:
      target_vs: VS3 → VS6
      value_stream: VStr-3（継続・定着）
      activated_vl3:
        - VL3-1-1-2: "腸内環境への好影響"  # 継続理由
        - VL3-1-1-3: "機能性成分"  # 差別化維持
        - VL3-2-1-2: "季節限定フレーバー"  # 飽き防止

    # ロイヤルティ段階で活性化するVL3
    loyalty_stage:
      target_vs: VS5 → VS7
      value_stream: VStr-4（ロイヤルティ・推奨）
      activated_vl3:
        - VL3-1-2-1: "100年以上の製造実績"  # 推奨理由
        - VL3-2-2-1: "一緒に作る・割る体験"  # 世代継承
        - VL3-2-1-1: "伝統の味わい"  # 思い出共有

  # Value Stream 詳細定義
  value_streams:
    VStr-1:
      name: "ブランド認知・選好ストリーム"
      customer_state_transition: VS0(未認知) → VS2(選好)

      customer_segment_focus: |
        健康意識のある家族層の中で、
        まだカルピスを認知していない/選好していない層

      value_delivery_goal: |
        HD価値「健康で豊かな生活」を
        「乳酸菌で家族の健康を守る」という
        具体的なブランド価値として認知させる

      operational_stages:
        - OS-1-1:
            name: "ブランドメッセージ発信"
            delivers_vl3: [VL3-1-2-1, VL3-2-1-1]
            capability: CL2-1-1

        - OS-1-2:
            name: "健康価値訴求"
            delivers_vl3: [VL3-1-1-2]
            capability: CL2-1-2

        - OS-1-3:
            name: "購買接点最適化"
            delivers_vl3: [VL3-2-1-1]
            capability: CL2-1-3

    VStr-2:
      name: "初回体験ストリーム"
      customer_state_transition: VS2(選好) → VS3(体験完了)

      customer_segment_focus: |
        カルピスを選好し、初めて購入・体験する家族

      value_delivery_goal: |
        OpCo価値「潤い」を
        「希釈して作る楽しさ」という
        体験価値として提供する

      operational_stages:
        - OS-2-1:
            name: "製品品質確保"
            delivers_vl3: [VL3-1-1-1]
            capability: CL2-2-1

        - OS-2-2:
            name: "体験価値提供"
            delivers_vl3: [VL3-2-2-1, VL3-2-1-1]
            capability: CL2-2-2

    VStr-3:
      name: "継続・定着ストリーム"
      customer_state_transition: VS3(体験完了) → VS6(習慣化)

      customer_segment_focus: |
        カルピスを体験し、継続購入を検討する家族

      value_delivery_goal: |
        Brand価値「心身の健やかさ」を
        継続的な健康効果の実感として定着させる

      operational_stages:
        - OS-3-1:
            name: "健康効果訴求"
            delivers_vl3: [VL3-1-1-2, VL3-1-1-3]
            capability: CL2-3-1

        - OS-3-2:
            name: "体験価値設計"
            delivers_vl3: [VL3-2-1-2]
            capability: CL2-3-2

    VStr-4:
      name: "ロイヤルティ・推奨ストリーム"
      customer_state_transition: VS5(定着) → VS7(推奨者)

      customer_segment_focus: |
        カルピスを習慣的に消費し、他者に勧める家族

      value_delivery_goal: |
        Brand価値「家族の絆」を
        世代を超えた推奨行動として実現する

      operational_stages:
        - OS-4-1:
            name: "ロイヤルティ醸成"
            delivers_vl3: [VL3-1-2-1, VL3-2-2-1]
            capability: CL2-4-1

        - OS-4-2:
            name: "推奨促進"
            delivers_vl3: [VL3-2-1-1]
            capability: CL2-4-2
```

## 4. 価値継承の追跡性（トレーサビリティ）

```
価値トレーサビリティマトリクス
══════════════════════════════

HD価値要素      OpCo価値要素     Brand価値要素    VL3            VStr/OS
─────────────────────────────────────────────────────────────────────────
```
