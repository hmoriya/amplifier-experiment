# 1. 検証概要

## 対象組織階層

```
アサヒグループホールディングス（持株会社）
    └── アサヒ飲料（事業会社）
            └── カルピス（ブランド）
```

## 検証スコープ

- Phase 1: 組織コンテキスト
- Phase 2: 価値定義（VL1→VL3）+ VC分析
- Phase 3: バリューストリーム + OS定義

---

# 2. V5.6 新機能サマリー

| 新機能 | 概要 |
|--------|------|
| **Operational Stage (OS)** | バリューストリーム内の運用段階定義 |
| **Stage依存TVDC** | 顧客ステージでTVDCが変化 |
| **VC分析** | VL3を5つの価値次元で分析 |
| **3軸価値分解** | SH/時間/性質の3軸で分解 |

---

# 新機能①: Operational Stage

## Before（V5.5）
- VStrとCapabilityの関係が**不明確**
- 「いつ」活性化するか定義されていない

## After（V5.6）
- OS経由で**1:1対応**
- 価値蓄積(value_accumulation)を明示

---

# OS-Capability 1:1対応

```
VStr-1: ブランド認知・選好ストリーム
├── OS-1-1 ブランドメッセージ発信
│       └── CL2-1-1 ブランドコミュニケーション [Core]
├── OS-1-2 健康価値訴求
│       └── CL2-1-2 健康エビデンス管理 [VCI]
└── OS-1-3 購買接点最適化
        └── CL2-1-3 購買接点管理 [Supporting]
```

**10個のOS = 10個のCapability**

---

# 新機能②: Stage依存TVDC

## 同一Capabilityが顧客ステージで異なるTVDCを持つ

| Capability | VS0-1 | VS1-2 | VS3-6 |
|------------|-------|-------|-------|
| 健康エビデンス管理 | Supp | **VCI** | **Core** |
| 体験価値設計 | - | Supp | **Core** |
| ブランドコミュニケーション | **Core** | **Core** | Supp |

**投資判断の動的最適化が可能に**

---

# 新機能③: VC分析

## 5つのValue Component

| VC | 内容 | カルピス例 |
|----|------|----------|
| VC1 | 製品価値 | 発酵技術 |
| VC2 | サービス価値 | 入手性 |
| VC3 | 体験価値 | 味わい・希釈 |
| VC4 | 関係価値 | 家族の絆 |
| VC5 | 情報価値 | 健康知識 |

---

# VCパターン分析

| VL3 | パターン | 主VC | 副VC |
|-----|---------|------|------|
| カルピス菌発酵 | 製品主導型 | VC1 | VC3, VC5 |
| 腸内環境好影響 | 情報主導型 | VC5 | VC1, VC4 |
| 伝統の味わい | 体験主導型 | VC3 | VC1, VC4 |
| 一緒に作る体験 | **関係主導型** | VC4 | VC3, VC1 |

# 3. カルピス価値階層

```
VL1-1: 心身の健やかさを支える
├── VL2-1-1: 乳酸菌による健康価値
│     ├── VL3-1-1-1: カルピス菌発酵
│     ├── VL3-1-1-2: 腸内環境好影響
│     └── VL3-1-1-3: 機能性成分
└── VL2-1-2: 安心・信頼の品質

VL1-2: 世代を超えた幸福な記憶を紡ぐ
├── VL2-2-1: 懐かしさと新しさ
└── VL2-2-2: 家族のコミュニケーション触媒
```

---

# 4つのバリューストリーム

```
VStr-1: ブランド認知・選好 (VS0→VS2)
        ↓ 購入決定
VStr-2: 初回体験 (VS2→VS3)
        ↓ 満足体験
VStr-3: 継続・定着 (VS3→VS6)
        ↓ 愛着形成
VStr-4: ロイヤルティ・推奨 (VS5→VS7)
```

---

# 4. 主要な発見

## 発見①: 情緒的価値が機能的価値を上回る

| 性質 | 比率 |
|------|------|
| 情緒的価値 | **45%** |
| 機能的価値 | 35% |
| 社会的価値 | 20% |

**カルピスの競争優位は「機能訴求」より「情緒訴求」にある**

---

# 発見②: Stage依存TVDCの実用性

## 健康エビデンス管理の変化

| ステージ | TVDC | 理由 |
|----------|------|------|
| 認知(VS0-1) | Supporting | 詳細不要 |
| 選好(VS1-2) | **VCI** | 差別化要因 |
| 継続(VS3-6) | **Core** | リピート動機 |

**投資タイミングの論理的根拠を提供**

---

# 発見③: OS定義による責任明確化

## Before
「ブランドコミュニケーションは誰の責任？」
→ マーケティング全体で曖昧

## After
「OS-1-1ブランドメッセージ発信 = CL2-1-1」
→ 特定チームの明確な責任

# 6. 結論

## V5.6は価値駆動型ビジネス設計の質的飛躍

| 観点 | V5.5 | V5.6 |
|------|------|------|
| VStr-Capability | 間接的 | OS経由で1:1 |
| TVDC | 固定 | Stage依存 |
