# Parasol V5.7: バリエーションポイント分析

**Variation Point Analysis - 企業類型別の価値継承パターン**

---

## 1. 核心的洞察

> **企業によって「バリエーションが出るポイント」が異なる**
>
> そのポイントを特定し、継承関係をリサーチすることで、
> **価値創造の方向性**が明らかになる

```
バリエーションポイント = 価値分岐点 = 戦略的投資判断の焦点
```

### 2.2 類型別の継承パターン

```yaml
variation_point_patterns:

  # Type A: コングロマリット型（子会社軸）
  conglomerate:
    variation_point: "子会社/事業会社"
    examples:
      - 三菱商事（商社、金融、不動産、エネルギー...）
      - ソフトバンクグループ（通信、投資、AI、物流...）
      - 伊藤忠商事
      - アサヒグループHD

    inheritance_pattern: |
      HD価値（非常に抽象的）
        ├─► 事業A価値（領域特化）
        ├─► 事業B価値（領域特化）← 完全に異なる領域
        └─► 事業C価値（領域特化）

    strategic_focus: |
      - 各子会社の自律性が高い
      - HD価値は「傘」として機能
      - 投資判断は子会社単位

    context_analysis_focus:
      - 子会社間の関係性（シナジー or 独立）
      - HD価値がどこまで具体的か
      - 子会社の顧客セグメントの重複度

  # Type B: マルチブランド型（製品/ブランド軸）
  multi_brand:
    variation_point: "ブランド/製品ライン"
    examples:
      - P&G（Tide, Pampers, Gillette, SK-II...）
      - 花王（ビオレ、アタック、メリーズ...）
      - ユニリーバ
      - アサヒ飲料（カルピス、三ツ矢、ワンダ...）

    inheritance_pattern: |
      会社価値（事業領域定義）
        └─► 共通基盤価値
              ├─► ブランドA価値（セグメント特化）
              ├─► ブランドB価値（セグメント特化）
              └─► ブランドC価値（セグメント特化）

    strategic_focus: |
      - ブランドポートフォリオ管理
      - セグメント間のカニバリゼーション回避
      - 共通基盤（R&D、製造、物流）の活用

    context_analysis_focus:
      - 各ブランドのターゲットセグメント
      - ブランド間の差別化ポイント
      - 共有する価値要素と固有の価値要素

  # Type C: サービス多角化型（サービス軸）
  multi_service:
    variation_point: "サービスライン"
    examples:
      - リクルート（HR、住宅、旅行、飲食、美容...）
      - 楽天（EC、金融、通信、旅行...）
      - サイバーエージェント（広告、メディア、ゲーム...）
      - Amazon（EC、AWS、Prime、デバイス...）

    inheritance_pattern: |
      会社価値（顧客中心 or テクノロジー中心）
        ├─► サービスA価値（生活領域A）
        ├─► サービスB価値（生活領域B）
        └─► サービスC価値（生活領域C）
              └─► 共通: プラットフォーム/会員基盤

    strategic_focus: |
      - クロスセル・アップセル
      - 会員基盤の共有
      - データ活用によるサービス連携

    context_analysis_focus:
      - サービス間の顧客動線
      - 共通会員基盤の存在
      - サービス間でのデータ連携

  # Type D: 地域展開型（地域軸）
  regional_expansion:
    variation_point: "地域/市場"
    examples:
      - ファーストリテイリング（日本、中国、欧米、東南アジア）
      - トヨタ自動車（各地域法人）
      - マクドナルド（各国法人）

    inheritance_pattern: |
      グローバル価値（普遍的価値）
        ├─► 地域A価値（文化適応）
        ├─► 地域B価値（文化適応）
        └─► 地域C価値（文化適応）
              └─► 共通: ブランドコア

    strategic_focus: |
      - グローバル標準 vs ローカル適応のバランス
      - 地域ごとの顧客ニーズ理解
      - ブランドコアの維持

    context_analysis_focus:
      - グローバル価値の定義
      - 地域適応の範囲（何を変えて良いか）
      - 各地域の顧客特性

  # Type E: 単一ブランド集中型
  single_brand_focus:
    variation_point: "製品ライン/機能"
    examples:
      - Apple（iPhone, Mac, iPad, Watch, Services）
      - ダイソン（掃除機、ドライヤー、空気清浄機）
      - テスラ（Model S, 3, X, Y, Cybertruck）
      - Notion（Free, Plus, Business, Enterprise）

    inheritance_pattern: |
      ブランド価値（強いコア価値）
        ├─► 製品A（コア価値の具現化A）
        ├─► 製品B（コア価値の具現化B）
        └─► 製品C（コア価値の具現化C）
              └─► 共通: デザイン哲学、技術思想

    strategic_focus: |
      - ブランドコアの一貫性
      - 製品間のエコシステム構築
      - プレミアム維持

    context_analysis_focus:
      - ブランドコア価値の明確化
      - 製品ライン間の関係性
      - 顧客のエコシステム内行動
```

```
          │

          │
          └── 事業会社の価値
                  │

    分岐点 = バリエーションポイント
```

## 5. 継承関係リサーチの重要性

### 5.1 継承関係が示すもの

```
継承関係のリサーチ = 価値創造の方向性の発見
════════════════════════════════════════════

継承タイプ          それが示す価値創造の方向性
──────────────────────────────────────────────────────────────

specialization      上位価値の「深掘り」
（特殊化）          → その領域でNo.1を目指す戦略
                    例: 食→飲料→乳酸菌飲料

divergence          上位価値からの「分岐」
（分岐）            → ポートフォリオ多角化戦略
                    例: 食→(酒類|飲料|食品)

segmentation        同一領域内の「棲み分け」
（セグメント化）    → マーケットシェア最大化戦略
                    例: 飲料→(家族向け|若者向け|ビジネス向け)

extension           既存価値への「付加」
（拡張）            → 差別化・プレミアム戦略
                    例: 健康+家族の絆（情緒的価値の追加）

localization        同一価値の「地域適応」
（地域化）          → グローバル展開戦略
                    例: 日本仕様→中国仕様→欧米仕様
```

### 5.2 方向性の判断

## 6. カルピス事例での適用

### 6.1 バリエーションポイント特定

```
アサヒグループのバリエーションポイント分析
═══════════════════════════════════════════

Q1: 最も多様性があるのは？
    → 子会社も多い（ビール、飲料、食品）
    → ブランドも多い（カルピス、三ツ矢、ワンダ...）
    → 複合型（コングロマリット × マルチブランド）

Q2: 価値の分岐点は？
    → 第1分岐: HD → OpCo（事業領域分岐）
    → 第2分岐: OpCo → Brand（セグメント分岐）

    HD「食で健康と豊かさ」
         │
    ┌────┴────┐ 第1分岐（事業領域）
    ↓         ↓
  ビール    飲料 ─────────────────┐
 「嗜好品」「健康と潤い」          │
                                  │
              ┌────┬────┐ 第2分岐（セグメント）
              ↓    ↓    ↓
           ｶﾙﾋﾟｽ 三ツ矢 ﾜﾝﾀﾞ
          「家族」「若者」「ビジネス」
```

### 6.2 継承関係から見える戦略方向性

```
カルピスの継承関係分析
═══════════════════════

継承チェーン:
  HD「食を通じて健康で豊かな生活」
        ↓ specialization（食→飲料）
  OpCo「飲料を通じて健康と潤い」
        ↓ segmentation（生活者→家族）
        ↓ extension（+家族の絆）
  Brand「乳酸菌発酵で健やかさと家族の絆」

読み取れる戦略方向性:
  1. 「健康」は継承の軸（HD→OpCo→Brand全てで継承）
     → 健康価値はカルピスの本質、投資継続必須

  2. 「家族の絆」はBrandで追加された価値
     → カルピス固有の差別化要素、強化すべき

  3. セグメントは「家族層」に絞り込み
     → 若者向け、シニア向けは別ブランドの役割

  4. OpCoの他ブランドとの補完関係
     → カルピス×三ツ矢×ワンダでセグメント網羅
```

## 7. コンテキスト分析への組み込み

### 7.1 Phase 1 拡張（バリエーションポイント診断追加）

```yaml
# phase1-extended-context.yaml に追加

# ===== バリエーションポイント分析 =====
variation_point_analysis:

  # 類型判定
  organization_type:
    primary: conglomerate  # 子会社軸が主
    secondary: multi_brand  # ブランド軸も存在
    complexity: high

  # バリエーションポイント特定
  variation_points:
    - level: HD → OpCo
      type: divergence
```

## 8. まとめ

### 8.1 核心メッセージ

```
バリエーションポイント = 価値創造の焦点
════════════════════════════════════════

企業によって「どこでバリエーションが出るか」が違う

  コングロマリット → 子会社単位で分岐
  マルチブランド  → ブランド単位で分岐
  サービス多角化  → サービス単位で分岐

その分岐点を特定し、継承関係をリサーチすることで:
  - 何が「共通価値」か
  - 何が「固有価値」か
  - どこに投資すべきか

が明らかになる
```

### 8.2 コンテキスト分析の重要性

```
コンテキスト分析が重要な理由
══════════════════════════════

1. バリエーションポイントは企業固有
   → 画一的な分析フレームでは見逃す

2. 継承関係が戦略方向性を示す
   → 「健康」が継承されている→健康投資継続
   → 「家族の絆」が追加されている→差別化要素

3. VStream設計の基盤になる
   → 継承された価値をどの段階で訴求するか
   → 固有価値をどう差別化に使うか

結論:
  丁寧なコンテキスト分析 = 正しいVStream設計の前提条件
```
