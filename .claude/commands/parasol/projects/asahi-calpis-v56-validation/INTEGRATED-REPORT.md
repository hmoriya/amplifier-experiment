# Parasol V5.6 統合レポート

**アサヒ飲料カルピスブランドによる検証結果**

---

## エグゼクティブサマリー

Parasol V5.6は、**価値駆動型ビジネス設計**の精度を大幅に向上させる4つの新機能を導入した。アサヒグループホールディングス傘下のカルピスブランドを対象とした検証により、これらの新機能が実務に適用可能であることを確認した。

### 主要な発見

| 発見事項 | ビジネスインパクト |
|----------|-------------------|
| カルピスの価値は**情緒的価値(45%)**が機能的価値(35%)を上回る | 競争優位は「機能訴求」より「情緒訴求」にある |
| 健康エビデンス管理は顧客ステージで**Supporting→Core**に変化 | 投資タイミングの最適化が可能 |
| 10個の運用段階(OS)とCapabilityが**1:1対応** | 価値創出プロセスの責任所在が明確化 |
