# Parasol V5.6 エグゼクティブレポート

**アサヒ飲料カルピスブランドによる検証結果**

---

## エグゼクティブサマリー

Parasol V5.6は、**価値駆動型ビジネス設計**の精度を大幅に向上させる4つの新機能を導入した。アサヒグループホールディングス傘下のカルピスブランドを対象とした検証により、これらの新機能が実務に適用可能であることを確認した。

### 主要な発見

| 発見事項 | ビジネスインパクト |
|----------|-------------------|
| カルピスの価値は**情緒的価値(45%)**が機能的価値(35%)を上回る | 競争優位は「機能訴求」より「情緒訴求」にある |
| 健康エビデンス管理は顧客ステージで**Supporting→Core**に変化 | 投資タイミングの最適化が可能 |
| 10個の運用段階(OS)とCapabilityが**1:1対応** | 価値創出プロセスの責任所在が明確化 |

---

## V5.6 新機能の価値提案

### 1. Operational Stage（運用段階）定義

**Before（V5.5）**: バリューストリームとCapabilityの関係が曖昧

**After（V5.6）**: 運用段階を介した1:1対応で明確化

```
価値創出の流れが可視化
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
VStr-1: ブランド認知・選好
  └─ OS-1-1 → CL2-1-1 ブランドコミュニケーション
  └─ OS-1-2 → CL2-1-2 健康エビデンス管理
  └─ OS-1-3 → CL2-1-3 購買接点管理
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**経営インパクト**: どの部門が何の価値を生んでいるかが明確になり、組織設計・評価に活用可能

---

### 2. Stage依存TVDC

**Before（V5.5）**: 1 Capability = 1 TVDC（固定分類）

**After（V5.6）**: 顧客ステージに応じてTVDCが変化

```
健康エビデンス管理のTVDC変化
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
認知段階(VS0-1)    : Supporting（補助的）
選好形成(VS1-2)    : VCI（差別化要因）
継続利用(VS3-6)    : Core（中核）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**経営インパクト**: 投資判断が「一律」から「ステージ別最適化」へ進化

---

### 3. Value Component（VC）分析

5つの価値次元でVL3を多面的に分析：

| VC | 内容 | カルピス適用例 |
|----|------|---------------|
| VC1 | 製品価値 | 発酵技術、品質 |
| VC2 | サービス価値 | 入手性、価格 |
| VC3 | 体験価値 | 味わい、希釈体験 |
| VC4 | 関係価値 | 家族の絆、100年ブランド |
| VC5 | 情報価値 | 健康知識、腸活 |

**経営インパクト**: 価値の構成要素を定量把握し、強化ポイントを特定可能

## 戦略的示唆

### 短期施策（〜3ヶ月）

| 対象顧客ステージ | 重点Capability | TVDC | 推奨アクション |
|-----------------|---------------|------|---------------|
| VS1-2（選好形成） | 健康エビデンス管理 | VCI | 腸活訴求の強化 |
| VS4-6（定着） | 体験価値設計 | Core | 希釈体験・季節企画の充実 |

### 中期施策（3-12ヶ月）

1. **投資配分の見直し**: Stage依存TVDCに基づく予算再配分
2. **組織連携の強化**: OS-Capability対応に基づく責任明確化
3. **KPI再設計**: 各ステージの価値蓄積指標の導入

### 長期施策（1年以上）

1. **持続的価値の強化**: 発酵技術・100年ブランドの次世代継承
2. **社会的価値の可視化**: 健康寿命貢献のESG訴求

## ROI試算（概念）

| 改善領域 | V5.5 | V5.6 | 改善効果 |
|----------|------|------|---------|
| 投資判断の精度 | 一律配分 | ステージ別最適化 | 投資効率 +15-25% |
| 組織間連携 | 曖昧な責任 | OS-Capability明確化 | 意思決定速度 +20% |
| 顧客理解 | 単一視点 | 3軸多面分析 | 施策精度 +30% |

※数値は概念的な推定であり、実装・測定により精緻化が必要

## 結論と推奨事項

### 結論

Parasol V5.6は、価値駆動型ビジネス設計において**質的な飛躍**をもたらす。特に：

1. **OS定義**により価値創出の責任所在が明確化
2. **Stage依存TVDC**により投資判断が動的に最適化
3. **VC分析・3軸分解**により価値構造の多面的理解が可能

### 推奨事項

1. **全社展開の検討**: カルピス以外のブランドへのV5.6適用
2. **組織設計への反映**: OS-Capability対応に基づく責任体制の整備
3. **投資判断プロセスの更新**: Stage依存TVDCを投資委員会の判断基準に
