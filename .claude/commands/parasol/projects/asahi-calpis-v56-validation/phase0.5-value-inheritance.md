# Phase 0.5: 価値継承分析

**対象**: カルピス（Calpis）ブランド
**分析日**: 2025-01-14
**Parasolバージョン**: V5.7

---

## 1. 組織トポロジー

### 1.1 判定結果

| 項目 | 内容 |
|------|------|
| **タイプ** | マルチブランド型 |
| **構造** | HD → OpCo → Brand[] |
| **判定理由** | アサヒグループHD配下に複数事業会社があり、アサヒ飲料配下に複数ブランドが存在 |

### 1.2 組織構造図

```
アサヒグループホールディングス株式会社（HD）
├── アサヒビール株式会社
├── アサヒ飲料株式会社（OpCo）← 対象
│   ├── 三ツ矢サイダー
│   ├── ワンダ
│   ├── 十六茶
│   ├── カルピス ← 分析対象
│   └── その他ブランド
├── アサヒグループ食品株式会社
└── その他事業会社
```

### 1.3 なぜマルチブランド型か

アサヒグループは「コングロマリット型」の要素も持ちますが、今回の分析対象であるカルピスは「アサヒ飲料」という単一OpCo配下のブランドです。そのため、**HD → OpCo → Brand**のマルチブランド型として分析することが適切です。

## 2. 価値継承チェーン

### 2.1 HDレベル: アサヒグループホールディングス

#### ビジョン・ミッション

| 項目 | 内容 |
|------|------|
| **グループ理念** | 期待を超えるおいしさ、楽しい生活文化の創造 |
| **ビジョン** | 高付加価値ブランドを核として成長する "グローカルな価値創造企業" を目指す |
| **ミッション** | 世界の人々の健康で豊かな生活に貢献する |

#### コア価値（HDレベル）

| ID | 価値名 | 説明 |
|----|--------|------|
| **HV-01** | 期待を超えるおいしさ | 品質と味へのこだわり、常に期待を上回る価値提供 |
| **HV-02** | 楽しい生活文化の創造 | 飲食を通じた豊かなライフスタイルへの貢献 |
| **HV-03** | 健康への貢献 | 人々の健康的な生活をサポートする製品・サービス |

**情報源**: アサヒグループホールディングス 統合報告書2024、コーポレートサイト

---

### 2.2 OpCoレベル: アサヒ飲料

#### 事業ドメイン

| 項目 | 内容 |
|------|------|
| **事業領域** | 清涼飲料の製造・販売 |
| **カテゴリ** | 炭酸飲料、コーヒー、茶系飲料、乳性飲料、水 |
| **市場ポジション** | 国内清涼飲料市場シェア上位 |

#### コア価値（OpCoレベル）

| ID | 価値名 | 継承元 | 継承タイプ | 継承理由 |
|----|--------|--------|-----------|---------|
| **OV-01** | 飲料でのおいしさ追求 | HV-01 | **特殊化** | HD「期待を超えるおいしさ」を清涼飲料ドメインに特化。飲料特有の「のどごし」「爽快感」「香り」を追求 |
| **OV-02** | 飲料を通じた生活シーンの豊かさ | HV-02 | **特殊化** | HD「楽しい生活文化」を飲料シーン（リフレッシュ、リラックス、コミュニケーション）に特化 |
| **OV-03** | 飲料による健康サポート | HV-03 | **精緻化** | HD「健康への貢献」を飲料で実現可能な形（水分補給、機能性成分、カロリー管理）に具体化 |

**なぜ「特殊化」が主な継承タイプか**:
アサヒ飲料はアサヒグループの中で「清涼飲料」という明確な事業ドメインを担っています。HDの包括的な価値観を、飲料という特定の製品カテゴリに適用することで、事業としての方向性が定まります。

---

### 2.3 Brandレベル: カルピス

#### ブランド概要

| 項目 | 内容 |
|------|------|
| **ブランドタグライン** | カラダにピース。 |
| **創業** | 1919年（100年以上の歴史） |
| **製品特性** | 乳酸菌飲料、発酵技術 |
| **ブランドポジション** | 国民的ロングセラーブランド |

#### コア価値（Brandレベル）

| ID | 価値名 | 継承元 | 継承タイプ | 継承理由 |
|----|--------|--------|-----------|---------|
| **BV-01** | 乳酸菌のおいしさ | OV-01 | **精緻化** | アサヒ飲料「飲料でのおいしさ」を、乳酸菌発酵による独特の甘酸っぱさ・まろやかさに具体化。100年の発酵技術が生み出す唯一無二の味 |
| **BV-02** | 世代を超えた幸福な記憶 | OV-02 | **セグメント化** | アサヒ飲料「生活シーンの豊かさ」を、家族・世代間で共有される記憶体験に特化。「おばあちゃんの家で飲んだ」という共通体験の価値 |
| **BV-03** | 乳酸菌による心身の健やかさ | OV-03 | **セグメント化** | アサヒ飲料「健康サポート」を、乳酸菌がもたらす腸内環境・免疫サポートという特定の健康価値に特化 |

#### なぜこの継承タイプか

**BV-01（精緻化）の理由**:
カルピスの味わいは「飲料のおいしさ」の一般論ではなく、乳酸菌発酵という特定技術による固有のおいしさです。OpCoの価値をより具体的・限定的に定義しています。

**BV-02, BV-03（セグメント化）の理由**:
カルピスは「すべての飲料消費者」ではなく、「乳酸菌飲料に価値を見出す健康志向層」「世代間の思い出を大切にする家族層」という特定セグメントに向けた価値を提供しています。

## 3. 顧客セグメントツリー

### 3.1 セグメント階層

| 階層 | セグメント定義 | 絞り込み理由 |
|------|---------------|-------------|
| **HD** | すべての消費者（国内外） | グループ全体としての対象 |
| **OpCo** | 清涼飲料を購入・消費する人々 | 飲料事業に限定 |
| **Brand** | 乳酸菌飲料に価値を見出す、健康意識と世代間共有を重視する消費者 | カルピス固有の価値に共感する層 |

### 3.2 セグメント絞り込みの可視化

```
HD: すべての消費者
│
├─ ビール消費者 → アサヒビール
├─ 食品消費者 → アサヒグループ食品
│
└─ OpCo: 清涼飲料消費者
    │
    ├─ 炭酸飲料を求める層 → 三ツ矢サイダー
    ├─ コーヒーを求める層 → ワンダ
    ├─ 茶系飲料を求める層 → 十六茶
    │
    └─ Brand: 乳酸菌飲料を求める層 → カルピス
        │
        ├─ 健康意識層（腸活・免疫）
        ├─ ファミリー層（世代間共有）
        └─ ノスタルジー層（思い出・安心感）
```

### 3.3 包含関係の検証

| 検証項目 | 結果 | 説明 |
|----------|------|------|
| Brand ⊆ OpCo | ✅ OK | 乳酸菌飲料消費者は清涼飲料消費者の部分集合 |
| OpCo ⊆ HD | ✅ OK | 清涼飲料消費者はアサヒグループ顧客の部分集合 |
| 論理的整合性 | ✅ OK | セグメント階層に矛盾なし |

### 4.2 継承タイプの分布

| 継承 | 特殊化 | 精緻化 | セグメント化 | 合計 |
|------|--------|--------|-------------|------|
| HD → OpCo | 2 | 1 | 0 | 3 |
| OpCo → Brand | 0 | 1 | 2 | 3 |

**分析**: HD→OpCoでは「事業ドメインへの特殊化」が主、OpCo→Brandでは「顧客セグメントへの特化」が主となっています。これは組織階層の役割と整合しています。

## 5. Phase 1以降への接続

### 5.1 この分析から得られた制約と方向性

| 項目 | 内容 | Phase 1-3への影響 |
|------|------|------------------|
| **HDからの制約** | 「期待を超える」という高い品質基準 | VStreamで品質管理の重要度が高い |
| **OpCoからの方向性** | 飲料としての「おいしさ」「健康」軸 | 価値定義で味と健康の両立が必須 |
| **Brand固有要素** | 100年の歴史、世代間共有体験 | VStreamで「体験価値」の設計が重要 |

### 5.2 既存Phase 2価値定義との整合確認

既存の`phase2-value-definition.yaml`で定義されたVL1-VL3は、このPhase 0.5分析と以下のように整合します：

| Phase 0.5 Brand価値 | Phase 2 VL1 | 整合性 |
|---------------------|-------------|--------|
| BV-01: 乳酸菌のおいしさ | - | △ VL1に明示なし（VL2以下に含まれる可能性） |
| BV-02: 世代を超えた幸福な記憶 | VL1-2: 世代を超えた幸福な記憶を紡ぐ | ✅ 完全一致 |
| BV-03: 乳酸菌による心身の健やかさ | VL1-1: 心身の健やかさを支える | ✅ 完全一致 |

**発見**: 既存のVL1定義は、企業価値継承の観点からも妥当であることが確認できました。「おいしさ」についてはVL2以下で扱われていると推測されます。

## 6. まとめ

### 6.1 価値継承分析の結論

カルピスの価値は、アサヒグループの企業理念から一貫して継承されています：

1. **HD「期待を超えるおいしさ」** → OpCo「飲料でのおいしさ」→ Brand「乳酸菌のおいしさ」
2. **HD「楽しい生活文化」** → OpCo「生活シーンの豊かさ」→ Brand「世代を超えた記憶」
3. **HD「健康への貢献」** → OpCo「飲料による健康」→ Brand「乳酸菌の健やかさ」

### 6.2 この分析の価値

- **「なぜこの価値か」が説明可能**: カルピスの価値は恣意的ではなく、企業構造から導出される
- **セグメント妥当性の証明**: ターゲット顧客が論理的に包含関係を満たしている
- **戦略的位置づけの明確化**: グループ内でのカルピスの役割が可視化された
