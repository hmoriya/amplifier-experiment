## 1. 継承チェーン概要

カルピスの価値は、アサヒグループの企業理念から一貫して継承されています。

```
アサヒグループHD ──────────────────────────────────────────────────
│
│  「期待を超えるおいしさ、楽しい生活文化の創造」
│
│  HV-01: 期待を超えるおいしさ
│  HV-02: 楽しい生活文化の創造
│  HV-03: 健康への貢献
│
└─────────┬────────────────────────────────────────────────────────
          │
          │ 継承: 特殊化（飲料事業ドメインへ）
          ▼
アサヒ飲料 ────────────────────────────────────────────────────────
│
│  「飲料を通じて、心身のリフレッシュと健康をお届けする」
│
│  OV-01: 飲料でのおいしさ追求 ← HV-01
│  OV-02: 飲料を通じた生活シーンの豊かさ ← HV-02
│  OV-03: 飲料による健康サポート ← HV-03
│
└─────────┬────────────────────────────────────────────────────────
          │
          │ 継承: セグメント化（乳酸菌飲料・健康志向層へ）
          ▼
カルピス ──────────────────────────────────────────────────────────

  「カラダにピース。」

  BV-01: 乳酸菌のおいしさ ← OV-01
  BV-02: 世代を超えた幸福な記憶 ← OV-02
  BV-03: 乳酸菌による心身の健やかさ ← OV-03

───────────────────────────────────────────────────────────────────
```

## 2. HD → 事業会社 継承分析

### 2.1 アサヒグループHDの価値

| ID | 価値名 | 説明 |
|----|--------|------|
| HV-01 | 期待を超えるおいしさ | 品質と味への徹底したこだわり。常に顧客の期待を上回る |
| HV-02 | 楽しい生活文化の創造 | 飲食を通じた豊かなライフスタイルへの貢献 |
| HV-03 | 健康への貢献 | 世界の人々の健康で豊かな生活に貢献する |

**情報源**: アサヒグループHD統合報告書2024

### 2.2 アサヒ飲料への継承

| HD価値 | 事業会社価値 | 継承タイプ | 継承理由 |
|--------|---------|-----------|---------|
| HV-01 | OV-01: 飲料でのおいしさ追求 | **特殊化** | HDの「おいしさ」を清涼飲料という事業ドメインに特化。飲料特有の「のどごし」「爽快感」「香り」といった要素で具体化 |
| HV-02 | OV-02: 飲料を通じた生活シーンの豊かさ | **特殊化** | HDの「生活文化」を飲料シーン（リフレッシュ、リラックス、食事、コミュニケーション）に特化 |
| HV-03 | OV-03: 飲料による健康サポート | **精緻化** | HDの「健康貢献」を飲料で実現可能な形（水分補給、機能性成分、カロリー管理、乳酸菌）に具体化 |

### 2.3 なぜ「特殊化」が主な継承タイプか

アサヒ飲料は、アサヒグループの中で**清涼飲料**という明確な事業ドメインを担当しています。

- HDの価値は「すべての事業」に適用可能な抽象度
- 事業会社はそれを「自社の事業領域」に適用
- 結果として「特殊化」（specialization）が自然な継承パターン

## 3. 事業会社 → Brand 継承分析

### 3.1 アサヒ飲料の価値

| ID | 価値名 | 説明 |
|----|--------|------|
| OV-01 | 飲料でのおいしさ追求 | 各カテゴリで最高のおいしさを実現 |
| OV-02 | 飲料を通じた生活シーンの豊かさ | 様々な生活シーンに寄り添う飲料の提供 |
| OV-03 | 飲料による健康サポート | 機能性飲料、乳酸菌飲料等による健康貢献 |

### 3.2 カルピスへの継承

| 事業会社価値 | Brand価値 | 継承タイプ | 継承理由 |
|---------|----------|-----------|---------|
| OV-01 | BV-01: 乳酸菌のおいしさ | **精緻化** | 「飲料のおいしさ」を、乳酸菌発酵による独特の甘酸っぱさ・まろやかさに具体化。100年の発酵技術が生み出す唯一無二の味わい |
| OV-02 | BV-02: 世代を超えた幸福な記憶 | **セグメント化** | 「生活シーンの豊かさ」を、家族・世代間で共有される記憶体験に特化。「おばあちゃんの家で飲んだカルピス」という共通体験 |
| OV-03 | BV-03: 乳酸菌による心身の健やかさ | **セグメント化** | 「健康サポート」を、乳酸菌がもたらす腸内環境・免疫サポートという特定の健康価値に特化 |

### 3.3 なぜ「セグメント化」が主な継承タイプか

カルピスは、アサヒ飲料の中で**特定の顧客セグメント**に向けた価値を提供しています。

- アサヒ飲料は「すべての飲料消費者」が対象
- カルピスは「乳酸菌飲料に価値を見出す層」「世代間共有を重視する家族層」に特化
- 結果として「セグメント化」（segmentation）が自然な継承パターン

## 4. 顧客セグメント継承

### 4.1 セグメントツリー

```
HD: すべての消費者（国内外の飲食を楽しむ人々）
│
│   絞り込み: 飲料消費に限定
│
└── 事業会社: 清涼飲料を購入・消費する人々
    │
    │   絞り込み: 乳酸菌飲料に価値を見出す層
    │
    └── Brand: カルピスのターゲット顧客
        │
        ├── 健康意識層
        │   └── 腸活・免疫に関心、乳酸菌の効果を期待
        │
        ├── ファミリー層
        │   └── 子どもと一緒に楽しむ、世代間共有
        │
        └── ノスタルジー層
            └── 思い出・安心感、日本の夏の風物詩
```

### 4.2 包含関係の検証

| 検証 | 結果 | 説明 |
|------|------|------|
| Brand ⊆ 事業会社 | ✅ | カルピス顧客は清涼飲料顧客の部分集合 |
| 事業会社 ⊆ HD | ✅ | 清涼飲料顧客はアサヒグループ顧客の部分集合 |

**結論**: セグメント包含関係に矛盾なし

## 5. 継承パターンの分析

### 5.1 継承タイプ分布

| 継承 | 特殊化 | 精緻化 | セグメント化 |
|------|--------|--------|-------------|
| HD → 事業会社 | 2 | 1 | 0 |
| 事業会社 → Brand | 0 | 1 | 2 |

### 5.2 パターンの意味

```
HD → 事業会社: 「事業ドメインへの特殊化」が主
           グループ価値を特定事業領域に適用

事業会社 → Brand: 「顧客セグメントへの特化」が主
              事業価値を特定顧客層に適用
```

このパターンは組織階層の役割と整合しています：
- **HD**: 全事業に共通する価値観を定義
- **事業会社**: 事業領域に応じた価値を具体化
- **Brand**: 顧客セグメントに応じた価値を提供

## 6. カルピスへの制約と方向性

### 6.1 上位から継承される制約

| 継承元 | 制約 | カルピスへの影響 |
|--------|------|-----------------|
| HD | 「期待を超える」という高い品質基準 | 品質管理・研究開発への継続投資が必須 |
| HD | サステナビリティへのコミットメント | 環境配慮型パッケージ等の対応 |
| 事業会社 | 飲料としての安全性・おいしさ基準 | 乳酸菌飲料としての品質維持 |

### 6.2 上位から継承される方向性

| 継承元 | 方向性 | カルピスへの示唆 |
|--------|--------|-----------------|
| HD | 「健康」「生活文化」の重視 | 乳酸菌の健康価値を前面に |
| HD | 高付加価値ブランド戦略 | プレミアム製品ラインの展開 |
| 事業会社 | カテゴリ多様性によるシーン対応 | 多様な製品形態（原液、RTD、機能性） |

## 7. 継承分析のまとめ

### 7.1 カルピスの価値の正当性

カルピスの3つの価値（BV-01〜03）は、アサヒグループの企業理念から論理的に導出されています：

| カルピスの価値 | 継承経路 | 正当性 |
|---------------|---------|--------|
| BV-01: 乳酸菌のおいしさ | HV-01 → OV-01 → BV-01 | HD「おいしさ」の乳酸菌発酵による具体化 |
| BV-02: 世代を超えた記憶 | HV-02 → OV-02 → BV-02 | HD「生活文化」の家族体験への特化 |
| BV-03: 心身の健やかさ | HV-03 → OV-03 → BV-03 | HD「健康」の乳酸菌効果への特化 |

### 7.2 この分析の価値

- **「なぜこの価値か」が説明可能**: 恣意的ではなく、企業構造から導出
- **戦略的整合性の確認**: グループ戦略との一貫性を担保
- **差別化ポイントの明確化**: 継承と独自性のバランスが可視化
