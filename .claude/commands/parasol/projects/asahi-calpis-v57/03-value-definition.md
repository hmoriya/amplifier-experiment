## 前提: 継承された価値

`02-value-inheritance.md`で分析した継承価値を基に、カルピスのVL1-VL3を定義します。

| 継承価値（BV） | 説明 | VL1への展開 |
|---------------|------|------------|
| BV-01: 乳酸菌のおいしさ | 発酵技術による唯一無二の味 | VL2以下で具体化 |
| BV-02: 世代を超えた幸福な記憶 | 家族・世代間の共有体験 | → VL1-2 |
| BV-03: 乳酸菌による心身の健やかさ | 腸内環境・免疫サポート | → VL1-1 |

---

## 1. VL1: 本質的価値（Why - なぜカルピスは存在するか）

### VL1-1: 心身の健やかさを支える

| 項目 | 内容 |
|------|------|
| **価値名** | 心身の健やかさを支える |
| **継承元** | BV-03（乳酸菌による心身の健やかさ） |
| **説明** | 100年以上にわたり、乳酸菌の力で人々の心身の健康を支えてきた。「カラダにピース。」という約束を通じて、日常の健やかさに貢献する |

**なぜこの価値か（継承の観点）**:
- HD「健康への貢献」→ 事業会社「飲料による健康サポート」→ カルピス「乳酸菌による健やかさ」
- 乳酸菌発酵という独自技術が、この価値を他ブランドと差別化

**ステークホルダーへの影響**:

| ステークホルダー | 影響 |
|-----------------|------|
| 顧客 | 健康維持・増進への安心感、腸活への期待 |
| 社会 | 国民の健康寿命延伸への貢献 |

---

### VL1-2: 世代を超えた幸福な記憶を紡ぐ

| 項目 | 内容 |
|------|------|
| **価値名** | 世代を超えた幸福な記憶を紡ぐ |
| **継承元** | BV-02（世代を超えた幸福な記憶） |
| **説明** | 親から子へ、祖父母から孫へ。カルピスは家族の絆と幸福な記憶を100年以上紡いできた日本の文化的資産 |

**なぜこの価値か（継承の観点）**:
- HD「楽しい生活文化の創造」→ 事業会社「生活シーンの豊かさ」→ カルピス「世代間の記憶共有」
- 100年の歴史が、複数世代にわたる共通体験を可能にしている

**ステークホルダーへの影響**:

| ステークホルダー | 影響 |
|-----------------|------|
| 顧客 | 家族との幸福な時間、ノスタルジーによる安心感 |
| 社会 | 日本の食文化継承、世代間コミュニケーション促進 |

---

## 2. VL2: 中核価値（What - 何を提供するか）

### VL1-1「心身の健やかさ」から展開

#### VL2-1-1: 乳酸菌による健康価値

| 項目 | 内容 |
|------|------|
| **価値名** | 乳酸菌による健康価値 |
| **親VL1** | VL1-1 |
| **説明** | 独自の乳酸菌発酵技術による身体への好影響。腸内環境の改善、免疫力サポート |

**なぜこの価値か**:
継承価値BV-03「乳酸菌による健やかさ」を、具体的な健康効果として定義。科学的根拠に基づく機能性が差別化要素。

#### VL2-1-2: 安心・信頼の品質

| 項目 | 内容 |
|------|------|
| **価値名** | 安心・信頼の品質 |
| **親VL1** | VL1-1 |
| **説明** | 100年の実績に裏打ちされた安全性と品質への信頼。継続的な研究開発投資 |

**なぜこの価値か**:
HD「期待を超えるおいしさ」の品質基準が継承。100年ブランドとしての信頼が、健康価値の裏付けとなる。

---

### VL1-2「世代を超えた記憶」から展開

#### VL2-2-1: 懐かしさと新しさの融合

| 項目 | 内容 |
|------|------|
| **価値名** | 懐かしさと新しさの融合 |
| **親VL1** | VL1-2 |
| **説明** | 伝統の味わいを守りながら、時代に合わせた進化。変わらない安心と新しい発見の両立 |

**なぜこの価値か**:
100年続くブランドが持つ「変わらない価値」と、現代のニーズに応える「進化」のバランス。

#### VL2-2-2: 家族のコミュニケーション触媒

| 項目 | 内容 |
|------|------|
| **価値名** | 家族のコミュニケーション触媒 |
| **親VL1** | VL1-2 |
| **説明** | カルピスを囲んだ家族の会話と絆づくり。共同作業（希釈）による親子の時間 |

**なぜこの価値か**:
HD「生活文化の創造」が、家族というコミュニティにおける「絆づくり」として具体化。

---

## 3. VL3: 詳細価値（How - どのように実現するか）

### VL2-1-1「乳酸菌による健康価値」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-1-1-1 | カルピス菌による発酵 | 100年以上受け継がれた独自の乳酸菌による発酵技術 | 乳酸菌数、発酵時間、風味スコア |
| VL3-1-1-2 | 腸内環境への好影響 | 乳酸菌が腸内フローラに与えるポジティブな効果 | 腸内細菌バランス改善率 |
| VL3-1-1-3 | 機能性成分の効果 | 届く乳酸菌など機能性表示可能な健康効果 | 機能性エビデンス、臨床試験データ |

### VL2-1-2「安心・信頼の品質」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-1-2-1 | 厳格な品質管理 | 原材料から製品まで一貫した品質管理体制 | 品質検査合格率、クレーム発生率 |
| VL3-1-2-2 | 100年ブランドの信頼 | 1919年創業以来の実績と消費者からの信頼 | ブランド信頼度スコア、NPS |

### VL2-2-1「懐かしさと新しさの融合」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-2-1-1 | 伝統の甘酸っぱい味わい | 世代を超えて愛される独特の風味 | 官能評価スコア、リピート率 |
| VL3-2-1-2 | 時代に合わせた商品展開 | 濃いめ、ソーダ、機能性など多様なニーズへの対応 | SKU数、新商品売上比率 |

### VL2-2-2「家族のコミュニケーション触媒」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-2-2-1 | 一緒に作る・割る体験 | 希釈タイプによる親子での調理体験 | 希釈タイプ購入率、家庭内利用シーン |
| VL3-2-2-2 | 季節・イベントとの結びつき | 夏の風物詩、七夕、お盆など季節行事との連携 | 季節商品売上、イベント関連認知度 |

## 4. 価値階層の全体像

```
VL1-1: 心身の健やかさを支える（← BV-03継承）
│
├── VL2-1-1: 乳酸菌による健康価値
│   ├── VL3-1-1-1: カルピス菌による発酵
│   ├── VL3-1-1-2: 腸内環境への好影響
│   └── VL3-1-1-3: 機能性成分の効果
│
└── VL2-1-2: 安心・信頼の品質
    ├── VL3-1-2-1: 厳格な品質管理
    └── VL3-1-2-2: 100年ブランドの信頼


VL1-2: 世代を超えた幸福な記憶を紡ぐ（← BV-02継承）
│
├── VL2-2-1: 懐かしさと新しさの融合
│   ├── VL3-2-1-1: 伝統の甘酸っぱい味わい
│   └── VL3-2-1-2: 時代に合わせた商品展開
│
└── VL2-2-2: 家族のコミュニケーション触媒
    ├── VL3-2-2-1: 一緒に作る・割る体験
    └── VL3-2-2-2: 季節・イベントとの結びつき
```

## 5. 継承価値との整合性確認

### BV-01「乳酸菌のおいしさ」の扱い

継承価値BV-01（乳酸菌のおいしさ）は、VL1レベルでは明示していません。

**理由**:
- VL1は「なぜ存在するか（Why）」を定義
- 「おいしさ」は「何を提供するか（What）」に該当
- よってVL2-2-1「懐かしさと新しさ」およびVL3-2-1-1「伝統の甘酸っぱい味わい」で表現

これは設計上の判断であり、継承価値を無視しているわけではありません。
