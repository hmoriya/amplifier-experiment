## プロジェクト概要

V5.7の新しいアプローチ「全体構造 → 継承関係 → カレントコンテキスト」を検証するプロジェクトです。

### V5.7の核心

**従来（V5.6以前）**:
```
対象（カルピス）の分析開始 → VL定義 → VStream（顧客セグメント別複数）
```

**V5.7**:
```
全体組織構造の把握 → コンテキスト選択 → 継承関係分析 → VL定義 → VStream（継承構造と同じ構成）

核心原則: 価値継承構造 = VStream構成
  VStreamの本数は固定ではなく、価値継承構造から導出される

  例：カルピス（HD → 事業会社 → Brand の3階層構造）
      → 結果として3本のVStream（HD + 事業会社 + Brand）
```

## V5.7の価値

### 1. 「なぜこの価値か」が説明可能に

従来: 「カルピスの価値は健康と家族の絆」（根拠不明確）

V5.7: 「HDの『健康への貢献』が事業会社で『飲料による健康サポート』に特殊化され、カルピスで『乳酸菌による健やかさ』にセグメント化された」（継承チェーンで説明）

### 2. 戦略的整合性の担保

ブランド戦略がグループ戦略と矛盾しないことを構造的に確認できる。

### 3. 新規プロジェクトのオンボーディング

全体構造から始めることで、新しいメンバーも文脈を理解しやすい。
