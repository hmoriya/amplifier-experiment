## V5.7原則の確認

### 核心原則: 価値継承構造 = VStream構成

**ターゲットコンテキスト**: Brand（カルピス）

V5.7では、**価値定義の継承構造と同じ構造でVStreamも構成される**という原則に従います。

```
【V5.7 VStream構成原則】

ターゲット: Brand（カルピス）の場合

  価値継承構造              VStream構成
  ─────────────────────     ─────────────────────
  HD価値（HV-01〜03）   →   HD VStream（概要）
        ↓ 継承
  OpCo価値（OV-01〜03） →   OpCo VStream（中間）
        ↓ 継承
  Brand価値（BV-01〜03）→   Brand VStream（詳細）★

→ 結果として3本のVStream（継承構造から導出）
```

> ⚠️ VStreamの本数は固定ではなく、**価値継承構造から導出**されます。
> カルピスの場合、HD→OpCo→Brandの3階層なので、結果として3本のVStreamで構成されます。
> 各VStreamは、それぞれの階層の顧客に価値を届ける流れを定義します。

## 5. 各VS段階の詳細（Brand VStream）

### 5.1 VS0→VS1: 未認知→認知

**顧客状態変化**:
- Before: カルピスを知らない、または意識していない状態
- After: カルピスの存在と基本的な特徴を認識している状態

**主に実現する価値**: VL1-2（世代を超えた記憶）

| 項目 | 内容 |
|------|------|
| **価値実現の仕組み** | 家族・知人からの伝播、季節的な露出（夏のCM等） |
| **顧客の認識** | 「カルピスという飲み物がある」「昔から知っている」 |
| **主要タッチポイント** | 家族からの紹介、TV/Web広告、店頭での視認 |
| **成功指標** | 認知率、ブランド想起率 |

**Operational Stages (OS)**:

| OS ID | 名称 | 価値蓄積 | 対応Capability |
|-------|------|---------|---------------|
| OS-1-1 | 世代間伝播支援 | 家族からの紹介機会創出 | ブランドマーケティング |
| OS-1-2 | マス認知獲得 | 広告による認知形成 | マスマーケティング |
| OS-1-3 | 店頭認知 | 売場での視認性確保 | 店頭販促管理 |

## 8. まとめ

### 8.1 V5.7原則の適用確認

| 原則 | 適用結果 |
|------|---------|
| 価値継承構造 = VStream構成 | ✅ Brand target = 3本のVStream（HD + OpCo + Brand） |
| 各階層に対応するVStream | ✅ HD/OpCo/Brand各レベルのVStreamを定義 |
| ターゲット階層を詳細定義 | ✅ Brand VStreamを詳細に、上位は概要で定義 |
| 価値継承の明示 | ✅ 各VStreamの価値コンポーネントの継承チェーンを確認 |
| 顧客包含関係 | ✅ 全消費者 ⊇ 清涼飲料消費者 ⊇ 乳酸菌飲料価値層 |

### 8.2 VStream構成の特徴

カルピスターゲットのVStream構成は以下の特徴を持ちます：

1. **継承構造との一致**: 価値継承（HD→OpCo→Brand）と同じ構造でVStreamも3本構成
2. **上位価値コンポーネントの活用**: Brand VStreamの価値は上位から継承し、「なぜこの価値か」が説明可能
3. **世代間循環構造**: VS6→VS7（推奨）が新たなVS0→VS1（認知）を生む循環
4. **価値の相互強化**: VL1-1（健やかさ）とVL1-2（記憶）が相互に強化し合う
