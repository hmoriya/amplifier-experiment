| 項目 | 内容 |

### コンテキストパス


---

## 2. 選択理由

### 2.1 なぜBrandレベルか

| 観点 | 説明 |
|------|------|
| **具体性** | Brandレベルは顧客価値を直接設計できる粒度 |
| **独立性** | カルピスは独自のターゲット・価値提案を持つ |
| **検証適性** | V5.7の継承関係検証に最適な階層 |

### 2.2 なぜカルピスか

| 観点 | 説明 |
|------|------|
| **歴史** | 100年以上の歴史を持つロングセラーブランド |
| **独自性** | 乳酸菌発酵という固有の技術・価値 |
| **継承関係** | HD→OpCo→Brandの継承が明確に追跡可能 |
| **データ可用性** | 公開情報が豊富で分析しやすい |

## 3. スコープ定義

### 3.1 含むもの（In Scope）

| カテゴリ | 内容 |
|---------|------|
| **製品群** | カルピスブランド全製品（原液、希釈、RTD、機能性） |
| **価値分析** | ブランドレベルのVL1-VL3、VStream |
| **顧客セグメント** | カルピスのターゲット顧客層 |
| **継承分析** | HD・OpCoからの価値継承関係 |

### 3.2 含まないもの（Out of Scope）

| カテゴリ | 内容 | 理由 |
|---------|------|------|
| 他ブランド詳細 | 三ツ矢、ワンダ等の詳細分析 | 別コンテキストとして分析すべき |
| OpCo全体戦略 | アサヒ飲料のポートフォリオ戦略 | 上位コンテキストの分析対象 |
| 製品別施策 | カルピスウォーターの個別施策 | 下位コンテキストの分析対象 |

## 4. 継承関係の確認ポイント

### 4.1 上位からの継承（確認必須）

カルピスの価値を設計する前に、以下の継承関係を明確にします：

| 継承元 | 確認事項 |
|--------|---------|
| **HD → カルピス** | グループ理念がどのようにカルピスに反映されているか |
| **OpCo → カルピス** | アサヒ飲料の事業価値がどのようにカルピスに特化されているか |

### 4.2 下位への影響（参考）

| 影響先 | 考慮事項 |
|--------|---------|
| **カルピス → 製品群** | ブランド価値が各製品にどう展開されるか |
| **カルピス → 施策** | VStreamが具体的な施策にどう落ちるか |
