## 本プロジェクトの目的

カルピスブランドの価値設計を、**アサヒグループ全体の企業価値から導出**し、「なぜこの価値なのか」を構造的に説明可能にする。

## VStream（価値実現の顧客体験）

### V5.7 核心原則

**価値継承構造 = VStream構成**

V5.7では、**価値定義の継承構造を基盤として、VStreamも構造的整合性を持って構成される**という原則を確立します。

**重要**: VStreamの本数は固定ではなく、**組織の価値継承構造から導出**されます。

| 継承構造の例 | VStream構成 | 詳細度 |
|-------------|------------|--------|
| HD のみ | HD VStream（1本） | 詳細 |
| HD → 事業会社 | HD + 事業会社 VStream（2本） | 上位概要、ターゲット詳細 |
| **HD → 事業会社 → Brand** | **HD + 事業会社 + Brand VStream（3本）** | **上位概要、ターゲット詳細** |
| HD → 事業会社 → Brand → Product | 4本 | 上位概要、ターゲット詳細 |

## V5.7の価値

### 従来との違い

| 観点 | 従来（V5.6以前） | V5.7 |
|------|-----------------|------|
| 起点 | 対象ブランドから開始 | 全体組織構造から開始 |
| 価値の根拠 | 暗黙的 | 継承チェーンで明示 |
| VStream構成 | 顧客セグメント別複数 | **継承構造と一致（階層数分）** |
| 価値コンポーネント | 階層別に独立 | **上位から継承して利用** |
| 「なぜ」の説明 | 困難 | 構造的に可能 |
| 戦略整合性 | 個別確認 | 自動的に担保 |

### 本プロジェクトで実証されたこと

1. **全体→部分の導出**: カルピスの価値がHDの企業理念から論理的に導出できる
2. **継承の可視化**: 各階層での価値変換（特殊化・セグメント化）が明確
3. **整合性の担保**: グループ戦略との一貫性が構造的に確認できる
4. **価値継承構造=VStream構成**: Brand target = 3本のVStream（HD + 事業会社 + Brand）で構成
5. **上位価値コンポーネントの活用**: 継承元の価値がなぜBrand価値になったかを構造的に説明可能

## 結論

カルピスの価値「心身の健やかさ」「世代を超えた記憶」は、アサヒグループの企業理念「期待を超えるおいしさ、楽しい生活文化の創造、健康への貢献」から**構造的に導出**されています。

V5.7の核心原則「**価値継承構造 = VStream構成**」により：
- **VStreamは継承構造に従い構成**される（Brand target = 3本）
- **上位価値コンポーネントを継承**して活用
- 「なぜこの価値か」「なぜこのVStream構成か」が**構造的に説明可能**

この原則により：
- ブランド戦略の**正当性**が説明可能
- グループ戦略との**整合性**が担保
- 新規メンバーへの**オンボーディング**が容易
