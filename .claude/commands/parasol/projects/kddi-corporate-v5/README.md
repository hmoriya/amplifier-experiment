## KDDIの法人向け事業の特徴

### 事業ブランド

2024年2月より「**KDDI BUSINESS**」ブランドを展開。国内外の法人顧客に対して、通信インフラを基盤としたDXソリューションを提供。

### 主要サービス領域

1. **モバイル** - スマートフォン、タブレット、データ通信
2. **IoT** - 通信・ネットワーク、データ分析、モニタリング
3. **クラウド（SaaS）** - Web会議、業務効率化、バックオフィス
4. **IaaS・データセンター** - クラウドインフラ
5. **セキュリティ** - クラウドセキュリティ、デバイス管理
6. **ネットワーク** - VPN、インターネット接続
7. **音声・電話** - IP電話、コンタクトセンター
8. **生成AI** - 環境構築、データ利活用

### 対応業種

製造、運輸・エネルギー、金融・保険、建設・不動産、流通・小売、サービス、教育、官公庁、農業、医療・福祉、情報通信
