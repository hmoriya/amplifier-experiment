## 2. アクター詳細プロファイル

### 2.1 ACT-C01: 大企業IT部門

| 項目 | 内容 |
|------|------|
| **ペルソナ** | 情報システム部門マネージャー、40代男性 |
| **責任範囲** | 社内IT基盤の企画・運用・セキュリティ |
| **課題** | レガシーシステム刷新、セキュリティ強化、IT人材不足 |
| **意思決定基準** | 信頼性、実績、サポート体制 |
| **情報源** | 展示会、業界誌、ベンダー提案 |

### 2.2 ACT-C02: 大企業経営層

| 項目 | 内容 |
|------|------|
| **ペルソナ** | CIO/デジタル推進役員、50代男性 |
| **責任範囲** | 経営戦略としてのDX推進、IT投資判断 |
| **課題** | DX成果の可視化、人材確保、競争力強化 |
| **意思決定基準** | ROI、戦略適合性、リスク |
| **情報源** | 経営者ネットワーク、コンサル、IR資料 |

### 2.3 ACT-C03: 中堅企業IT担当

| 項目 | 内容 |
|------|------|
| **ペルソナ** | 情報システム担当（兼任多い）、30-40代 |
| **責任範囲** | IT全般の企画・運用（少人数） |
| **課題** | リソース不足、クラウド移行判断、コスト最適化 |
| **意思決定基準** | コストパフォーマンス、導入容易性 |
| **情報源** | Web検索、セミナー、同業他社 |

### 2.4 ACT-C04: 中小企業経営者

| 項目 | 内容 |
|------|------|
| **ペルソナ** | 中小企業社長/取締役、40-60代 |
| **責任範囲** | 経営全般（ITは専門外） |
| **課題** | IT投資判断の難しさ、人手不足、コスト |
| **意思決定基準** | 簡単さ、価格、信頼できる担当者 |
| **情報源** | 知人紹介、地域の営業担当 |

### 2.5 ACT-C05: 官公庁IT担当

| 項目 | 内容 |
|------|------|
| **ペルソナ** | 自治体情報政策課担当、30-50代 |
| **責任範囲** | 行政システムの企画・調達・運用 |
| **課題** | デジタル庁方針対応、セキュリティ、予算制約 |
| **意思決定基準** | 法令遵守、実績、価格（入札） |
| **情報源** | デジタル庁通達、自治体間ネットワーク |
