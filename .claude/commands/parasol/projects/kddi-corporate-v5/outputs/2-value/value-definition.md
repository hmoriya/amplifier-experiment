# 価値定義（VL1-VL3）

**プロジェクト**: KDDI / 法人向け事業
**Parasol Version**: V5
**作成日**: 2025-01-15

---

## 前提: 継承された価値

`value-inheritance.md`で分析した継承価値を基に、ビジネス事業のVL1-VL3を定義します。

| 継承価値（OV/BV） | 説明 | VL1への展開 |
|------------------|------|------------|
| OV-01: つなぐチカラで事業成長支援 | 通信基盤で企業をつなぐ | → VL1-1 |
| OV-02: DX一気通貫伴走 | 通信〜DXまでワンストップ支援 | → VL1-2 |
| OV-03: 社会課題解決貢献 | 企業DXを通じた社会貢献 | → VL1-3 |

---

## 1. VL1: 本質的価値（Why - なぜKDDI法人事業は存在するか）

### VL1-1: 企業の「つながる」を支え、事業継続と成長を実現する

| 項目 | 内容 |
|------|------|
| **価値名** | 企業の「つながる」を支え、事業継続と成長を実現する |
| **継承元** | OV-01（つなぐチカラで事業成長支援） |
| **説明** | 通信インフラを基盤に、企業の人・モノ・情報・システムをつなぎ、24時間365日の事業継続と成長を支える |

**なぜこの価値か（継承の観点）**:
- 企業理念「豊かなコミュニケーション社会」→ VISION「つなぐチカラ」→ 事業価値「企業をつなぐ」
- 通信キャリアとしての本質的使命をB2Bで体現

**ステークホルダーへの影響**:

| ステークホルダー | 影響 |
|-----------------|------|
| 顧客企業 | 事業の安定稼働、機会損失の防止 |
| 顧客企業の従業員 | どこでも働ける環境、生産性向上 |
| 社会 | 産業インフラとしての通信の安定 |

---

### VL1-2: 企業のDXを一気通貫で実現し、競争力を高める

| 項目 | 内容 |
|------|------|
| **価値名** | 企業のDXを一気通貫で実現し、競争力を高める |
| **継承元** | OV-02（DX一気通貫伴走） |
| **説明** | 通信からクラウド、セキュリティ、AIまで、企業のデジタル変革を構想から運用まで伴走し、競争力強化を支援する |

**なぜこの価値か（継承の観点）**:
- 企業理念「期待を超える感動」→ VISION「思いを実現」→ 事業価値「DXの実現」
- 単なる通信提供ではなく、ビジネス成果まで責任を持つ姿勢

**ステークホルダーへの影響**:

| ステークホルダー | 影響 |
|-----------------|------|
| 顧客企業 | デジタル競争力の獲得、業務効率化 |
| 顧客企業のIT部門 | 戦略的パートナー、運用負荷軽減 |
| 経営層 | DX投資のROI最大化 |

---

### VL1-3: 企業DXを通じて社会課題の解決に貢献する

| 項目 | 内容 |
|------|------|
| **価値名** | 企業DXを通じて社会課題の解決に貢献する |
| **継承元** | OV-03（社会課題解決貢献） |
| **説明** | 企業のDX支援を通じて、人手不足、地方創生、環境問題、高齢化など社会課題の解決に貢献する |

**なぜこの価値か（継承の観点）**:
- 企業理念「社会の発展に貢献」→ VISION「誰もが思いを実現」→ 事業価値「社会課題解決」
- 企業価値と社会価値の両立（CSV: Creating Shared Value）

**ステークホルダーへの影響**:

| ステークホルダー | 影響 |
|-----------------|------|
| 顧客企業 | 社会的意義のある事業への貢献、ESG対応 |
| 地域社会 | 地方創生、過疎化対策、雇用維持 |
| 社会全体 | サステナブルな社会の実現 |

---

## 2. VL2: 中核価値（What - 何を提供するか）

### VL1-1「つながる支援」から展開

#### VL2-1-1: 高品質・高信頼の通信基盤

| 項目 | 内容 |
|------|------|
| **価値名** | 高品質・高信頼の通信基盤 |
| **親VL1** | VL1-1 |
| **説明** | 24時間365日止まらない、高品質なモバイル・固定通信サービス |

**なぜこの価値か**:
通信キャリアとしての本業。「つなぐチカラ」の基盤となる核心価値。

#### VL2-1-2: IoTによるモノのつながり

| 項目 | 内容 |
|------|------|
| **価値名** | IoTによるモノのつながり |
| **親VL1** | VL1-1 |
| **説明** | 4,550万回線のIoT基盤により、企業のモノ・設備・車両をつなぐ |

**なぜこの価値か**:
「つなぐ」対象を人だけでなくモノに拡張。コネクテッドカー2,400万回線が象徴。

---

### VL1-2「DX一気通貫」から展開

#### VL2-2-1: クラウド・インフラ基盤

| 項目 | 内容 |
|------|------|
| **価値名** | クラウド・インフラ基盤 |
| **親VL1** | VL1-2 |
| **説明** | IaaS、データセンター、ハイブリッドクラウドによる柔軟なIT基盤 |

**なぜこの価値か**:
DXの基盤となるインフラ。通信と組み合わせた一気通貫提供の核。

#### VL2-2-2: 業務ソリューション（SaaS）

| 項目 | 内容 |
|------|------|
| **価値名** | 業務ソリューション（SaaS） |
| **親VL1** | VL1-2 |
| **説明** | Web会議、業務効率化、バックオフィス等のクラウドアプリケーション |

**なぜこの価値か**:
DXの実現手段としてのアプリケーション層。顧客の業務改善に直結。

#### VL2-2-3: セキュリティ保護

| 項目 | 内容 |
|------|------|
| **価値名** | セキュリティ保護 |
| **親VL1** | VL1-2 |
| **説明** | ゼロトラスト時代のクラウドセキュリティ、デバイス管理、監視サービス |

**なぜこの価値か**:
DX推進の前提条件としてのセキュリティ。信頼の基盤。

---

### VL1-3「社会課題解決」から展開

#### VL2-3-1: 産業DXソリューション

| 項目 | 内容 |
|------|------|
| **価値名** | 産業DXソリューション |
| **親VL1** | VL1-3 |
| **説明** | 製造、農業、医療、物流等の産業特化型DXソリューション |

**なぜこの価値か**:
社会課題解決の具体的手段。業種ごとの課題に対応したソリューション。

#### VL2-3-2: 地域・行政DX支援

| 項目 | 内容 |
|------|------|
| **価値名** | 地域・行政DX支援 |
| **親VL1** | VL1-3 |
| **説明** | 自治体DX、地方創生、スマートシティ等の地域課題解決支援 |

**なぜこの価値か**:
地方創生、行政効率化という社会課題への直接的貢献。

## 3. VL3: 詳細価値（How - どのように実現するか）

### VL2-1-1「通信基盤」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-1-1-1 | 法人モバイル | スマートフォン、タブレット、データ通信端末 | 法人回線数、ARPU |
| VL3-1-1-2 | 固定ネットワーク | VPN、イントラネット、インターネット接続 | 回線数、稼働率 |
| VL3-1-1-3 | 音声・電話 | IP電話、クラウドPBX、コンタクトセンター | 通話品質、導入企業数 |

### VL2-1-2「IoT」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-1-2-1 | IoT通信 | IoT専用回線、SORACOM連携 | IoT回線数（4,550万） |
| VL3-1-2-2 | コネクテッドカー | 車両向け通信、テレマティクス | 車載回線数（2,400万） |
| VL3-1-2-3 | IoTモニタリング | 設備監視、環境センシング | 監視デバイス数 |

### VL2-2-1「クラウド基盤」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-2-1-1 | IaaS | クラウドサーバー、ストレージ | リソース利用量 |
| VL3-2-1-2 | データセンター | TELEHOUSE、コロケーション | DC利用面積、PUE |
| VL3-2-1-3 | ハイブリッドクラウド | オンプレ・クラウド連携 | 導入企業数 |

### VL2-2-2「業務ソリューション」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-2-2-1 | コミュニケーション | Web会議、ビジネスチャット | アクティブユーザー数 |
| VL3-2-2-2 | 業務効率化 | ワークフロー、RPA | 導入企業数、自動化率 |
| VL3-2-2-3 | 生成AI活用 | AI環境構築、データ利活用 | AI導入企業数 |

### VL2-2-3「セキュリティ」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-2-3-1 | クラウドセキュリティ | SASE、ゼロトラスト | 導入企業数 |
| VL3-2-3-2 | デバイス管理 | MDM、EDR | 管理デバイス数 |
| VL3-2-3-3 | セキュリティ運用 | SOC、インシデント対応 | 監視対象数 |

### VL2-3-1「産業DX」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-3-1-1 | スマートファクトリー | 製造業向けIoT・AI | 導入工場数 |
| VL3-3-1-2 | スマート農業 | 農業IoT、データ分析 | 導入農家数 |
| VL3-3-1-3 | 遠隔医療・介護 | 医療ICT、見守り | 導入施設数 |

### VL2-3-2「地域・行政DX」から展開

| ID | 価値名 | 説明 | 測定指標 |
|----|--------|------|---------|
| VL3-3-2-1 | 自治体DX | 行政システム、住民サービス | 導入自治体数 |
| VL3-3-2-2 | スマートシティ | 都市OS、MaaS | 実証・導入都市数 |
| VL3-3-2-3 | 地方創生 | 地域課題解決、関係人口 | 支援地域数 |

## 4. 価値階層の全体像

```
VL1-1: 企業の「つながる」を支え、事業継続と成長を実現する
│
├── VL2-1-1: 高品質・高信頼の通信基盤
│   ├── VL3-1-1-1: 法人モバイル
│   ├── VL3-1-1-2: 固定ネットワーク
│   └── VL3-1-1-3: 音声・電話
│
└── VL2-1-2: IoTによるモノのつながり
    ├── VL3-1-2-1: IoT通信
    ├── VL3-1-2-2: コネクテッドカー
    └── VL3-1-2-3: IoTモニタリング


VL1-2: 企業のDXを一気通貫で実現し、競争力を高める
│
├── VL2-2-1: クラウド・インフラ基盤
│   ├── VL3-2-1-1: IaaS
│   ├── VL3-2-1-2: データセンター
│   └── VL3-2-1-3: ハイブリッドクラウド
│
├── VL2-2-2: 業務ソリューション（SaaS）
│   ├── VL3-2-2-1: コミュニケーション
│   ├── VL3-2-2-2: 業務効率化
│   └── VL3-2-2-3: 生成AI活用
│
└── VL2-2-3: セキュリティ保護
    ├── VL3-2-3-1: クラウドセキュリティ
    ├── VL3-2-3-2: デバイス管理
    └── VL3-2-3-3: セキュリティ運用


VL1-3: 企業DXを通じて社会課題の解決に貢献する
│
├── VL2-3-1: 産業DXソリューション
│   ├── VL3-3-1-1: スマートファクトリー
│   ├── VL3-3-1-2: スマート農業
│   └── VL3-3-1-3: 遠隔医療・介護
│
└── VL2-3-2: 地域・行政DX支援
    ├── VL3-3-2-1: 自治体DX
    ├── VL3-3-2-2: スマートシティ
    └── VL3-3-2-3: 地方創生
```
