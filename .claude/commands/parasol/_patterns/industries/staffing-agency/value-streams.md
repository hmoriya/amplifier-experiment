# 人材派遣業 - 価値軸バリューストリーム（両面市場統合型）

## 概要

人材派遣業における価値軸アプローチは「両面市場統合型」モデルです。スタッフ（求職者）とクライアント（企業）の両者に対して「グッドマッチによる持続的成長」という共通価値を提供する構造を定義します。

## 基本構造

```
                    【グッドマッチによる持続的成長】
    ┌──────────────────────────────────────────────────┐
    │              共 通 価 値                          │
    │  ○ 成長機会の最大化  ○ 継続的関係構築  ○ 市場拡大 │
    └────────────────────┬─────────────────────────────┘
                         │
    ┌────────────────────┴─────────────────────────────┐
    │              人材派遣プラットフォーム              │
    │  ┌─────────┐   ┌─────────┐   ┌─────────┐      │
    │  │スタッフ  │←→│派遣会社  │←→│クライアント│    │
    │  │（求職者）│   └────┬────┘   │ （企業）  │    │
    │  └─────────┘        │         └─────────┘      │
    │                      │                          │
    │           コーディネーター                       │
    │        （グッドマッチ実現者）                    │
    └──────────────────────────────────────────────────┘
```

## コアコンセプト：グッドマッチ戦略

```
【従来：ベストマッチ幻想】
完璧な条件一致を追求 → マッチング率10% → 機会損失90%
    ↓
【新戦略：グッドマッチ最大化】
70点マッチ＋育成でフィット → マッチング率40% → 市場拡大4倍
```

## バリューストリーム定義

### VS0: ビジョン・戦略立案（グッドマッチ思想）

**目的**: 「すべての働く人に機会を、すべての企業に人材を」

| ステークホルダー | 価値提供 |
|-----------------|---------|
| スタッフ | より多くの就業機会、成長を前提とした採用 |
| クライアント | 柔軟な人材確保、育成型派遣の活用 |
| コーディネーター | 成長支援のプロフェッショナルとしての役割 |
| 社会 | 労働市場の流動性向上、雇用機会拡大 |

**主要活動**:
- グッドマッチ vs ベストマッチ戦略の選択
- 市場拡大型ビジョンの策定
- 両面市場における価値バランス設計
- 成熟市場での差別化戦略

---

### VS1: 市場分析・ポテンシャル発見

**目的**: 両面市場の潜在可能性を発見

| 分析領域 | スタッフ側 | クライアント側 |
|----------|-----------|---------------|
| 顕在層 | 即戦力人材 | 明確な要件 |
| 潜在層 | 転用可能スキル保持者 | 調整可能要件 |
| 育成層 | 未経験・ブランク層 | 育成意欲のある企業 |

**市場拡大の可能性**:
- 異業種転職市場（スキル転用）
- ブランク人材市場（復職支援）
- 未経験者市場（育成前提）
- シニア人材市場（経験活用）

---

### VS2: グッドマッチング実現・価値創造

**目的**: 70%一致から120%成功への変換

#### Phase 1: 拡張マッチング

| 要素 | 従来 | グッドマッチ |
|------|------|-------------|
| 一致基準 | 100%一致 | コア要件70%一致 |
| マッチング率 | 10% | 40% |
| 評価軸 | スキル完全一致 | 成長可能性含む |

**マッチングアルゴリズム2.0**:
- 必須スキル（30%）
- 類似経験（20%）
- 成長可能性（20%）
- カルチャーフィット（15%）
- 学習能力（15%）

#### Phase 2: フィット調整

| スタッフ側調整 | クライアント側調整 |
|---------------|-------------------|
| 事前研修（1-2週） | 要件の優先順位整理 |
| メンター制度 | OJT計画策定 |
| 期待値すり合わせ | 受入準備 |
| 不安解消 | 評価基準調整 |

---

### VS2.5: エンゲージメント＆グロース（最重要）

**目的**: グッドマッチを通じた継続的価値向上

```
グッドマッチ（70%）→ 30%ギャップを埋める → 濃密なフォロー必須
                          ↓
              フォローROIが10倍以上
```

#### 高頻度タッチポイント戦略

| 頻度 | スタッフ向け | クライアント向け |
|------|-------------|-----------------|
| 週次 | パルス（モチベーション、困りごと） | パルス（パフォーマンス、課題） |
| 月次 | スキル開発、キャリア相談 | 生産性向上支援、プロセス改善 |
| 四半期 | キャリアプラン、処遇改善 | 戦略的人材計画、正社員化検討 |

**継続率向上メカニズム**:
- Small Wins の可視化と称賛
- 成長実感の演出（スキルマップ更新）
- 帰属意識の醸成（派遣先/派遣元両方）
- キャリアパスの明確化

---

### VS3: 育成型ブランド・マーケティング

**目的**: 「育てる派遣」ブランドの確立

| 対象 | 訴求メッセージ |
|------|---------------|
| スタッフ向け | 「未経験でも大丈夫」「成長できる環境」「市場価値が上がる」 |
| クライアント向け | 「一緒に育てるパートナー」「柔軟な人材ソリューション」「将来の正社員候補」 |

**具体施策**:
- 成功事例の発信（Before/After）
- スキルアップセミナーの開催
- キャリアカウンセリングの無料提供
- クライアント向け育成ノウハウ共有

---

### VS4: プロアクティブ営業・育成

**目的**: 潜在層の開拓と事前育成

| スタッフ開拓 | クライアント啓蒙 |
|-------------|-----------------|
| キャリアチェンジ層 | 要件の本質化コンサル |
| ブランク層 | グッドマッチ成功事例 |
| 他業界経験者 | トータルコスト提案 |
| 未経験者 | 育成型派遣のメリット |

**育成プログラム**:
- 業界別基礎研修
- ビジネスマナー
- ITリテラシー
- 受入体制構築支援

---

### VS5: 成長支援・品質保証

**目的**: 継続的な成長と品質向上

| スキルアップ支援 | パフォーマンス向上支援 |
|-----------------|---------------------|
| オンライン学習プラットフォーム | 定期的な振り返り会 |
| 資格取得支援 | ベストプラクティス共有 |
| 社内勉強会 | 改善提案制度 |
| メンタリング | フィードバックループ |

---

### VS6: データ駆動型最適化

**目的**: グッドマッチを実現するAI/データ活用

| 領域 | 活動 |
|------|------|
| 拡張マッチング | 類似性＋補完性評価、成長曲線予測 |
| 継続予測 | 離脱リスク早期検知、介入効果測定 |
| 成長追跡 | スキル習得速度、パフォーマンス改善率 |
| AI活用 | 自動アラート、パーソナライズ学習推奨 |

---

### VS7: 業務オペレーション最適化

|------|------|

| 人材育成 | コーディネーターのスキル強化 |

## 成功指標（KPI）の進化

### 従来KPI → 成熟期KPI

| カテゴリ | 従来KPI | 成熟期KPI |
|----------|---------|-----------|
| マッチング | マッチング率 10% | グッドマッチ率 40% |
| 継続性 | 3ヶ月継続率 60% | 12ヶ月継続率 80% |
| 成長 | - | スキルアップ実現率 70% |
| 満足度 | 初回満足度 | 継続満足度スコア 4.5/5 |
| 収益性 | 新規売上 | LTV（3年間総収益） |

### 新規KPI

1. **フィット向上率**: 就業3ヶ月後の評価改善度
2. **相互NPS**: スタッフ→企業、企業→スタッフの推奨度
3. **成長実現率**: 計画したキャリアパスの達成度
4. **調整成功率**: グッドマッチから優良継続への転換率

## 両面市場統合の設計ポイント

### 統合サービス例

1. **グッドマッチプログラム**
   - スタッフ: 成長機会の提供
   - クライアント: 育成型人材の活用
   - コーディネーター: 両者の成長支援

2. **キャリアアップ型派遣**
   - スタッフ: スキルアップによる市場価値向上
   - クライアント: 正社員登用の選択肢
   - コーディネーター: キャリアパス設計支援

3. **紹介予定派遣との連携**
   - スタッフ: 正社員への道
   - クライアント: 見極め期間の活用
   - コーディネーター: 双方のフィット調整
