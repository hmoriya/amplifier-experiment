# 人材派遣業 - 軸選択ガイド

## 業界特性

| 特性 | 説明 |
|------|------|
| **両面市場構造** | スタッフ（求職者）とクライアント（企業）の2者を同時にサービス |
| **市場成熟化** | 「ベストマッチ追求」から「グッドマッチ最大化」への戦略シフト |
| **継続性重視** | 獲得重視から継続フォロー重視への競争軸変化 |
| **規制産業** | 労働者派遣法、雇用安定措置義務 |
| **デジタル化圧力** | マッチングAI、オンライン面談、リモートワーク対応 |

## 軸選択マトリクス

```
                    両面市場統合の度合い
                    低い            高い
                 ┌──────────────┬──────────────┐
   事業規模      │ 事業部軸     │ フュージョン型 │
      大規模    │（専門分化）   │              │
               ├──────────────┼──────────────┤
      中小規模  │ケーパビリティ │   価値軸     │
               │   統合型     │      ◎      │
               └──────────────┴──────────────┘
```

## 推奨軸と理由

### 第1推奨: 価値軸（両面市場統合型）

**理由**:
- スタッフとクライアントへの価値を統合的に設計できる
- 「グッドマッチ」という共通価値で両者を結びつける
- 成熟市場における差別化は継続的価値創造による

**適用パターン**:
```
VS0: ビジョン・戦略立案（グッドマッチ思想）
VS1: 市場分析・ポテンシャル発見（両面市場インサイト）
VS2: グッドマッチング実現・価値創造
VS2.5: エンゲージメント＆グロース（継続支援）
VS3: 育成型ブランド・マーケティング
VS4: プロアクティブ営業・育成
VS5: 成長支援・品質保証
VS6: データ駆動型最適化
VS7: 業務オペレーション最適化
```

### 第2推奨: フュージョン型

**理由**:
- 大規模派遣会社で事業領域が多岐にわたる場合に有効
- 共通層（IT基盤、コンプライアンス）と事業固有層を分離

**適用パターン**:
```
共通層: VS0（統合戦略）、VS6（データ基盤）、VS7（バックオフィス）
事業層: 一般派遣、紹介予定派遣、技術者派遣を独立運用
```

### 検討が必要: 事業部軸

**適用条件**:
- 一般派遣、紹介予定派遣、アウトソーシングで明確に分離
- 各事業部がP/L責任を持つ大規模組織

**リスク**:
- スタッフの事業部間異動が困難になる
- クライアントへのワンストップ提案ができない
- 両面市場の統合価値が活かせない

### 非推奨: ケーパビリティ統合型

**理由**:
- 人材派遣業の強みは「マッチング力」「関係構築力」
- 機能別統合は両者の接点を希薄化
- スタッフ・クライアント双方の信頼関係が最大資産

## 判断フローチャート

```
Q1: スタッフとクライアントを統合的にサービスしているか？
    │
    ├─ Yes → 価値軸（両面市場統合型）を推奨
    │
    └─ No → Q2へ

Q2: 複数の派遣事業（一般、紹介予定、技術者等）があるか？
    │
    ├─ Yes → Q3へ
    │
    └─ No → 価値軸を再検討

Q3: 事業間でスタッフ・クライアントの共有があるか？
    │
    ├─ Yes → フュージョン型を推奨
    │
    └─ No → 事業部軸も検討可能
```

## 人材派遣業固有の考慮事項

### VS別の重点ポイント

| VS | 人材派遣業での重点 |
|----|--------------------|
| VS0 | 「グッドマッチ」vs「ベストマッチ」戦略の選択 |
| VS1 | 両面市場のポテンシャル分析（潜在市場開拓） |
| VS2 | 拡張マッチングアルゴリズム（70%一致の許容） |
| VS2.5 | 継続フォローによる価値向上（成熟期の差別化要因） |
| VS3 | 「育てる派遣」ブランドの確立 |
| VS4 | 未経験者・ブランク層の開拓 |
| VS5 | スキルアップ支援、キャリアパス設計 |
| VS6 | 成長予測AI、離脱リスク早期検知 |
| VS7 | 週次タッチポイントの標準化 |

### 規制対応の組み込み

```
全VS共通: 労働者派遣法対応
VS2: 雇用安定措置義務（3年ルール）
VS3: 同一労働同一賃金対応
VS5: 派遣先での安全配慮義務
VS6: 個人情報保護、データ活用同意
VS7: 36協定管理、勤怠管理
```

### 両面市場の統合視点

人材派遣業の価値軸では、各VSで両面の価値を同時に設計：

| VS | スタッフ価値 | クライアント価値 |
|----|-------------|-----------------|
| VS0 | より多くの就業機会 | 柔軟な人材確保 |
| VS2 | 成長を前提とした採用 | 育成型派遣の活用 |
| VS2.5 | 市場価値の継続向上 | 人材投資リスク分散 |
| VS5 | スキルアップ機会 | パフォーマンス向上 |
