# 地方銀行 - 軸選択ガイド

## 業界特性

| 特性 | 説明 |
|------|------|
| **多面ステークホルダー** | 個人顧客・法人顧客・行員・地域社会の4者構造 |
| **規制産業** | 金融庁規制、バーゼル規制、マネロン対策など |
| **地域密着** | 地域経済との一体化、地方創生の担い手 |
| **デジタル化圧力** | ネット銀行との競争、DX推進の必要性 |
| **収益構造変化** | 低金利環境、手数料ビジネスへのシフト |

## 軸選択マトリクス

```
                    地域コミットメントの強さ
                    低い            高い
                 ┌──────────────┬──────────────┐
   経営統合      │ ケーパビリティ │ フュージョン型 │
   の可能性      │   統合型      │              │
      高い      │              │              │
               ├──────────────┼──────────────┤
      低い     │  事業部軸    │   価値軸     │
               │              │      ◎      │
               └──────────────┴──────────────┘
```

## 推奨軸と理由

### 第1推奨: 価値軸（地域共創型）

**理由**:
- 地方銀行の存在意義は「地域社会への貢献」という統一理念
- 個人・法人・行員が「地域」という共通価値でつながる
- 地方創生・SDGsの文脈で全活動を一貫説明可能

**適用パターン**:
```
VS0: 地域ビジョン・戦略立案（地域の10年後を描く）
VS1: 地域インサイト・機会発見
VS2: 地域体験価値創造（ライフステージ×成長ステージ）
VS3: 地域ブランド・発信
VS4: 地域経済循環（資金の地産地消）
VS5: 地域セーフティネット
VS6: 地域データ基盤
VS7: 地域オペレーション最適化
```

### 第2推奨: フュージョン型

**理由**:
- 経営統合（持株会社化）を見据える場合に有効
- 共通層（VS0, VS6, VS7）で統合効果を出しつつ、地域特性を維持

**適用パターン**:
```
共通層: VS0（統合戦略）、VS6（IT基盤）、VS7（バックオフィス）
地域層: VS1-VS5を各地域銀行で独自運用
```

### 検討が必要: 事業部軸

**適用条件**:
- リテール・ホールセールで組織が明確に分離
- セグメント別のP/L管理が徹底されている

**リスク**:
- 地域一体での価値創造が弱まる
- 顧客接点で分断が生じる

### 非推奨: ケーパビリティ統合型

**理由**:
- 地方銀行の強みは「地域密着」「関係性」
- 機能別統合は顔の見える関係を希薄化
- メガバンクとの差別化ポイントを失う

## 判断フローチャート

```
Q1: 地域への貢献を経営の中心に据えているか？
    │
    ├─ Yes → 価値軸（地域共創型）を推奨
    │
    └─ No → Q2へ

Q2: 他行との経営統合・持株会社化を検討中か？
    │
    ├─ Yes → フュージョン型を推奨
    │
    └─ No → Q3へ

Q3: リテール・ホールセールの分離が進んでいるか？
    │
    ├─ Yes → 事業部軸も検討可能
    │
    └─ No → 価値軸を再検討
```

## 地方銀行固有の考慮事項

### VS別の重点ポイント

| VS | 地方銀行での重点 |
|----|-----------------|
| VS0 | 地域ビジョンとの整合、SDGs/ESG戦略 |
| VS1 | 地域データの独自性（取引データ、地域情報） |
| VS2 | 3者価値の統合（個人・法人・行員） |
| VS3 | 地域ブランドの発信、UIJターン促進 |
| VS4 | 地域経済循環、クラウドファンディング |
| VS5 | 金融包摂、高齢者見守り、災害対応 |
| VS6 | 勘定系・情報系の統合、AI活用 |
| VS7 | 店舗網最適化、デジタルチャネル強化 |

### 規制対応の組み込み

```
全VS共通: 金融庁ガイドライン対応
VS4: マネロン対策、KYC
VS5: 顧客保護、説明責任
VS6: サイバーセキュリティ、FISC基準
VS7: BCP、システムリスク管理
```

### 3者統合の視点

地方銀行の価値軸では、各VSで3者の価値を同時に設計：

| VS | 個人顧客価値 | 法人顧客価値 | 行員価値 |
|----|-------------|-------------|---------|
| VS0 | 豊かな生活設計 | 持続可能な事業 | 地域プロデューサー |
| VS2 | ライフステージ支援 | 成長ステージ支援 | キャリア成長 |
| VS4 | 資産形成 | 資金調達 | 地域貢献実感 |
