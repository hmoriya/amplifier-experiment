# 地方銀行 - 事業部軸バリューストリーム

## 概要

地方銀行における事業部軸アプローチは、リテール（個人）とホールセール（法人）を軸に組織を構成するモデルです。セグメント別P/L管理が徹底されている場合に適用可能です。

## 構造

```
┌─────────────────────────────────────────────────────────┐
│                    共通層（本部）                        │
│  VS0: 経営戦略・リスク管理・コンプライアンス           │
│  共通機能: IT基盤、バックオフィス、人事              │
└─────────────────────────────────────────────────────────┘
        │               │               │
        ▼               ▼               ▼
   ┌─────────┐     ┌─────────┐     ┌─────────┐
   │リテール │     │ホールセール│    │ 市場部門 │
   │ VS1-VS7 │     │ VS1-VS7 │     │ VS1-VS7 │
   └─────────┘     └─────────┘     └─────────┘
```

---

## 共通層（本部）

### VS0: 経営戦略・リスク管理

**責任範囲**:
- 中期経営計画
- ALM（資産負債管理）
- 統合リスク管理
- コンプライアンス・AML

**共通機能**:
| 機能 | 内容 |
|------|------|
| IT基盤 | 勘定系、情報系、サイバーセキュリティ |
| バックオフィス | 事務センター、決済 |
| 人事 | 採用、研修、評価 |
| 監査 | 内部監査、外部監査対応 |

## 事業部軸の注意点

**メリット**:
- セグメント別収益管理が明確
- 専門性の深化
- 責任の明確化

**デメリット**:
- 地域一体での価値創造が弱まる
- 顧客接点での分断
- 重複投資のリスク

**対策**:
- オーナー経営者向けの部門横断チーム
- 地域担当制との併用
- 定期的な情報共有会議
