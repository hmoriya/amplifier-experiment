# 飲料・食品業界 - 軸選択ガイド

## 業界特性

| 特性 | 説明 |
|------|------|
| **事業多角化** | 酒類・飲料・食品・健康食品など複数カテゴリ |
| **ブランド重視** | 消費者向けブランド価値が競争優位の源泉 |
| **サプライチェーン複雑性** | 原材料調達から販売チャネルまで多層構造 |
| **規制対応** | 食品安全、酒類規制、健康表示規制など |
| **サステナビリティ** | 水資源、容器リサイクル、環境負荷低減 |

## 軸選択マトリクス

```
                    事業間シナジーの重要度
                    低い            高い
                 ┌──────────────┬──────────────┐
   組織変革     │  事業部軸    │ フュージョン型 │
   への許容度   │              │      ◎       │
      低い     │              │              │
               ├──────────────┼──────────────┤
      高い     │ケーパビリティ│   価値軸     │
               │   統合型     │              │
               └──────────────┴──────────────┘
```

## 推奨軸と理由

### 第1推奨: フュージョン型

**理由**:
- 飲料・食品は事業間に共通性（製造・物流・品質管理）と独自性（商品特性・チャネル）の両方がある
- 段階的な統合が現実的
- 企業理念（例：「水と生きる」）を共通層で体現しつつ、事業特性を活かせる

**適用パターン**:
```
共通層: VS0（戦略）、VS6（データ）、VS7（品質・サステナビリティ）
事業層: VS1-VS5を事業別にカスタマイズ
```

### 第2推奨: 価値軸

**理由**:
- 企業理念を核とした統一的な価値創造を実現
- 事業横断のシナジー（研究開発、ブランド）を最大化

**注意点**:
- 事業部の独立性が高い場合は抵抗が生じやすい
- トップダウンでの推進力が必要

### 非推奨: 事業部軸（単独）

**理由**:
- 飲料・食品業界では事業間シナジーが価値創造の鍵
- 縦割りを強化すると競争力低下のリスク

## 判断フローチャート

```
Q1: 企業理念が事業横断で浸透しているか？
    │
    ├─ Yes → Q2へ
    │
    └─ No → 事業部軸から開始し、段階的にフュージョン型へ移行

Q2: 研究開発・製造・物流の共通化が進んでいるか？
    │
    ├─ Yes → 価値軸 または フュージョン型
    │
    └─ No → フュージョン型（共通層から段階的に構築）

Q3: トップの変革へのコミットメントは？
    │
    ├─ 強い → 価値軸
    │
    └─ 中程度 → フュージョン型
```

## 飲料・食品業界固有の考慮事項

### VS別の重点ポイント

| VS | 飲料・食品での重点 |
|----|-------------------|
| VS0 | サステナビリティ戦略の統合 |
| VS1 | 消費者インサイト・トレンド予測 |
| VS2 | 健康価値・機能性の研究開発 |
| VS3 | ブランドポートフォリオ管理 |
| VS4 | 複合チャネル（小売・外食・自販機・EC） |
| VS5 | 食品安全・品質保証 |
| VS6 | 消費者データ統合・パーソナライズ |
| VS7 | 環境負荷低減・循環型オペレーション |

### 規制対応の組み込み

```
酒類事業: VS3, VS4に酒類規制対応を組み込み
健康食品: VS2, VS3に機能性表示規制を組み込み
全事業: VS5に食品安全規制を組み込み
```
