# 人材派遣業バリューストリーム構成（両面市場型・成熟期対応モデル）

## 概要

人材派遣業は典型的な両面市場型ビジネスです。成長期から成熟期への移行に伴い、「ベストマッチの追求」から「グッドマッチの最大化」へ、そして「獲得重視」から「継続フォロー重視」へと競争軸が変化します。

## コアコンセプト：グッドマッチ戦略

### ベストマッチ vs グッドマッチ

```
【従来：ベストマッチ幻想】
完璧な条件一致を追求 → マッチング率10% → 機会損失90%
    ↓
【新戦略：グッドマッチ最大化】
70点マッチ＋育成でフィット → マッチング率40% → 市場拡大4倍

スタッフ側調整                    クライアント側調整
・スキルギャップ研修              ・要件の本質化
・マインドセット形成              ・受入体制整備
・期待値調整                      ・OJT設計
```

## 成功指標（KPI）の進化

### 従来KPI → 成熟期KPI

| カテゴリ | 従来KPI | 成熟期KPI |
|----------|---------|-----------|
| マッチング | マッチング率 10% | グッドマッチ率 40% |
| 継続性 | 3ヶ月継続率 60% | 12ヶ月継続率 80% |
| 成長 | - | スキルアップ実現率 70% |
| 満足度 | 初回満足度 | 継続満足度スコア 4.5/5 |
| 収益性 | 新規売上 | LTV（3年間総収益） |

### 新規KPI

1. **フィット向上率**: 就業3ヶ月後の評価改善度
2. **相互NPS**: スタッフ→企業、企業→スタッフの推奨度
3. **成長実現率**: 計画したキャリアパスの達成度
4. **調整成功率**: グッドマッチから優良継続への転換率

## 実装上の重要ポイント

### 1. マインドセットの転換

- 「完璧な人材」→「育てばフィットする人材」
- 「要件100%」→「コア要件＋成長可能性」
- 「初回評価」→「成長曲線評価」

### 2. オペレーションの変革

- 週次タッチポイントの標準化
- 介入プレイブックの整備
- 成長支援チームの設置

### 3. テクノロジーの活用

- AIによる成長予測
- 自動アラートシステム
- パーソナライズ学習推奨

### 4. 価格戦略の見直し

- 「即戦力プレミアム」から「育成投資モデル」へ
- 継続期間による段階的価格設定
- 成長成果に応じた成功報酬

## グッドマッチによる好循環メカニズム

### なぜグッドマッチはフォローの重要性を高めるのか

```
【従来のベストマッチモデル】
100%フィット → 自走可能 → 最小限フォロー → 成長なし → 市場価値停滞
                                                    ↓
                                            派遣会社の付加価値低下

【グッドマッチモデルの好循環】
70%フィット → ギャップ存在 → 濃密フォロー必須 → 継続的成長
    ↓              ↓              ↓              ↓
市場4倍拡大    成長余地大    高頻度接触     スキルアップ
    ↓              ↓              ↓              ↓
    └──────────────────────────────────────────┘
              持続的な付加価値創造
```

### フォローROIの劇的改善

| 指標 | ベストマッチ | グッドマッチ | 倍率 |
|------|-------------|-------------|------|
| フォロー頻度 | 月1回 | 週2-3回 | 10倍 |
| 成長率/年 | 5% | 30% | 6倍 |
| 継続率 | 60% | 85% | 1.4倍 |
| LTV | 100万円 | 500万円 | 5倍 |
| 紹介/口コミ | 10% | 40% | 4倍 |

### 成長支援型ビジネスモデルへの転換

1. **フォローが収益の源泉に**
   - 成長支援サービスの有料化
   - スキルアップ研修の提供
   - キャリアコンサルティング

2. **データ蓄積による競争優位**
   - 成長パターンの可視化
   - 介入効果の定量化
   - 予測モデルの高度化

3. **エコシステムの形成**
   - 教育機関との連携
   - 資格取得支援
   - キャリアコミュニティ

## まとめ

このモデルは、人材派遣業を「マッチングビジネス」から「人材成長支援ビジネス」へと再定義します。グッドマッチ戦略により市場を4倍に拡大し、継続的なフォローにより差別化を実現します。

**グッドマッチの本質は「70%の出会いを、フォローを通じて120%の成功に育てる」こと。** これにより、フォロー活動が単なるコストセンターから、最大の価値創造源へと変貌します。成熟期における持続的競争優位の源泉は、このデータに基づく成長支援能力にあります。
