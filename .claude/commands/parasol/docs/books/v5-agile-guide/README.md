# はじめてのParasol V5 × アジャイル実践ガイド（初版）

## 書籍概要

本書は、Parasol V5とアジャイル開発手法を統合し、価値駆動開発を実現するための実践的なガイドブックです。スクラムフレームワークにV5の価値体系を組み込む具体的な方法を解説します。

## コンセプト

**価値駆動スクラムで変わるチーム開発**

従来のアジャイル開発に不足していた「価値」の視点を、V5のフレームワークで補完し、真に価値を生み出すアジャイル開発を実現します。
