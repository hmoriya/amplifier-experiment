**Parasol V5における「良い設計とは何か」**

- 設計の「正しさ」と「良さ」の違いを解説
- 実践的な設計例を豊富に収録

**アジャイル原点回帰ガイド with Parasol V5（改訂版）**
- アジャイル本来の価値志向を再発見
