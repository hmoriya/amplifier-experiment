# アジャイル原点回帰ガイド with Parasol V5（改訂版）

## 書籍概要

本書は、アジャイル本来の価値志向を再発見し、Parasol V5を活用してその理想を実現するための実践ガイドです。アジャイルは元々価値駆動であったという認識に立ち、V5を「アジャイルの理想を具現化する手段」として位置付けます。

## 改訂版の背景

リサーチの結果、アジャイル宣言の第一原則が「価値あるソフトウェアの早期・継続的提供」であることを再確認。V5は新しい価値駆動手法ではなく、アジャイル本来の価値志向を実現する具体的手段として再定義されました。

## コンセプト

**失われた価値を取り戻す実践手法**

アジャイルの理想と現実のギャップに悩む実践者に寄り添い、本来のアジャイル（価値駆動開発）を取り戻すための具体的な方法を提供します。

## 改訂版の特徴

1. **共感的アプローチ**
   - アジャイル実践者の悩みに寄り添う
   - 批判ではなく、理想と現実のギャップを認識

2. **原点回帰**
   - アジャイルを否定しない
   - 既存の枠組みを最大限活用
   - 「本来のアジャイル」への回帰として位置付け

3. **段階的導入**
   - 一気に変えようとしない
   - 小さな成功体験の積み重ね
   - 抵抗を最小化する導入方法
