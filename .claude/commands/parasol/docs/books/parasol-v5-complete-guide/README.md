## 本書の目的

本書は、Parasol V5フレームワークを体系的に学び、実践レベルまで習得するための完全ガイドです。Value Stream駆動アプローチの理論から実装まで、段階的に理解を深められるよう構成されています。

## 本書の特徴

### 1. 体系的な学習構造
- 基礎から発展まで8部構成
- 価値領域・問題領域・解決領域を明確に分離
- 各章は理論→実践→応用の流れ
- 豊富な実例とケーススタディ

### 2. 実践重視のアプローチ
- Claude Code統合による実践例
- 実際のプロジェクトを通じた解説
- すぐに使えるテンプレート提供

### 3. 産業特化の知識
- 業界別パターンカタログ
- カスタマイズ手法の詳細解説
- 実績に基づくベストプラクティス

## 本書の活用方法

### 初心者の方へ
1. 第I部（全章）で基本思想を理解
2. 第II部（第6-7章）でプロジェクトの始め方を学習
3. 第III部（第9章）で価値ストリームの基本を理解
4. 第VII部（第32章）でClaude Code統合から実践

### 実践者の方へ
1. 第I部（第2、4章）で3スペースとAxiomatic Designを確認
2. 第III-V部（全章）で価値から実装までの流れを体系的に学習
3. 第VI部（全章）で価値の追跡と実現方法を習得

### 上級者の方へ
1. 第IV部（第15-16章）でZIGZAGとDesign Matrixを深掘り
2. 第V部（セクション3-4）で実装とプラットフォームの詳細を確認
3. 第VII部（第35章）でカスタムパターン開発
4. 第VIII部（全章）で最新動向と将来展望を把握

### マネージャーの方へ
1. 第I部（第1-2章）で全体像を把握
2. 第III部（全章）で価値領域の管理方法を理解
3. 第VI部（第29-30章）で価値追跡の仕組みを学習
4. 第VII部（第33章）でチーム編成とスケーリングを確認
