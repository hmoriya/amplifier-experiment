# Parasol V5 メタファー代替案

## 現在の登山メタファーの問題点
- Phase 0→7が「登り」のイメージだが、実際の作業量はPhase 2-3がピーク
- 後半に向かって「大変になる」印象を与えてしまう
- 実際は後半は収束・具体化のフェーズ

## メタファー候補

### 1. 🏗️ 建築・建設メタファー「夢の家を建てる」

**フェーズの対応**：
- Phase 0-1: 土地調査・地盤調査
- Phase 2-3: 設計図作成（最も創造的）
- Phase 4-5: 詳細図面・施工計画
- Phase 6-7: 建設・内装工事

**利点**：
- 設計（Phase 2-3）が最も重要という本質を表現
- 具体化していくプロセスが自然
- 誰もが理解しやすい

**使用例**：
「Phase 2-3は建築家が理想の家を設計する段階。ここで全体の良し悪しが決まります」

### 2. 🎨 彫刻メタファー「大理石から像を掘り出す」

**フェーズの対応**：
- Phase 0-1: 大理石の選定・観察
- Phase 2-3: 像の形を見出す（ビジョン）
- Phase 4-5: 荒削り・形を整える
- Phase 6-7: 細部の仕上げ・磨き

**利点**：
- ミケランジェロの「像は既に石の中にある」という哲学と合致
- 発見的なプロセスを表現
- 収束していく作業の流れが自然

**使用例**：
「価値分析は、ビジネスの中に眠る本質的な形を見出す作業です」

### 3. 🧩 パズル/ジグソーパズルメタファー

**フェーズの対応**：
- Phase 0-1: 箱を開けて全体像を確認
- Phase 2-3: ピースを分類・エッジを見つける
- Phase 4-5: セクションごとに組み立て
- Phase 6-7: 最後のピースをはめ込む

**利点**：
- 全体と部分の関係が明確
- 段階的な組み立てプロセス
- 完成に向かう収束感

### 4. 🌱 園芸・栽培メタファー「理想の庭園を育てる」

**フェーズの対応**：
- Phase 0-1: 土壌調査・環境確認
- Phase 2-3: 庭園設計・植栽計画（創造的）
- Phase 4-5: 植え付け・配置
- Phase 6-7: 成長管理・手入れ

**利点**：
- 生きたシステムという概念
- 継続的な成長と変化
- 環境との調和

### 5. 🔍 探検・発見メタファー「新大陸への航海」

**フェーズの対応**：
- Phase 0-1: 航海準備・海図確認
- Phase 2-3: 新大陸発見・地図作成（最も冒険的）
- Phase 4-5: 航路確立・拠点建設
- Phase 6-7: 定期航路化・貿易確立

**利点**：
- 発見の興奮を表現
- 未知から既知への変化
- ビジネス価値の「発見」という本質

### 6. 🍳 料理メタファー「究極の料理を作る」

**フェーズの対応**：
- Phase 0-1: 食材調達・下準備
- Phase 2-3: レシピ考案・味の設計（創造性のピーク）
- Phase 4-5: 調理手順・火加減調整
- Phase 6-7: 盛り付け・仕上げ

**利点**：
- 身近で理解しやすい
- 創造性と技術の融合
- 完成に向けた収束

### 7. 🎭 演劇・舞台メタファー「感動の舞台を創る」

**フェーズの対応**：
- Phase 0-1: 脚本選定・キャスティング
- Phase 2-3: 演出プラン・稽古（創造的ピーク）
- Phase 4-5: 舞台装置・音響照明
- Phase 6-7: リハーサル・本番

**利点**：
- 協働作業の重要性
- 創造と実現のバランス
- ステークホルダー（観客）意識

## 推奨案：建築メタファー

**理由**：
1. Phase 2-3の「設計」が最重要という本質を最もよく表現
2. ビジネスとシステムの関係が理解しやすい
3. 段階的な具体化プロセスが自然
4. 品質の重要性（基礎工事など）も表現可能
5. プロジェクトの進行イメージと合致

**第2章での適用例**：

「Parasol V5は、理想のビジネスシステムという『建築物』を建てるプロセスです。

まず土地（企業）を調査し（Phase 0-1）、どんな建物（システム）がふさわしいか理解します。

次に、建築家として最も創造的な仕事—設計（Phase 2-3）を行います。ここで建物の価値、機能、美しさすべてが決まります。

設計が固まったら、詳細な図面（Phase 4-5）を作成し、最後に実際の建設（Phase 6-7）に入ります。

優れた建築物と同じように、優れたシステムも、しっかりした設計なくしては生まれません。」
