## 💡 読書のコツ

### 効果的な読み方
1. **目次を俯瞰** - 全体構造の把握
2. **図表から入る** - ビジュアルで理解
3. **事例を重視** - 具体例から学ぶ
4. **演習を実施** - 手を動かして定着

### 理解度チェック
- [ ] 各章の「Chapter Summary」が説明できる
- [ ] 主要な図表を描ける
- [ ] 事例を自組織に置き換えられる
- [ ] 他者に教えられる
