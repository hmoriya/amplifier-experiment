# 第I部：基礎編「Parasol V5への招待」

**ページ数**: 50ページ  
**対象読者**: Parasol V5を初めて学ぶ方、基本概念を体系的に理解したい方

## 概要

第I部では、Parasol V5フレームワークの基本的な考え方と思想を学びます。なぜこのフレームワークが必要なのか、どのような問題を解決するのか、そしてどのような原理に基づいて設計されているのかを理解することで、後続の章での学習の土台を築きます。

## 学習目標

この部を読み終えると、以下のことが理解できるようになります：

1. **Parasol V5の存在意義** - なぜ今このフレームワークが必要なのか
2. **3つのスペースの概念** - WHY（価値）、WHAT（問題）、HOW（解決）の明確な分離
3. **基本哲学とマインドセット** - フレームワークの根底にある思想
4. **Axiomatic Designの原理** - 設計公理による品質保証の仕組み
5. **DDDとの融合** - ドメイン駆動設計との相乗効果
