# V5アジャイルガイド 新旧構成比較

## 基本コンセプトの変化

| 観点 | 旧版 | 改訂版 |
|------|------|--------|
| **基本認識** | アジャイルに価値視点が不足 | アジャイルは本来価値駆動 |
| **V5の位置付け** | 価値駆動の新手法 | アジャイルの理想を実現する手段 |
| **メッセージ** | アジャイル＋V5で価値駆動へ | アジャイル本来の姿を取り戻す |
| **アプローチ** | 新しい概念の導入 | 原点回帰と具現化 |

## タイトルの変化

### 旧版
- **メイン**: 「はじめてのParasol V5 × アジャイル実践ガイド」
- **サブ**: 「価値駆動スクラムで変わるチーム開発」
- **含意**: V5という新しいものを学ぶ

### 改訂版案
- **メイン**: 「アジャイル原点回帰ガイド with Parasol V5」
- **サブ**: 「失われた価値を取り戻す実践手法」
- **含意**: アジャイル本来の姿を再発見する

## 章構成の比較

### 第1部の変化

| 章 | 旧版 | 改訂版 |
|----|------|--------|
| **第1章** | なぜ価値駆動アジャイルが必要なのか | アジャイルが約束した価値創造 |
| 焦点 | アジャイルの問題点を指摘 | アジャイル本来の理想を確認 |
| トーン | 批判的・問題提起型 | 共感的・原点確認型 |

| **第2章** | V5とアジャイルの美しい統合 | 価値の空洞化 ― 現実のアジャイル実践 |
| 焦点 | V5という解決策の提示 | 理想と現実のギャップの共感的描写 |
| トーン | 解決策提示型 | 共感・問題認識型 |

| **第3章** | 価値駆動スプリントの全体像 | V5で取り戻すアジャイルの真価 |
| 焦点 | 新しい方法論の説明 | 本来の姿を実現する方法 |
| トーン | 教示的 | 希望提示型 |

### ストーリーラインの違い

#### 旧版のストーリー
```
1. アジャイルには価値視点が欠けている（問題）
2. V5を統合すれば価値駆動になる（解決策）
3. こうやって実践しよう（方法）
```

#### 改訂版のストーリー
```
1. アジャイルは元々価値を重視していた（原点）
2. でも現実は価値が見失われがち（共感）
3. V5でアジャイル本来の姿を実現しよう（方法）
```

## 読者への訴求の違い

### 旧版
- 「新しい手法を学びましょう」
- 「アジャイルを改善しましょう」
- 「V5という優れた方法があります」

### 改訂版
- 「あなたの悩みは正しい」
- 「アジャイルの理想を一緒に実現しましょう」
- 「既にあるものを活かしつつ改善できます」

## 具体例の違い

### 第1章冒頭の例

#### 旧版
```
チームA（失敗例）vs チームB（成功例）の対比
→ 教訓的、理想化された例
```

#### 改訂版
```
金曜日のスプリントレビューでの気まずい沈黙
→ リアルな「あるある」体験、共感を誘う
```

### 価値の説明

#### 旧版
```
V5の価値定義：
VL1 → VL2 → VL3
（新しい概念として導入）
```

#### 改訂版
```
アジャイル宣言の再読：
"valuable software"（価値あるソフトウェア）
（既にある概念の再発見）
```

## メリットとデメリット

### 旧版のメリット
- V5の新規性が明確
- 体系的な学習が可能
- 完全に新しいスタートが切れる

### 旧版のデメリット
- 既存のアジャイル投資が無駄に感じる
- 「また新しい手法か」という抵抗感
- アジャイルを否定していると誤解される

### 改訂版のメリット
- アジャイル実践者の共感を得やすい
- 既存の投資を活かせる
- 抵抗感が少ない
- 本質的な改善として受け入れられる

### 改訂版のデメリット
- V5の独自性が薄れる可能性
- 「新しさ」のインパクトが弱い
- アジャイルの枠内に収まってしまう印象

## 推奨される使い分け

### 旧版が適している場合
- V5を独立した手法として学びたい組織
- アジャイルに失望している組織
- 全く新しいアプローチを求めている人

### 改訂版が適している場合
- アジャイルを実践中で改善したい組織
- アジャイルの理想と現実のギャップに悩む人
- 段階的な改善を望む組織

## 結論

改訂版は、アジャイル実践者により寄り添い、V5を「アジャイルを否定するもの」ではなく「アジャイルの理想を実現する手段」として位置付けることで、受け入れやすさと実践しやすさを高めています。

特に日本のようにアジャイル導入が進んでいる市場では、改訂版のアプローチの方が効果的と考えられます。
