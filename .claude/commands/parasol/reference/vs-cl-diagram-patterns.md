# VS-CL階層図パターン集

**バージョン**: V5.4
**最終更新**: 2025-01-06

---

## 概要

このドキュメントは、Value Stage（VS）とCapability Level（CL）の関係を視覚化するための図パターンを収録しています。コンサルティング資料やアーキテクチャ文書で使用できるASCIIアート形式の図を提供します。

---

## 図の構成要素

### 横軸: Value Stage（WHEN軸）

顧客ライフサイクルの段階を表すシェブロン（矢印型）で表現：

| Stage | 名称 | 説明 |
|-------|------|------|
| VS0 | 認知 | 製品・サービスを知る |
| VS1 | 関心 | 興味を持ち情報収集 |
| VS2 | 検討 | 比較検討・評価 |
| VS3 | 購入 | 購入決定・契約 |
| VS4 | 利用 | 製品・サービスを使用 |
| VS5 | 満足 | 価値を実感 |
| VS6 | 推奨 | 他者に推薦 |
| VS7 | 継続 | リピート・ロイヤルティ |

### 縦軸: Capability Level（包含関係）

ネストしたボックスで包含関係を表現：

| Level | 名称 | 説明 |
|-------|------|------|
| CL1 | Activity Area | 活動領域（価値を生む活動の単位） |
| CL2 | Capability | ケイパビリティ（活動を実現する能力） |
| CL3 | Business Operation | 業務操作（システムが提供する具体的操作） |

**包含関係**: CL1 ⊃ CL2 ⊃ CL3

### プロンプト例

```

- 横方向にValue Stage（VS0〜VS4）をシェブロンで配置
- 右端に「顧客価値達成」のゴールボックス

- VS0: 認知 → CL1: 認知獲得 → CL2: 広告管理 → CL3: 配信/停止/分析
- VS1: 関心 → CL1: 関心醸成 → CL2: リード管理 → CL3: 登録/更新/削除

```

## カスタマイズ用プロンプトテンプレート

### 基本テンプレート

```
VS-CL階層図を作成してください。

【形式】
- パターン: （基本型/複数CL2型/シンプル型/分離型/スイムレーン型/フルステージ型）
- VSの範囲: VS{開始}〜VS{終了}
- CLの深さ: CL1のみ / CL1+CL2 / CL1+CL2+CL3
- 凡例: あり/なし

【内容】
- VS{N}: {ステージ名}
  - CL1: {活動領域名}
    - CL2: {ケイパビリティ名}
      - CL3: {操作1}/{操作2}/{操作3}

【追加要素】（オプション）
- Actor別スイムレーン: {Actor1}/{Actor2}/{Actor3}
- タイムライン表示: あり/なし
- ゴールボックス: {顧客価値の内容}
```

### 業界別カスタマイズ例

**EC/小売業向け:**
```
VS0:認知 → VS1:関心 → VS2:検討 → VS3:購入 → VS4:配送 → VS5:利用 → VS6:推奨
```

**B2B/SaaS向け:**
```
VS0:認知 → VS1:関心 → VS2:評価 → VS3:契約 → VS4:導入 → VS5:活用 → VS6:更新
```

**金融/保険向け:**
```
VS0:認知 → VS1:関心 → VS2:相談 → VS3:申込 → VS4:審査 → VS5:契約 → VS6:継続
```
