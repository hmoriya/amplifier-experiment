## エージェント利用時の参照ガイドライン

### 1. フェーズ実行時の章参照

各フェーズエージェントは、対応する章を参照して具体例や理論的背景を提供：

```markdown
Phase 1を実行する際は、Chapter 7「組織コンテキスト確立」の
市場分析フレームワークを参考にしてください。
```

### 2. Design Matrixの一貫した活用

Phase 3以降では、必ずDesign Matrixを参照：

```markdown
ZIGZAGプロセスの各イテレーションで、Chapter 13で説明された
Design Matrix進化プロセスに従い、FR-DPマッピングを更新します。
```

### 3. 実例の活用

各章の冒頭ストーリーを参考例として活用：

```markdown
組織アセスメントの重要性については、Chapter 6のTarget社の
20億ドル損失事例が参考になります。
```
